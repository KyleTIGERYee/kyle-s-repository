# 集团小程序功能说明

集团小程序功能说明

# 概述

集团小程序是安居乐寓为集团客户、经纪人及门店管理人员提供的移动端业务管理平台。通过小程序，用户可以完成楼盘浏览、客户推荐、预约看房、客户管理、签到打卡等核心业务操作。本功能说明涵盖集团小程序的主要功能模块、操作流程及权限配置。

本系统适用于以下场景：
- 经纪人推荐客户并跟踪客户状态
- 客户浏览楼盘信息并预约看房
- 门店管理人员进行客户分配和跟进
- 用户签到打卡及业绩统计

# 功能模块

## 一、登录与授权

### （一）操作前提

- 用户已注册并开通小程序账号
- 用户已绑定手机号码
- 若为经纪人，需完成经纪人认证

### （二）操作步骤

1. 打开小程序，进入**登录**页面
2. 点击**微信授权登录**，获取微信用户信息
3. 输入手机号码，点击**获取验证码**
4. 填写短信验证码，点击**登录**
5. 首次登录需完善个人信息（头像、昵称等）

### （三）注意事项

- 一个手机号码只能绑定一个微信账号
- 经纪人账号需通过后台审核后方可使用推荐功能
- 登录状态有效期为30天，过期需重新登录

## 二、首页

### （一）功能概述

首页是用户进入小程序后的主界面，展示楼盘推荐、金刚位入口、公告通知等核心内容。

### （二）操作步骤

1. **楼盘浏览**
   - 首页展示推荐楼盘卡片，包含楼盘封面图、楼盘名称、价格区间等信息
   - 左右滑动可浏览更多楼盘
   - 点击楼盘卡片进入**楼盘详情**页面

2. **金刚位入口**
   - 首页顶部展示金刚位图标入口，包括：
     - **我要找房**：进入楼盘搜索页面
     - **我要推荐**：进入客户推荐页面
     - **我的客户**：进入客户管理页面
     - **扫码签到**：打开扫码功能进行签到

3. **公告查看**
   - 首页顶部轮播展示系统公告
   - 点击公告可查看详情

### （三）注意事项

- 首页楼盘推荐根据用户偏好和历史浏览记录智能排序
- 金刚位图标支持后台配置，可根据运营需求调整

## 三、楼盘浏览与详情

### （一）操作前提

- 用户已登录小程序

### （二）操作步骤

1. **楼盘列表浏览**
   - 在首页点击**我要找房**或在金刚位点击**楼盘**入口
   - 进入楼盘列表页面，展示楼盘卡片列表
   - 支持按区域、价格、户型等条件筛选

2. **楼盘详情查看**
   - 点击楼盘卡片进入**楼盘详情**页面
   - 详情页包含以下模块：
     - **基本信息**：楼盘名称、地址、均价、开盘时间等
     - **楼盘图片**：支持查看效果图、实景图、户型图
     - **户型介绍**：展示各户型面积、朝向、价格
     - **周边配套**：展示交通、教育、商业等配套信息
     - **售楼说明书**：查看楼盘详细资料

3. **图片查看**
   - 点击楼盘图片进入**图片-全部**页面
   - 支持分类查看：效果图、实景图、户型图、样板房
   - 支持左右滑动浏览和放大查看

4. **VR看房**（如适用）
   - 点击**VR看房**按钮进入VR动态图页面
   - 支持360度全景浏览

### （三）注意事项

- 楼盘信息实时同步PC端管理系统
- 价格信息仅供参考，以实际成交为准
- VR功能需楼盘方提供VR素材方可使用

## 四、客户推荐

### （一）操作前提

- 用户已完成经纪人认证
- 经纪人账号状态正常

### （二）操作步骤

1. **发起推荐**
   - 在首页点击**我要推荐**或在楼盘详情页点击**推荐客户**
   - 进入**推荐客户**页面

2. **填写客户信息**
   - 填写客户姓名（必填）
   - 填写客户手机号码（必填）
   - 选择意向楼盘（可多选）
   - 填写客户备注信息（选填）

3. **提交推荐**
   - 点击**提交**按钮完成推荐
   - 系统校验客户信息有效性
   - 推荐成功后显示"推荐成功"提示

4. **查看推荐记录**
   - 进入**我的客户**页面
   - 查看推荐客户列表及状态
   - 支持按状态筛选：待跟进、已到访、已成交、无效

### （三）注意事项

- 同一客户被多个经纪人推荐时，以首次推荐为准
- 客户保护期为30天，保护期内其他经纪人无法重复推荐
- 客户成交后，佣金按推荐关系归属计算

## 五、客户管理

### （一）操作前提

- 用户已登录小程序
- 经纪人可查看自己推荐的客户
- 店长/管理员可查看门店全部客户

### （二）操作步骤

1. **客户列表查看**
   - 点击金刚位**我的客户**进入**客户管理**页面
   - 列表展示客户姓名、手机号（脱敏）、意向楼盘、跟进状态
   - 支持按客户状态、推荐时间筛选

2. **客户详情查看**
   - 点击客户卡片进入**客户详情**页面
   - 查看客户基本信息、推荐记录、跟进记录
   - 查看客户预约记录和到访记录

3. **客户跟进**
   - 在客户详情页点击**跟进**
   - 填写跟进内容和下次跟进计划
   - 支持上传跟进附件（图片、语音）

4. **客户分配**（店长/管理员权限）
   - 在客户列表页点击**分配**
   - 选择分配对象（经纪人）
   - 确认分配后客户归属变更

### （三）注意事项

- 客户手机号默认脱敏显示，点击可查看完整号码
- 客户跟进记录实时同步，支持多设备查看
- 客户分配后，原跟进记录保留，新跟进人可查看历史记录

## 六、预约看房

### （一）操作前提

- 用户已登录小程序
- 客户已通过推荐或自主注册

### （二）操作步骤

1. **发起预约**
   - 在楼盘详情页点击**预约看房**
   - 或在客户详情页点击**帮客户预约**
   - 进入**预约看房**页面

2. **填写预约信息**
   - 选择预约楼盘（如从楼盘页进入则自动填充）
   - 选择预约日期和时间段
   - 填写看房人数
   - 填写备注信息（选填）

3. **提交预约**
   - 点击**提交预约**按钮
   - 系统校验预约时间是否可约
   - 预约成功后发送短信通知客户

4. **查看预约记录**
   - 进入**我的预约**页面
   - 查看历史预约记录及状态
   - 支持取消未完成的预约

### （三）注意事项

- 同一时间段内预约人数有限制，约满后该时段不可选
- 预约成功后如需改期，需先取消原预约再重新预约
- 客户到访后，经纪人需在系统中确认到访

## 七、签到管理

### （一）操作前提

- 用户已登录小程序
- 已开启手机定位权限
- 在楼盘签到范围内

### （二）操作步骤

1. **扫码签到**
   - 在首页点击**扫码签到**
   - 扫描楼盘现场签到二维码
   - 系统自动记录签到位置和时间

2. **签到记录查看**
   - 进入**签到管理**页面
   - 查看历史签到记录
   - 支持按日期、楼盘筛选

3. **签到统计**（店长/管理员）
   - 查看门店人员签到统计
   - 导出签到报表

### （三）注意事项

- 签到需在楼盘指定范围内方可成功
- 每日可多次签到，系统记录首次和末次签到时间
- 签到数据用于考勤统计和业绩核算

## 八、我的

### （一）功能概述

"我的"页面展示用户个人信息、业绩数据、系统设置等功能入口。

### （二）操作步骤

1. **个人信息管理**
   - 点击头像进入**编辑信息**页面
   - 可修改头像、昵称、手机号码
   - 查看认证状态（未认证/认证中/已认证）

2. **业绩查看**
   - 点击**我的业绩**查看推荐业绩统计
   - 查看本月/本季度/本年度业绩数据
   - 查看佣金结算记录

3. **银行卡管理**
   - 进入**我的银行卡**页面
   - 添加/删除银行卡
   - 设置默认收款卡

4. **系统设置**
   - **偏好设置**：设置消息通知、隐私选项
   - **微信设置**：绑定/解绑微信
   - **关于我们**：查看版本信息、用户协议、隐私政策
   - **退出登录**：安全退出账号

### （三）注意事项

- 银行卡信息需与实名认证信息一致
- 佣金结算默认发放至默认收款卡
- 退出登录后需重新验证身份方可使用

## 九、消息通知

### （一）功能概述

系统通过消息中心向用户推送业务通知、系统公告等信息。

### （二）操作步骤

1. **消息查看**
   - 首页点击消息图标进入**首页通知_待办**页面
   - 查看未读消息列表
   - 点击消息查看详情

2. **消息类型**
   - **业务通知**：客户状态变更、预约提醒、成交通知
   - **系统公告**：功能更新、活动通知
   - **待办提醒**：待跟进客户、待确认预约

3. **消息设置**
   - 在**偏好设置**中设置消息推送开关
   - 可选择接收/屏蔽特定类型消息

### （三）注意事项

- 重要业务消息（如客户成交）会同时发送短信通知
- 消息保留期限为90天，过期自动清理

# 权限说明

| 角色 | 权限范围 |
| --- | --- |
| 普通用户 | 浏览楼盘、预约看房、查看个人信息 |
| 经纪人 | 推荐客户、查看自己推荐的客户、客户跟进、预约看房、签到、查看个人业绩 |
| 店长 | 查看门店全部客户、客户分配、查看门店业绩、签到统计、成员管理 |
| 管理员 | 全部功能权限，包括组织架构管理、权限配置、数据导出 |

# 配置说明

## 金刚位配置

在PC端管理系统【小程序配置】-【金刚位管理】中可配置：

- **图标顺序**：拖拽调整金刚位图标显示顺序
- **图标显示**：设置显示/隐藏特定功能入口
- **图标样式**：上传自定义图标（建议尺寸：88×88px）

## 楼盘推荐配置

- **推荐规则**：设置首页楼盘推荐算法（按热度/按最新/按距离）
- **置顶楼盘**：设置特定楼盘在首页置顶展示

## 客户保护期配置

- **保护期时长**：默认30天，可在后台调整
- **重复推荐规则**：设置是否允许跨门店重复推荐
