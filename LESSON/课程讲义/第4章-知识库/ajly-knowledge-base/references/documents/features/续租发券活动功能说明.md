# 续租发券活动功能说明

续租活动功能说明

# 功能概述

续租活动功能主要是通过配置活动规则、发放券等能力，实现用户在续约后自动发放对应优惠券。

# 新建活动

基本信息

活动名称将会展示在小程序页面，活动时间影响活动自动结束

活动规则

【适用续租周期】：可以从1个月选到12个月以上，可以多选，可以不选完，每个周期只能选一次

【适用预计居住时长】：如图所示可以选择7种不同的天数范围，不同续租周期可以选同样的预计居住时长，比如：续租周期为1个月的，可以选择居住时长为0-179天；点击“继续添加”，选择续租周期为2个月，也可以选择居住时长为0-179天。

【选择券】、【赠送量】的逻辑同其他营销活动，需要在优惠券模块中创建【活动专用】的优惠券。

按钮“继续添加”：可以添加不同续租周期的数据，最多可以添加12条。

发券规则：完成续约，并且缴纳续约的首期租金（完全缴纳），即按照用户的续租周期和累计居住时长自动发放优惠券。（企业包租合同不进行发券）

如果活动结束，那么这条活动下未发的优惠券都不发。

【续租还原】：如果续约后发券了，操作了续租还原，不会删除该优惠券，但会删除续租活动统计报表的数据；如果续约后还没交钱，操作了续租还原，会删除续租活动统计报表的数，后续也不会发券。

特殊逻辑：如果续约多次，在缴纳续约首期租金之后会发送多次优惠券（如果是多次续约，第一次续约的首期租金缴纳之后，会把每次续约对应的优惠券进行发放）；

如果续约多次后对某次续约记录进行还原，则会删除该续约记录对应的统计报表数据。
