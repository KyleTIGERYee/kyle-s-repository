# 员工业绩功能说明

员工业绩功能说明

# 权限说明

职位为管家的：只能查看自己的数据

其他职位，按【可看门店】查看数据，根据成交合同的门店进行查看，如：

## 某管家甲工作门店是门店A，但是他带客户在门店B成交了，门店B的店长只拥有门店B的可看门店权限。这时候，门店B的店长不仅能看到门店B的管家的成交业绩，也可以看到管家甲（属于门店A）的成交业绩。

# 基础计算方法

## 业绩=出铺单量*业绩系数，其中商铺出铺单量=  商铺的面积    /  配置参数X（在偏好配置中设置）

公寓：业绩系数=租期系数*合作系数*渠道系数*签约类型系数

商铺：业绩系数=租期系数*合作系数*渠道系数*签约类型系数

其中，企业包租不进行业绩计算

# 公寓偏好设置

偏好设置是第二天凌晨按最新的配置参数生效

各项系数，只能填入大于等于0的整数

租期系数：在配合配置中按照1个月到36个月进行配置，超过36个月的签约，租期系数取100%，换房转租等非完整月份的租期系数向下取值，如1月30天，则取1月

合作系数：新签的情况下，商机录入人和成交人分别按照此系数计算业绩，续转换等场景，只有成交人计算业绩。

渠道系数：成交渠道不同，渠道系数也不同。实时读取“来源渠道”字典中的渠道，续约时没有渠道，所以在这里单独设置了一个续约的渠道系数。

签约类型

签约类型解释

商铺出铺单量

## =  商铺的面积    /  配置参数X（在偏好配置中设置）

# 业绩计算的触发

当合同已经开始，并且已经缴纳首期租金，则触发业绩计算，具体如下：

已到合同开始时间+未交首期，实时计算业绩

已交首期+未到合同开始时间，在合同开始后的第二天，回归计算头一天的业绩

# 无责退房扣除业绩

无责退的退房日期，如果与产生业绩的日期在同一自然月时，会在无责退后对业绩进行扣除，会在成交明细中体现一条负数的本单业绩扣除项。

本月新签+续约的情况下，如果当月进行了无责退，则该新签和续约的业绩均会扣除（其中续约业绩暂未开始计算。

本月新签+换房的情况下，如果当月进行了无责退，则该新签和换房的业绩均会扣除

# 七日内新签的计算方法

合同结束日正负七天内新签，都算作续约。

当天+6天或者-6天，如某用户所有合同中，有一份合同结束日期是15号，则在9号到21号之间新签都算作续约。
