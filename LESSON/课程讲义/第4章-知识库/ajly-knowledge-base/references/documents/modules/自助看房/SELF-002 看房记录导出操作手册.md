## 一、功能概述

- **功能目的**: 将自助看房列表数据导出为 Excel 文件，便于线下分析和存档
- **使用场景**: 
  - 门店经理需要统计看房数据
  - 运营人员需要分析看房转化率
  - 定期导出看房记录进行归档
- **业务价值**: 支持数据导出功能，方便线下数据分析和报表制作

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 自助看房 → 导出按钮

| 数据列表：                                                |
| 战区 | 门店 | 物业地址 | 房间号 | 看房人姓名 |            |

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 战区 | 下拉选择 | 否 | 无 | 支持 20+ 选项（第一战区/第二战区/.../虚拟战区/寓见安居） |
| 门店 | 下拉选择 | 否 | 依赖战区 | 根据所选战区动态加载门店列表 |
| 物业地址 | 下拉选择 | 否 | 依赖门店 | 根据所选门店动态加载物业地址列表 |
| 房间号 | 文本 | - | - | 支持多房间显示如"101 102" |
| 看房人姓名 | 文本 | - | - | 预约看房的客户姓名 |
| 看房人手机号 | 文本 | - | 脱敏显示 | 中间 4 位隐藏如 131****5824 |
| 看房时间段 | 日期时间 | - | 格式 YYYY-MM-DD HH:mm-HH:mm | 预约的看房时间范围 |
| 看房状态 | 文本 | - | 枚举值 | 待看房/已看房/已过期/已失效 |
| 看房评价 | 文本 | - | - | 无评价显示"-" |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 筛选区右侧 | 导出当前筛选条件下的看房记录为 Excel 文件 | 自助看房模块访问权限 |
| 搜索 | 筛选区右侧 | 根据筛选条件查询数据 | 无特殊要求 |
| 重置 | 筛选区右侧 | 清空所有筛选条件 | 无特殊要求 |
| 偏好配置 | 页面顶部 | 配置自助看房偏好设置 | 配置权限 |
| 楼栋设置 | 页面顶部 | 配置自助看房楼栋范围 | 配置权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航至"营销推广 → 自助看房"
步骤 2: （可选）在筛选区选择战区/门店/物业地址进行筛选
步骤 3: 点击"导出"按钮
步骤 4: 系统弹出确认对话框"确认要导出吗？"
步骤 5: 点击"确定"按钮确认导出
步骤 6: 系统生成 Excel 文件并下载到本地
步骤 7: 导出完成后页面底部显示成功提示
```

### 4.2 流程图

```
开始
  ↓
进入自助看房页面
  ↓
是否需要筛选？
  ├─ 是 → 选择筛选条件 → 点击搜索
  └─ 否 → 直接导出
  ↓
点击"导出"按钮
  ↓
弹出确认对话框
  ↓
用户确认？
  ├─ 取消 → 关闭对话框 → 结束
  └─ 确定 → 执行导出
  ↓
系统生成 Excel 文件
  ↓
下载完成
  ↓
显示成功提示
  ↓
结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 无必填字段 | - | - |

**说明**: 导出功能无必填筛选条件，可直接导出全部数据

### 5.2 格式校验

| 校验项 | 校验规则 | 错误提示 | 示例 |
|--------|---------|---------|------|
| 数据存在性 | 当前筛选条件下必须有数据 | "没有可导出数据!" | 筛选条件过于严格导致无数据时 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 导出权限 | 用户必须具有导出权限 | 无权限时按钮不可用或提示无权限 |
| 二次确认 | 导出操作需要用户二次确认 | 弹出"确认要导出吗？"对话框 |
| 级联筛选 | 门店依赖战区，物业地址依赖门店 | 未选择上级条件时下拉框禁用 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 无符合条件的数据 | "没有可导出数据!" | 调整筛选条件或确认数据是否存在 |
| EX-002 | 网络超时 | "网络繁忙，请稍后再试！" | 检查网络连接后重试 |
| EX-003 | 无导出权限 | 按钮不可用或点击后提示无权限 | 联系管理员开通导出权限 |
| EX-004 | 导出文件过大 | 浏览器可能阻止下载或下载缓慢 | 建议增加筛选条件缩小数据范围 |
| EX-005 | 浏览器阻止下载 | 浏览器弹出下载拦截提示 | 在浏览器设置中允许该站点下载 |
| EX-006 | 数据加载失败 | "数据加载失败，请刷新页面" | 刷新页面后重试 |

## 九、注意事项

1. **导出文件大小**: 当数据量较大时（如 288 条），导出文件可能较大，下载需要一定时间
2. **筛选条件**: 建议先使用筛选条件缩小数据范围再导出，提高导出效率
3. **数据时效性**: 导出的数据为导出时刻的快照，不会自动更新
4. **手机号显示**: 列表页面手机号脱敏显示（131****5824），但导出文件中手机号完整显示
5. **浏览器兼容性**: 不同浏览器对下载的处理方式可能不同，请留意浏览器下载提示
