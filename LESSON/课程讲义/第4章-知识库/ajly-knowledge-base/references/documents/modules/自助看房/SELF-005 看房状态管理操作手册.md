## 一、功能概述

- **功能目的**: 查看和管理客户自助预约看房的状态，包括待看房、已看房、已过期、已失效四种状态
- **使用场景**: 
  - 查看当前所有待看房的预约记录
  - 跟踪已过期和已失效的看房预约
  - 统计各状态的看房记录数量
  - 筛选特定状态的看房记录进行分析
- **业务价值**: 帮助运营人员掌握看房预约的整体情况，及时跟进待看房客户，分析过期/失效原因优化预约流程

---

### 3.2 看房状态说明

| 状态   | 说明                               | 触发条件                      |
|------|----------------------------------|-----------------------------|
| 待看房 | 客户已预约，看房时间未到或刚到           | 预约成功且当前时间在看房时间段之前或之中   |
| 已看房 | 客户已完成看房                        | 系统自动标记或人工标记为已看房        |
| 已过期 | 看房时间已过，客户未看房                | 当前时间超过看房时间段结束时间        |
| 已失效 | 看房预约被取消或房源不可用              | 房源被下架/锁定/删除，或预约被取消    |

## 四、操作流程

### 4.1 查看待看房记录

```
步骤 1: 进入自助看房页面
步骤 2: 在筛选区选择战区/门店/物业地址（可选）
步骤 3: 点击"搜索"按钮
步骤 4: 在数据列表中查看"看房状态"列
步骤 5: 筛选出"待看房"状态的记录进行跟进
```

### 4.2 筛选特定状态

```
步骤 1: 进入自助看房页面
步骤 2: 查看数据列表中的"看房状态"列
步骤 3: 根据状态颜色/文字识别不同状态：
        - 待看房：等待客户看房
        - 已看房：客户已完成看房
        - 已过期：看房时间已过
        - 已失效：预约已取消或房源不可用
步骤 4: 可导出数据进行线下分析
```

### 4.3 流程图

```
开始
  ↓
进入自助看房页面
  ↓
查看数据列表
  ↓
看房状态列显示
  ├── 待看房 → 跟进客户按时看房
  ├── 已看房 → 记录看房结果
  ├── 已过期 → 分析过期原因
  └── 已失效 → 检查房源状态
  ↓
结束
```

## 五、校验规则

### 5.1 状态自动变更规则

| 状态变更       | 触发条件                      | 系统行为          |
|--------------|-----------------------------|-----------------|
| 待看房 → 已过期 | 当前时间 > 看房时间段结束时间      | 系统自动更新状态     |
| 待看房 → 已失效 | 房源被下架/锁定/删除，或预约被取消   | 系统自动更新状态     |
| 待看房 → 已看房 | 客户完成看房（系统自动或人工标记）    | 系统自动更新状态     |

### 5.2 列表显示规则

| 规则项     | 说明                          |
|---------|-----------------------------|
| 手机号脱敏  | 中间 4 位隐藏，如 131****5824     |
| 多房间显示  | 支持多房间显示，如"101 102"       |
| 无评价显示  | 看房评价为空时显示"-"              |
| 分页显示   | 每页约 20 条记录，支持页码跳转       |

### 5.3 业务校验

| 校验项     | 校验规则                      | 错误提示                 |
|---------|-----------------------------|----------------------|
| 权限校验  | 需营销推广管理权限               | 无权限时按钮不可见或点击无响应   |
| 数据加载  | 需网络连接正常                 | 网络超时显示加载失败        |
| 级联筛选  | 门店依赖战区，物业地址依赖门店       | 未选战区时门店下拉为空或禁用    |

## 六、异常场景

| 异常编号 | 异常场景          | 系统提示                  | 处理方式                    |
|-------|---------------|------------------------|-------------------------|
| EX-001 | 无符合条件的数据     | 列表显示为空                | 调整筛选条件或确认是否有预约数据    |
| EX-002 | 网络超时          | 加载失败或提示"网络繁忙，请稍后再试！" | 检查网络连接后重试             |
| EX-003 | 无操作权限        | 按钮不可见或点击无响应          | 联系管理员开通营销推广管理权限      |
| EX-004 | 数据加载失败       | 列表显示为空或错误提示          | 检查系统日志或联系技术支持        |
| EX-005 | 导出失败         | 提示导出失败或无响应           | 检查浏览器下载权限和网络连接       |
| EX-006 | 状态显示异常       | 状态与实际不符               | 刷新页面或联系技术支持核查数据     |
