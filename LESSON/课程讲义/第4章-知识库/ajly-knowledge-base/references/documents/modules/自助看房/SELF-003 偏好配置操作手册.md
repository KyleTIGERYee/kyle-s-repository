## 一、功能概述

- **功能目的**: 配置自助看房功能的偏好设置，控制同一楼栋的预约限制规则
- **使用场景**: 管理员需要设置用户在一定天数内是否可以对同一楼栋重复预约自助看房
- **业务价值**: 防止用户频繁预约同一楼栋，提高看房资源利用效率，减少无效预约

## 三、页面结构

### 3.1 页面布局

点击"偏好配置"按钮后弹出"偏好设置"对话框：

```
+----------------------------------------------------------+
| 偏好设置                                             [×]  |
+----------------------------------------------------------+
|                                                           |
|  预约限制同一楼栋                                         |
|  ┌─────┐ ┌─────┐  ┌─────────┐  天内不可以重复预约自助看房 |
|  │ 关  │ │ 开  │  │    1    │                            |
|  └─────┘ └─────┘  └─────────┘                            |
|                                                           |
+----------------------------------------------------------+
|                    [确认]    [取消]                       |
+----------------------------------------------------------+
```

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 预约限制同一楼栋 - 开关 | 开关控件 | 否 | 无 | 控制是否启用同一楼栋预约限制功能。开=启用限制，关=不限制 |
| 预约限制同一楼栋 - 天数 | 数字输入框 | 否 | 正整数（0-999） | 设置限制天数。当开关开启时，表示用户在设置天数内不可重复预约同一楼栋；当开关关闭时，值为 0，表示不限制 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 关 () | 对话框中部 | 关闭预约限制功能，点击后天数输入框自动变为 0 | 营销推广管理权限 |
| 开 () | 对话框中部 | 开启预约限制功能，可设置具体天数 | 营销推广管理权限 |
| 天数输入框 | 对话框中部 | 输入限制天数（0-999 的正整数） | 营销推广管理权限 |
| 确认 | 对话框底部 | 保存配置并关闭对话框，显示"设置成功!"提示 | 营销推广管理权限 |
| 取消 | 对话框底部 | 放弃修改并关闭对话框，不保存任何更改 | 营销推广管理权限 |
| Close (×) | 对话框右上角 | 关闭对话框，不保存任何更改 | 营销推广管理权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航至"营销推广 → 自助看房"
步骤 2: 点击页面顶部"偏好配置"按钮，弹出"偏好设置"对话框
步骤 3: 查看当前配置状态（开关状态和天数设置）
步骤 4: 根据业务需求调整配置：
        - 如需启用限制：点击"开"开关，在数字输入框输入限制天数
        - 如需关闭限制：点击"关"开关，天数自动变为 0
步骤 5: 点击"确认"按钮保存配置
步骤 6: 系统显示"设置成功!"提示，对话框关闭，返回自助看房列表页
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 无必填字段 | 无 | 所有字段均为可选 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 天数输入框 | 正整数（0-999） | 输入无效字符时无法输入 | 正确：1、7、30；错误：-1、abc、1.5 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 开关与天数联动 | 当开关切换到"关"时，天数自动变为 0 | 无提示，自动联动 |
| 权限校验 | 用户需具有营销推广管理权限 | 无权限时按钮不可点击或点击无响应 |
| 网络校验 | 保存时需要网络连接 | 网络超时可能显示"网络繁忙，请稍后再试！" |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 网络超时 | "网络繁忙，请稍后再试！" | 检查网络连接后重新尝试保存 |
| EX-002 | 无操作权限 | 按钮不可点击或点击无响应 | 联系管理员分配营销推广管理权限 |
| EX-003 | 配置数据加载失败 | 对话框打开后显示默认值或空值 | 刷新页面后重新尝试，如仍失败联系技术支持 |
| EX-004 | 输入非数字字符 | 输入框不接受非数字字符输入 | 仅输入 0-9 的数字字符 |
| EX-005 | 输入负数 | 输入框不接受负号 | 仅输入 0 及以上的正整数 |

## 七、关联影响

### 7.1 影响模块

- 自助看房查询
- 自助看房楼栋设置
- 客户预约小程序端（用户端预约时会校验此配置）

- 自助看房偏好配置表
- 用户预约记录（预约时会校验同一楼栋的限制规则）

### 7.3 后续操作

- 配置完成后，用户在小程序端预约自助看房时会受到限制规则约束
- 如需验证配置效果，可在小程序端尝试预约同一楼栋进行验证
- 定期检查配置是否满足业务需求，根据实际运营情况调整限制天数
