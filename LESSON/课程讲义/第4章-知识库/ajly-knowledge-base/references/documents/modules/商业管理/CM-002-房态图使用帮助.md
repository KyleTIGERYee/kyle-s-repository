# CM-002 房态图使用帮助

> **模块**: 商业管理 (Commercial Management)  
> **功能编号**: CM-002  
> **优先级**: P2  
> **完成日期**: 2026-03-21  
> **验证状态**: ✅ 已实际操作浏览器验证

---

## 一、功能概述

房态图使用帮助功能用于指导用户了解和使用商业管理模块的房态图界面。该功能通过功能收纳面板展示系统各模块的功能列表，帮助用户快速定位和了解各项功能的使用方式。

---

## 二、入口方式

### 2.1 导航路径
```
租赁管理 → 商业管理 → 房态图使用帮助
```

### 2.2 按钮位置
- 位于商业管理房态图页面顶部功能按钮区
- 在"登记项目"按钮右侧
- 按钮图标：📋（帮助图标）
- 按钮文字："房态图使用帮助"

## 四、操作步骤

### 4.1 查看房态图

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 进入商业管理页面 | 系统默认显示房态图界面 |
| 2 | 查看房态列表 | 显示所有商业项目的房间状态 |
| 3 | 查看房间详情 | 每个房间显示定价、状态、操作按钮 |

### 4.2 使用查询筛选

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 选择查询对象 | 切换"空间查询"或"项目查询"模式 |
| 2 | 选择店面 | 从下拉列表选择特定店面进行筛选 |
| 3 | 输入门牌号 | 可输入门牌号进行精确查找 |
| 4 | 点击搜索 | 执行筛选查询 |
| 5 | 点击重置 | 清空所有筛选条件 |
| 6 | 点击展开 | 显示更多筛选选项 |

### 4.3 使用房态筛选

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击"可租" | 筛选显示所有可租房间 |
| 2 | 点击"已订" | 筛选显示所有已预订房间 |

### 4.4 导出数据

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击"导出空间" | 导出当前筛选条件下的空间列表 |
| 2 | 点击"导出租客" | 导出当前筛选条件下的租客信息 |

### 4.5 查看统计数据

| 统计项 | 说明 |
|--------|------|
| 总空置 | 当前空置房间总数 |
| 项目冻结 | 被冻结的项目数量 |
| 项目已到期 | 合同已到期的项目数量 |
| 特价 | 设置特价的房间数量 |
| 短租 | 短租房间数量 |
| 未租 | 未出租房间数量 |
| 已订 | 已预订房间数量 |
| 已租 | 已出租房间数量 |
| 冻结 | 被冻结的房间数量 |
| 临时房 | 临时房间数量 |
| 脏房 | 标记为脏房的数量 |
| 租客已到期 | 租客合同已到期的数量 |

### 4.6 使用功能收纳

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击"房态图使用帮助" | 打开功能收纳面板 |
| 2 | 浏览功能列表 | 查看系统各模块功能 |
| 3 | 勾选/取消勾选 | 自定义显示的功能模块 |
| 4 | 点击关闭按钮 | 关闭功能收纳面板 |

## 五、字段说明

### 5.1 项目信息字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 项目名称 | 文本 | 商业项目的名称 |
| 项目地址 | 文本 | 项目所在详细地址 |
| 店铺信息 | 文本 | 所属店铺名称（如【职能部门店】） |
| 合同期限 | 数字 + 单位 | 项目合同期限（如 5 年、3 年） |
| 收房价格 | 货币 | 项目收房价格（元/月） |
| 装修状态 | 文本 | 项目装修状态（如未装修） |
| 欠款天数 | 数字 | 项目欠款天数 |
| 本套利润 | 货币 | 项目利润（可能为负值） |

### 5.2 房间信息字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 房间号 | 文本 | 房间编号（如 1 号房、2 号房） |
| 房间类型 | 文本 | 房间类型（如主卧、次卧、小卧、明隔、暗隔、商铺、公区场地、设备、广告等） |
| 定价 | 货币 | 房间租金定价（元/月） |
| 状态 | 文本 | 房间当前状态（空、已租、已订等） |
| 空置天数 | 数字 | 房间空置天数 |
| 租客姓名 | 文本 | 当前租客姓名（如已租出） |
| 租价 | 货币 | 实际出租价格 |
| 付款方式 | 文本 | 付款方式（如月付、季付） |
| 欠租天数 | 数字 | 租客欠租天数 |
| 到期提醒 | 文本 | 合同到期提醒（如快 15 天） |

### 5.3 缺失统计字段

| 字段名 | 说明 |
|--------|------|
| 无装修后图片 | 缺少装修后图片的房间数量 |
| 无证件照 | 缺少证件照的房间数量 |
| 无水底数 | 缺少水表底数的房间数量 |
| 无电底数 | 缺少电表底数的房间数量 |
| 无气底数 | 缺少气表底数的房间数量 |
| 无电子合同 | 缺少电子合同的房间数量 |
| 无电子交割单 | 缺少电子交割单的房间数量 |
| 无定价 | 缺少定价信息的房间数量 |

## 七、使用场景

### 7.1 日常房态查看
- **场景**: 管理员每日查看商业项目房态
- **操作**: 进入商业管理页面，查看房态列表和统计数据
- **目的**: 了解整体出租情况和空置房间

### 7.2 快速查找房间
- **场景**: 需要查找特定店面或门牌号的房间
- **操作**: 使用店面筛选和门牌号筛选功能
- **目的**: 快速定位目标房间

### 7.3 筛选可租房源
- **场景**: 为客户寻找可租的商业空间
- **操作**: 点击"可租"按钮筛选
- **目的**: 快速查看所有可租房间

### 7.4 导出房态数据
- **场景**: 需要制作房态报表或进行数据分析
- **操作**: 点击"导出空间"或"导出租客"
- **目的**: 获取结构化数据用于后续处理

### 7.5 查看缺失信息
- **场景**: 检查房间信息完整性
- **操作**: 查看右侧统计面板的"缺失统计"
- **目的**: 发现需要补充信息的房间

### 7.6 了解系统功能
- **场景**: 新用户熟悉系统功能
- **操作**: 点击"房态图使用帮助"查看功能收纳
- **目的**: 了解系统各模块功能

## 八、注意事项

### 8.1 数据实时性
- 房态数据实时同步，但可能存在短暂延迟
- 建议在重要操作前刷新页面确认最新状态

### 8.2 筛选条件组合
- 多个筛选条件可以组合使用
- 使用"展开"按钮可显示更多筛选选项

### 8.3 统计数据解读
- 统计数据为实时计算，反映当前系统状态
- "总空置"与"未租"可能有细微差别（未租包含更多状态）

### 8.4 导出功能
- 导出文件格式通常为 Excel
- 大量数据导出可能需要等待

### 8.5 权限控制
- 不同角色用户可能看到不同的操作按钮
- 部分功能可能需要特定权限
