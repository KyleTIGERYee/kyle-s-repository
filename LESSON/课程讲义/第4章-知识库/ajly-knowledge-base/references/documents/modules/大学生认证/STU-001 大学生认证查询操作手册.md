## 一、功能概述

- **功能目的**: 查询和管理大学生学籍认证信息，验证大学生身份享受租房优惠
- **使用场景**: 
  - 客服查询用户认证状态
  - 运营人员审核认证申请
  - 查看认证数据统计
- **业务价值**: 确保大学生优惠政策的准确执行，防止虚假认证

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 手机号 | 文本 | 否 | 11 位数字 | 支持模糊搜索，脱敏显示（中间 4 位隐藏） |
| 姓名 | 文本 | 否 | 2-10 个字符 | 支持模糊搜索 |
| 毕业生认证结果 | 下拉选择 | 否 | 选项：全部/待审核/已通过/人工审核通过/人工审核拒绝 | 筛选认证状态 |
| 学籍类别 | 单选按钮 | 否 | 选项：全部/境外学历/境内学历 | 按学籍类别筛选 |
| 学籍状态 | 文本 | 只读 | - | 显示在学/已毕业等状态 |
| 学校 | 文本 | 只读 | - | 用户所在学校名称 |
| 学历 | 文本 | 只读 | - | 专科/本科/硕士/博士 |
| 专业 | 文本 | 只读 | - | 用户所学专业 |
| 毕业时间 | 日期 | 只读 | YYYY-MM-DD | 预计或实际毕业日期 |
| 学历认证码/学生证号 | 文本 | 只读 | - | 学信网认证码或学生证号 |
| 附件 | 文件链接 | 只读 | - | 上传的证明材料，点击可查看 |
| 毕业生认证结果 | 文本 | 只读 | - | 自动审核结果 |
| 操作人 | 文本 | 只读 | - | 最后操作该记录的管理员 |
| 操作 | 按钮组 | - | - | 查看/审核等操作按钮 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 筛选区下方 | 导出当前筛选条件下的数据为 Excel 文件 | 导出权限 |
| 搜索 | 筛选区下方 | 执行筛选查询 | 查看权限 |
| 重置 | 筛选区下方 | 清空所有筛选条件 | 查看权限 |
| 查看全部 | 学籍类别区 | 显示所有学籍类别数据 | 查看权限 |
| 境外学历 | 学籍类别区 | 仅显示境外学历数据 | 查看权限 |
| 境内学历 | 学籍类别区 | 仅显示境内学历数据 | 查看权限 |
| 查看 | 操作列 | 查看详细认证信息 | 查看权限 |
| 审核 | 操作列 | 进入审核页面（待审核状态显示） | 审核权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录管理后台，点击快捷入口"大学生认证"
步骤 2: 进入大学生认证列表页，默认显示"大学生认证"标签页
步骤 3: （可选）在筛选区输入查询条件（姓名/手机号/认证结果）
步骤 4: （可选）选择学籍类别筛选（全部/境外学历/境内学历）
步骤 5: 点击"搜索"按钮执行查询
步骤 6: 查看数据列表，浏览认证信息
步骤 7: （可选）点击"导出"按钮导出 Excel 报表
步骤 8: （可选）点击"查看"按钮查看详细认证信息
```

## 五、校验规则

### 5.1 必填校验

无必填字段，所有筛选条件均为可选。

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 手机号 | 11 位数字 | "请输入正确的手机号" | 13800138000 |
| 姓名 | 2-10 个字符 | "姓名长度为 2-10 个字符" | 张三 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 搜索条件 | 至少输入一个筛选条件 | 无提示，返回全部数据 |
| 导出权限 | 用户需有导出权限 | "您没有导出权限" |
| 数据权限 | 只能查看权限范围内的数据 | "您没有查看权限" |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 无认证数据 | 列表显示"暂无数据" | 正常状态，无需处理 |
| EX-002 | 筛选无结果 | 列表显示"暂无数据" | 调整筛选条件后重新搜索 |
| EX-003 | 网络超时 | "网络繁忙，请稍后再试！" | 检查网络连接，稍后重试 |
| EX-004 | 无查看权限 | "您没有查看权限" | 联系管理员分配权限 |
| EX-005 | 无导出权限 | "您没有导出权限" | 联系管理员分配导出权限 |
| EX-006 | 导出失败 | "导出失败，请稍后再试" | 检查数据量是否过大，稍后重试 |
| EX-007 | 附件无法查看 | "文件不存在或已过期" | 联系用户上传新的附件 |
| EX-008 | 数据加载超时 | "数据加载超时，请刷新页面" | 刷新页面重新加载 |
