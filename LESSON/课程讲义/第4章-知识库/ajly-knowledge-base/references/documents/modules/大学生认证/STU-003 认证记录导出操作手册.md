# STU-003 认证记录导出操作手册

> **功能编号**: STU-003  
> **功能名称**: 认证记录导出  
> **所属模块**: 大学生认证  
> **操作频率**: 低频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 导出大学生认证记录数据为 Excel 文件，用于线下统计分析
- **使用场景**: 需要离线分析大学生认证数据、制作报表、数据备份
- **业务价值**: 支持数据导出需求，便于线下处理和存档

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 大学生认证 → 导出按钮

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 大学生认证                                                |
+----------------------------------------------------------+
| [大学生认证] [偏好配置]                                   |
+----------------------------------------------------------+
| 筛选区：                                                  |
| 姓名 | 手机号 | 毕业生认证结果 |                           |
| [导出] [搜索] [重置]                                      |
+----------------------------------------------------------+
| 学籍类别：[全部] [境外学历] [境内学历]                     |
+----------------------------------------------------------+
| 数据列表：                                                |
| 手机号 | 姓名 | 学籍类别 | 学籍状态 | 学校 | 学历 |       |
| 专业 | 毕业时间 | 学历认证码/学生证号 | 附件 |             |
| 毕业生认证结果 | 操作人 | 操作 |                           |
+----------------------------------------------------------+
| 分页：第 1 页/共 0 条 | 1 前往 [页] [确认]                  |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 已登录系统并具有大学生认证模块查看权限
- [ ] 系统中存在大学生认证数据（无数据时导出会提示"没有可导出数据!"）

---

## 三、页面结构

### 3.1 页面布局

导出功能通过页面顶部筛选区的"导出"按钮触发，点击后系统自动执行导出操作。

### 3.2 字段说明

导出文件包含以下字段（与列表字段一致）：

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 手机号 | 文本 | 用户手机号（完整显示，不脱敏） |
| 姓名 | 文本 | 用户姓名 |
| 学籍类别 | 文本 | 境外学历/境内学历 |
| 学籍状态 | 文本 | 学籍验证状态 |
| 学校 | 文本 | 毕业学校名称 |
| 学历 | 文本 | 学历层次（本科/硕士/博士等） |
| 专业 | 文本 | 所学专业 |
| 毕业时间 | 日期 | 毕业日期（YYYY-MM-DD） |
| 学历认证码/学生证号 | 文本 | 学历认证码或学生证号 |
| 附件 | 链接 | 学历证明文件附件 |
| 毕业生认证结果 | 文本 | 待人工核验/人工核验通过/人工核验拒绝/自动核验通过/自动核验拒绝 |
| 操作人 | 文本 | 最后操作该记录的管理员 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 筛选区右侧 | 导出当前筛选条件下的大学生认证记录 | 大学生认证查看权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航至 营销推广 → 大学生认证
步骤 2: （可选）在筛选区设置筛选条件（姓名/手机号/毕业生认证结果）
步骤 3: （可选）选择学籍类别（全部/境外学历/境内学历）
步骤 4: 点击"导出"按钮
步骤 5: 系统检查是否存在可导出数据
步骤 6: 如有数据，自动下载 Excel 文件；如无数据，显示"没有可导出数据!"提示
```

### 4.2 流程图

```
开始 → 进入大学生认证页面 → 设置筛选条件（可选）→ 点击导出按钮
         ↓
    系统检查数据
         ↓
    ┌────────────┐
    │ 有数据？   │
    └────────────┘
         ↓ 是              ↓ 否
    下载 Excel 文件    显示"没有可导出数据!"提示
         ↓                    ↓
       结束                结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 无 | - | 导出功能无需填写任何字段 |

### 5.2 格式校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 数据存在性 | 当前筛选条件下必须有数据 | "没有可导出数据!" |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 导出权限 | 用户必须具有大学生认证查看权限 | 无权限时按钮不可见或点击无响应 |
| 数据范围 | 仅导出当前筛选条件下的数据 | - |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 当前无符合条件数据 | "没有可导出数据!" | 调整筛选条件或确认数据是否存在 |
| EX-002 | 网络超时 | 导出失败（无响应或超时提示） | 检查网络连接，重新尝试导出 |
| EX-003 | 无导出权限 | 按钮不可见或点击无响应 | 联系管理员开通权限 |
| EX-004 | 导出文件过大 | 导出时间较长或失败 | 缩小筛选范围分批导出 |
| EX-005 | 浏览器阻止下载 | 下载被拦截 | 检查浏览器下载设置，允许下载 |
