## 一、功能概述

批量录入房间配置用于通过 Excel 文件批量设置多个房间的基础配置信息。适用于新项目初始化时批量配置房间属性（如房型、面积、朝向、配套设施等），可大幅提高配置效率。

## 五、操作流程

### 5.1 正常流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  步骤1   │ → │  步骤2   │ → │  步骤3   │ → │  步骤4   │
│ 下载模板 │    │ 填写配置 │    │ 上传文件 │    │ 确认导入 │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**详细步骤**:

**步骤1: 下载模板**
1. 点击"批量录入房间配置"按钮
2. 在弹窗中点击"下载"按钮
3. 保存模板文件到本地

**步骤2: 填写配置**
1. 打开下载的Excel模板
2. 按模板格式填写房间配置信息
3. 保存文件

**步骤3: 上传文件**
1. 点击"上传文件"按钮，或拖拽文件到上传区域
2. 选择填写好的Excel文件
3. 等待文件上传和校验

**步骤4: 确认导入**
1. 系统显示数据预览和校验结果
2. 确认无误后点击"确认导入"
3. 等待导入完成提示

### 5.2 异常流程

**场景1: 文件格式错误**
- **触发条件**: 上传非xls/xlsx格式的文件
- **系统提示**: "仅支持 xls、xlsx 格式文件"
- **处理方式**: 重新选择正确格式的文件

**场景2: 模板格式不正确**
- **触发条件**: Excel表格字段与模板不一致
- **系统提示**: "模板格式错误，请下载最新模板"
- **处理方式**: 重新下载模板，按规范填写

**场景3: 数据校验失败**
- **触发条件**: 数据中存在非法值（如面积为负数）
- **系统提示**: 显示具体错误行和错误原因
- **处理方式**: 修改Excel中的错误数据，重新上传

**场景4: 房间不存在**
- **触发条件**: Excel中的房间号在系统中不存在
- **系统提示**: "房间号不存在"
- **处理方式**: 核对房间号，确保与系统中登记的房号一致

## 八、关联影响

### 8.1 影响范围
- 修改后的配置会立即在房态图中显示
- 影响房间详情页展示的信息
- 影响后续的定价和租赁业务

### 8.2 数据一致性
- 导入操作会覆盖房间原有配置
- 建议在导入前导出当前配置作为备份
- 配置变更会影响历史数据统计口径

## 十、注意事项

1. **数据备份**: 批量导入前建议先导出当前房间配置数据作为备份
2. **格式检查**: 严格按照模板格式填写，不要修改表头
3. **数据验证**: 导入前仔细检查Excel中的数据准确性
4. **分批导入**: 数据量较大时建议分批导入，避免超时
5. **权限检查**: 确保操作账号具有批量导入权限
6. **房型匹配**: 房型必须与系统预设的房型选项完全一致

## 十二、常见问题

**Q1: 为什么导入后房型没有更新？**
A: 检查Excel中的房型名称是否与系统预设完全一致（注意空格、标点等）。

**Q2: 可以新增系统没有的房型吗？**
A: 不可以，批量导入只能使用系统预设的房型，如需新增房型请联系管理员。

**Q3: 导入会覆盖已有的配置吗？**
A: 是的，导入会覆盖原有配置，建议导入前备份数据。

**Q4: 导入失败如何查看具体原因？**
A: 系统会显示错误提示，可下载错误报告查看具体失败行和原因。
