> **操作频率**: 低频（集中入住场景使用）  
> **最后更新**: 2026-03-20

---

## 一、功能概述

批量录入租客用于通过 Excel 文件批量登记多个房间的租客信息。适用于集中入住场景（如企业员工集中入住、学校开学等），可大幅提高录入效率。

**限制条件**:
- 每次最多录入 5000 行数据
- 文件大小不超过 10M
- 模板中仅附带当前未租房间
- 需关闭偏好设置中的必填项和身份认证设置

## 五、操作流程

### 5.1 正常流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  步骤1   │ → │  步骤2   │ → │  步骤3   │ → │  步骤4   │ → │  步骤5   │
│ 关闭必填 │    │ 下载模板 │    │ 填写数据 │    │ 上传文件 │    │ 查看结果 │
│ 项设置   │    │          │    │          │    │          │    │          │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**详细步骤**:

**步骤1: 关闭必填项设置**
1. 进入 公寓偏好设置
2. 关闭"必填项设置"中的所有必选项
3. 关闭"身份证实名认证设置"中的【启用租客人脸身份认证设置】

**步骤2: 下载模板**
1. 点击"批量录入租客"按钮
2. 点击"下载【录入租客】模板"链接
3. 保存模板文件到本地

**步骤3: 填写数据**
1. 打开下载的Excel模板
2. 模板中已预填充所有未租房间信息
3. 按模板格式填写租客信息
4. 保存文件（确保不超过10M）

**步骤4: 上传文件**
1. 点击"点击上传"按钮
2. 选择填写好的Excel文件
3. 等待系统处理

**步骤5: 查看结果**
1. 导入完成后查看成功/失败数量
2. 如有失败记录，点击"下载错误房源"查看具体原因
3. 修正错误后重新上传

### 5.2 异常流程

**场景1: 文件大小超限**
- **触发条件**: 文件大小 > 10M
- **系统提示**: "文件大小不能超过10M"
- **处理方式**: 分批录入，或删除不必要的数据

**场景2: 数据行数超限**
- **触发条件**: 数据行数 > 5000
- **系统提示**: "每次最多录入5000行数据"
- **处理方式**: 分批导入，每次不超过5000行

**场景3: 必填项未关闭**
- **触发条件**: 偏好设置中必填项未关闭
- **系统提示**: 导入失败，提示缺少必填字段
- **处理方式**: 先关闭必填项设置，再重新导入

**场景4: 房间状态不符**
- **触发条件**: 房间已不是"未租"状态（被其他人登记）
- **系统提示**: 该房间已出租
- **处理方式**: 更新模板，重新下载获取最新未租房间列表

## 七、状态变更

| 操作 | 状态变更 | 说明 |
|------|----------|------|
| 导入成功 | 空房 → 已租 | 房间状态变为已租，绑定租客信息 |
| 导入失败 | 无变更 | 数据错误，不会修改房间状态 |

## 八、关联影响

### 8.1 影响范围
- 导入成功的房间状态变为"已租"
- 房间绑定租客信息（姓名、手机号等）
- 生成租客合同记录
- 影响财务应收款项

### 8.2 数据一致性
- 批量录入后，可在"租客登记"中查看导入的租客信息
- 导入失败的房间仍保持"未租"状态
- 建议导入后在房态图中核对结果

## 十一、注意事项

1. **前置设置**: 导入前务必关闭必填项和身份认证设置
2. **数据备份**: 建议导入前导出当前房态作为备份
3. **模板时效**: 模板中的未租房间列表有有效期，建议下载后立即使用
4. **并发控制**: 多人同时操作时可能出现房间被占用的情况
5. **失败处理**: 导入失败后及时下载错误文件查看原因
6. **分批导入**: 数据量较大时分批导入，降低失败风险

## 十三、常见问题

**Q1: 为什么导入总是失败？**
A: 检查是否已关闭必填项设置和身份认证设置。

**Q2: 模板中没有我需要的房间？**
A: 模板只包含当前未租房间，如果房间已被登记，需要等待退房后才能录入新租客。

**Q3: 导入成功后在哪里查看租客信息？**
A: 可在房态图中点击房间查看，或进入租客管理模块查看。

**Q4: 可以修改已导入的租客信息吗？**
A: 可以，通过"完善房间信息"功能修改租客信息。
