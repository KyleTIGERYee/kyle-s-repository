# AM-006 批量导入定价操作手册

> **功能编号**: AM-006  
> **功能名称**: 批量导入定价  
> **所属模块**: 公寓管理  
> **操作频率**: 中频（新增项目或调价时使用）  
> **最后更新**: 2026-03-20

---

## 一、功能概述

批量导入定价用于通过 Excel 文件批量修改多个房间的定价信息。适用于新项目批量设置定价、或整体调价场景，可大幅提高工作效率。

### 4.1 导入模板区域

| 字段编号 | 元素 | 说明 |

## 五、操作流程

### 5.1 正常流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  步骤1   │ → │  步骤2   │ → │  步骤3   │ → │  步骤4   │
│ 下载模板 │    │ 填写数据 │    │ 上传文件 │    │ 确认导入 │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**详细步骤**:

**步骤1: 下载模板**
1. 点击"批量导入定价"按钮
2. 在弹窗中点击"下载"按钮
3. 保存模板文件到本地

**步骤2: 填写数据**
1. 打开下载的Excel模板
2. 按模板格式填写房间定价信息
3. 保存文件

**步骤3: 上传文件**
1. 点击"上传文件"按钮，或拖拽文件到上传区域
2. 选择填写好的Excel文件
3. 等待文件上传和校验

**步骤4: 确认导入**
1. 系统显示数据预览和校验结果
2. 确认无误后点击"确认导入"
3. 等待导入完成提示

### 5.2 异常流程

**场景1: 文件格式错误**
- **触发条件**: 上传非xls/xlsx格式的文件
- **系统提示**: "仅支持 xls、xlsx 格式文件"
- **处理方式**: 重新选择正确格式的文件

**场景2: 模板格式不正确**
- **触发条件**: Excel表格字段与模板不一致
- **系统提示**: "模板格式错误，请下载最新模板"
- **处理方式**: 重新下载模板，按规范填写

**场景3: 数据校验失败**
- **触发条件**: 数据中存在非法值（如定价为负数）
- **系统提示**: 显示具体错误行和错误原因
- **处理方式**: 修改Excel中的错误数据，重新上传

## 六、Excel模板字段说明

| 字段名 | 必填 | 类型 | 说明 | 示例 |
|--------|------|------|------|------|
| 店面 | 是 | 文本 | 楼栋/项目名称 | 朝阳新村1幢 |
| 楼层 | 是 | 文本 | 楼层信息 | 1层 |
| 房间号 | 是 | 文本 | 房间门牌号 | 1-1 |
| 定价 | 是 | 数字 | 房间定价（元） | 1500 |

**数据格式要求**:
- 定价必须为正整数或正小数
- 房间号必须与系统中已存在的房间匹配
- 不要修改模板表头名称

## 七、状态变更

| 操作 | 状态变更 | 说明 |
|------|----------|------|
| 导入成功 | 房间定价更新 | 批量修改房间定价 |
| 导入失败 | 无变更 | 数据错误，不会修改任何房间定价 |

---

## 八、关联影响

### 8.1 影响范围
- 修改后的定价会立即在房态图中显示
- 影响后续的租客登记时显示的定价
- 影响导出的房间数据中的定价字段

### 8.2 数据一致性
- 导入操作会覆盖房间原有定价
- 建议在导入前导出当前定价作为备份

## 十、注意事项

1. **数据备份**: 批量导入前建议先导出当前定价数据作为备份
2. **格式检查**: 严格按照模板格式填写，不要修改表头
3. **数据验证**: 导入前仔细检查Excel中的数据准确性
4. **分批导入**: 数据量较大时建议分批导入，避免超时
5. **权限检查**: 确保操作账号具有批量导入权限
