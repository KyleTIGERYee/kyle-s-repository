# AM-004 退房操作手册

> **功能编号**: AM-004  
> **功能名称**: 退房（租客退租）  
> **所属模块**: 公寓管理  
> **操作频率**: 中频  
> **最后更新**: 2026-03-20

---

## 一、功能概述

退房功能用于办理租客退租手续，完成费用结算、房间验收、状态变更等操作。这是公寓租赁业务中的重要环节，涉及财务结算、房间交接、数据统计等多个方面。

**业务流程**: 租客申请退房 → 费用结算 → 房间验收 → 状态变更为"空"

## 七、操作流程

### 7.1 正常退房流程
```
┌─────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────┐
│ 租客    │ → │ 点击房间    │ → │ 填写退房    │ → │ 费用    │
│ 申请    │    │ 【退房】按钮│    │ 信息并结算  │    │ 结算    │
└─────────┘    └─────────────┘    └─────────────┘    └─────────┘
                                                          ↓
┌─────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────┐
│ 房间    │ ← │ 确认退房    │ ← │ 上传照片    │ ← │ 房间    │
│ 状态变更│    │ 完成        │    │ 和交割单    │    │ 验收    │
└─────────┘    └─────────────┘    └─────────────┘    └─────────┘
```

### 7.2 详细步骤
1. 在公寓管理页面找到已租房间
2. 点击房间卡片的【退房】按钮
3. 在"退房信息"Tab填写：
   - 确认退房性质
   - 选择实际退房时间
   - 选择退房原因
   - 填写详细说明
4. 核对费用结算：
   - 确认应退金额
   - 确认应扣金额
   - 录入能源费底数（或点击"一键抄表"）
   - 添加其他应扣费用（如有）
5. 填写交割信息（水/电/热水止数）
6. 选择退房评价（好评/差评）
7. 填写备注信息
8. 切换到"租客信息"Tab核对信息
9. 切换到"退房照片"Tab上传验收照片
10. 切换到"电子合同及交割单"Tab确认文档
11. 点击【确定】完成退房

## 九、异常场景

### 9.1 费用计算异常
- **触发条件**: 应扣金额大于应退金额
- **系统响应**: 显示负数结算金额
- **处理方式**: 确认费用明细是否正确

### 9.2 能源费底数异常
- **触发条件**: 本次底数小于上次底数
- **系统响应**: 提示底数错误
- **处理方式**: 重新核对并输入正确底数

### 9.3 必填字段未填
- **触发条件**: 点击确定时必填字段为空
- **系统响应**: 字段边框变红，提示错误
- **处理方式**: 补充必填信息后重新提交

## 十三、注意事项

### 13.1 重要提醒
- ⚠️ **费用核对**: 退房前务必核对所有费用明细
- ⚠️ **能源底数**: 抄表时确保读数准确
- ⚠️ **照片留档**: 退房验收照片必须上传

### 13.2 业务规则
- 退房后房间状态立即变为"空"
- 退房结算金额正负均可（正=退给租客，负=租客补交）
- 能源费未结算无法完成退房
