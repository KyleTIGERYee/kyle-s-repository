## 一、功能概述

批量操作确认流程是公寓管理模块中所有批量导入功能的通用操作流程。包括：
- 批量导入定价（AM-006）
- 批量录入租客（AM-007）
- 批量录入房间配置（AM-008）

本流程文档记录批量操作的通用步骤、确认机制和注意事项。

## 四、批量操作通用流程

### 4.1 标准操作流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  步骤 1   │ → │  步骤 2   │ → │  步骤 3   │ → │  步骤 4   │ → │  步骤 5   │
│  下载模板 │    │  填写数据 │    │  上传文件 │    │  系统校验 │    │  确认导入 │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

**详细步骤**:

**步骤 1: 下载模板**
1. 点击对应的批量操作按钮（如"批量导入定价"）
2. 页面底部展开批量导入区域
3. 点击"导入模板"区域的下载按钮
4. 保存 Excel 模板文件到本地

**步骤 2: 填写数据**
1. 打开下载的 Excel 模板
2. 模板中已预填充相关基础数据（如房间号、楼栋信息等）
3. 按模板格式填写需要导入的数据
4. 保存文件（确保文件格式为 .xls 或 .xlsx）

**步骤 3: 上传文件**
1. 点击"上传文件"按钮
2. 选择填写好的 Excel 文件
3. 或将文件直接拖拽到上传区域

**步骤 4: 系统校验**
1. 系统自动校验文件格式
2. 系统校验数据格式和完整性
3. 显示校验结果（成功/失败记录数）

**步骤 5: 确认导入**
1. 查看校验结果
2. 确认导入数据
3. 系统执行批量导入操作
4. 显示导入结果

### 4.2 异常流程

**场景 1: 文件格式错误**
- **触发条件**: 上传非 xls/xlsx 格式文件
- **系统提示**: "仅支持 xls、xlsx 格式文件"
- **处理方式**: 转换文件格式后重新上传

**场景 2: 数据格式错误**
- **触发条件**: Excel 中数据格式不符合要求
- **系统提示**: 显示具体错误行和错误原因
- **处理方式**: 修正数据后重新上传

**场景 3: 必填字段缺失**
- **触发条件**: 模板中必填字段未填写
- **系统提示**: 显示缺失字段的行号
- **处理方式**: 补充必填字段后重新上传

**场景 4: 数据重复**
- **触发条件**: 导入的数据与现有数据冲突
- **系统提示**: 显示冲突的记录
- **处理方式**: 删除或修改重复数据后重新上传

## 六、校验规则

### 6.1 文件格式校验

| 校验项 | 规则 | 错误提示 |
|--------|------|----------|
| 文件扩展名 | 必须为 .xls 或 .xlsx | "仅支持 xls、xlsx 格式文件" |
| 文件大小 | 不超过 10MB | "文件大小不能超过 10M" |

### 6.2 数据格式校验

| 校验项 | 规则 | 错误提示 |
|--------|------|----------|
| 手机号 | 11 位数字 | "手机号格式错误" |
| 身份证号 | 18 位（如填写） | "身份证号格式错误" |
| 日期格式 | YYYY-MM-DD | "日期格式错误" |
| 数字字段 | 必须为有效数字 | "数字格式错误" |
| 必填字段 | 不能为空 | "必填字段不能为空" |

### 6.3 业务规则校验

| 校验项 | 规则 | 错误提示 |
|--------|------|----------|
| 房间存在性 | 房间必须存在于系统中 | "房间不存在" |
| 房间状态 | 房间状态必须符合操作要求 | "房间状态不符" |
| 数据重复 | 不能导入重复数据 | "数据重复" |

## 七、导入结果处理

### 7.1 成功导入

**显示内容**:
- 成功导入记录数
- 导入时间
- 受影响的数据范围

**后续操作**:
- 在列表中查看导入的数据
- 验证数据是否正确
- 如有问题及时修正

### 7.2 部分失败

**显示内容**:
- 成功记录数
- 失败记录数
- 失败原因详情

**处理方式**:
1. 点击下载错误文件
2. 查看具体失败原因
3. 修正错误数据
4. 重新上传修正后的文件

### 7.3 全部失败

**显示内容**:
- 失败记录数
- 失败原因汇总

**处理方式**:
1. 检查文件格式是否正确
2. 检查数据格式是否符合要求
3. 检查前置条件是否满足
4. 修正后重新上传

## 八、注意事项

1. **模板时效性**: 下载的模板有时效性，建议下载后立即使用
2. **数据备份**: 批量操作前建议导出当前数据作为备份
3. **分批导入**: 数据量较大时建议分批导入，降低失败风险
4. **并发控制**: 多人同时操作时可能出现数据冲突
5. **前置设置**: 部分批量操作需要先调整偏好设置
6. **格式要求**: 严格按照模板格式填写，不要修改模板结构
7. **文件限制**: 文件大小不超过 10MB，格式为 xls 或 xlsx

## 十一、常见问题

**Q1: 为什么上传文件后没有反应？**
A: 检查文件格式是否为 xls 或 xlsx，文件大小是否超过 10MB。

**Q2: 导入失败后如何查看错误原因？**
A: 点击"下载错误文件"按钮，查看具体的错误行和错误原因。

**Q3: 可以取消正在进行的导入吗？**
A: 导入开始后无法取消，请等待导入完成。

**Q4: 批量导入的数据在哪里查看？**
A: 导入完成后，在对应的列表页面查看导入的数据。

**Q5: 导入的数据可以撤销吗？**
A: 批量导入不支持撤销，建议导入前做好数据备份。
