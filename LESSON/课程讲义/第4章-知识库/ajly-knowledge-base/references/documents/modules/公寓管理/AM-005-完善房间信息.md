# AM-005 完善房间信息操作手册

> **功能编号**: AM-005  
> **功能名称**: 完善房间信息（租客信息编辑）  
> **所属模块**: 公寓管理  
> **操作频率**: 中频  
> **最后更新**: 2026-03-20

---

## 一、功能概述

完善房间信息功能用于编辑和补充已租房间的租客信息、合同信息、押金信息等。当租客信息需要修改或补充时，通过此功能进行更新。

**使用场景**:
- 租客信息有误需要修改
- 补充租客紧急联系人信息
- 修改能源费底数
- 更新备注信息

---

## 六、操作流程

### 6.1 正常流程
```
┌─────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────┐
│ 发现    │ → │ 点击房间    │ → │ 修改/补充   │ → │ 保存    │
│ 信息需  │    │ 【完善】按钮│    │ 租客信息    │    │ 完成    │
│ 修改    │    │             │    │             │    │         │
└─────────┘    └─────────────┘    └─────────────┘    └─────────┘
```

### 6.2 详细步骤
1. 在公寓管理页面找到已租房间
2. 点击房间卡片的【完善】按钮
3. 在弹窗中修改需要更新的信息
4. 点击【保存】按钮提交修改

## 八、异常场景

### 8.1 必填字段未填
- **触发条件**: 点击保存时必填字段为空
- **系统响应**: 字段边框变红，提示错误信息
- **处理方式**: 补充必填信息后重新保存

### 8.2 手机号格式错误
- **触发条件**: 输入非11位手机号
- **系统响应**: 提示"请输入有效的手机号码"
- **处理方式**: 输入正确的11位手机号

### 8.3 合同时间错误
- **触发条件**: 结束时间早于开始时间
- **系统响应**: 提示合同时间错误
- **处理方式**: 调整开始或结束时间
