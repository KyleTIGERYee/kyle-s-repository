## 一、功能概述

- **功能目的**: 控制续租活动的上线和下线状态，管理活动在小程序端的可见性
- **使用场景**: 
  - 活动配置完成后发布到小程序端供用户参与
  - 活动需要提前下架或暂停时取消发布
  - 活动到期后自动结束
- **业务价值**: 确保活动按时上线和下线，避免过期活动继续展示，保障营销活动生命周期管理

## 四、操作流程

### 4.1 发布操作流程

```
步骤 1: 登录系统，导航到"营销推广 → 营销活动 → 续租活动"
步骤 2: 在列表中找到状态为"未发布"的目标活动
步骤 3: 点击该活动行操作列的"发布"按钮
步骤 4: 系统直接执行发布操作（无需确认对话框）
步骤 5: 页面刷新，活动状态变为"已发布"
步骤 6: 操作按钮变为"详情/表单统计/取消发布"
步骤 7: 新增"复制"按钮，可复制活动链接分享到小程序
```

### 4.2 取消发布操作流程

```
步骤 1: 登录系统，导航到"营销推广 → 营销活动 → 续租活动"
步骤 2: 在列表中找到状态为"已发布"的目标活动
步骤 3: 点击该活动行操作列的"取消发布"按钮
步骤 4: 系统弹出确认对话框："确定取消发布吗"
步骤 5: 点击"确定"按钮确认操作
步骤 6: 页面刷新，活动状态变为"已取消"
步骤 7: 操作按钮变为"详情/表单统计/编辑/发布"
步骤 8: "复制"按钮消失
```

## 五、校验规则

### 5.1 发布前校验

| 校验项        | 校验规则                              | 错误提示                       | 触发时机   |
| ----------- | --------------------------------- | -------------------------- | ------ |
| 活动配置完整性   | 活动名称、活动时间、关联券配置必须完整            | 活动配置不完整，无法发布              | 点击发布时 |
| 活动时间校验    | 活动结束时间必须晚于开始时间                 | 活动时间设置不正确                | 点击发布时 |
| 关联券有效性    | 关联的优惠券必须为有效状态（未下架、未过期）       | 关联优惠券已失效，请重新配置          | 点击发布时 |
| 奖池配置校验    | 至少配置一个续租周期和对应优惠券              | 请至少配置一个续租周期和优惠券         | 点击发布时 |
| 权限校验      | 用户必须具有营销活动发布权限                 | 无权限执行此操作                 | 点击发布时 |

### 5.2 取消发布前校验

| 校验项     | 校验规则                    | 错误提示          | 触发时机   |
| -------- | ----------------------- | ------------- | ------ |
| 状态校验   | 只有"已发布"状态的活动可取消发布    | 该活动不可取消发布    | 点击取消时 |
| 权限校验   | 用户必须具有营销活动管理权限       | 无权限执行此操作     | 点击取消时 |

### 5.3 业务校验

| 校验项        | 校验规则                                   | 错误提示                    |
| ----------- | -------------------------------------- | ----------------------- |
| 活动进行中校验  | 已有用户参与的活动取消发布后，已参与用户数据保留，但新用户无法再参与 | 取消发布后新用户无法参与，已参与数据保留 |
| 时间校验     | 已过期的活动自动变为"已结束"状态，不可手动取消发布       | 活动已过期，无法取消发布         |

## 六、异常场景

| 异常编号  | 异常场景                      | 系统提示                      | 处理方式                       |
| ------ | ------------------------- | ------------------------- | -------------------------- |
| EX-001 | 活动配置不完整时尝试发布             | 活动配置不完整，无法发布            | 检查并完善活动配置后重新发布          |
| EX-002 | 关联优惠券已下架或过期时尝试发布         | 关联优惠券已失效，请重新配置          | 更换有效优惠券或重新创建活动          |
| EX-003 | 活动时间设置不正确（结束时间早于开始时间）   | 活动时间设置不正确               | 修正活动时间后重新发布             |
| EX-004 | 无权限执行发布/取消发布操作          | 无权限执行此操作                | 联系管理员开通营销活动管理权限        |
| EX-005 | 网络请求失败                   | 网络异常，请稍后重试              | 检查网络连接后重试                |
| EX-006 | 活动已过期时尝试取消发布             | 活动已过期，无法取消发布            | 已过期的活动自动变为"已结束"状态，无需操作 |
| EX-007 | 取消发布确认对话框点击"取消"          | 对话框关闭，操作取消               | 如需取消发布需重新点击"取消发布"按钮   |
| EX-008 | 数据加载超时                   | 数据加载超时，请刷新页面重试         | 刷新页面后重试                  |

## 七、关联影响

### 7.1 影响模块

- **小程序前端**: 发布后活动对用户可见，取消发布后活动隐藏
- **优惠券管理**: 发布时校验关联优惠券有效性
- **数据统计**: 发布后开始统计活动参与数据
- **续租记录查询**: 发布后产生的续租记录可在查询功能中查看
- **续租效果统计**: 发布后产生的续租数据纳入统计

### 7.2 影响数据

- **活动状态表**: 更新活动状态字段（未发布→已发布→已取消）
- **活动发布时间**: 记录发布操作时间
- **活动取消时间**: 记录取消发布时间
- **操作日志**: 记录发布/取消发布操作人和时间

### 7.3 后续操作

- **发布后**: 
  - 可复制链接分享到小程序
  - 可查看表单统计和续租效果
  - 可监控活动参与情况
- **取消发布后**:
  - 可编辑活动配置
  - 可重新发布活动
  - 已参与用户数据保留

## 九、注意事项

### 9.1 发布注意事项

1. **发布前检查**: 确保活动配置完整，关联优惠券有效
2. **发布时间**: 建议在活动正式开始前 1-2 小时发布，预留检查时间
3. **复制链接**: 发布后可复制链接分享到小程序或社群
4. **状态确认**: 发布后刷新页面确认状态已变为"已发布"

### 9.2 取消发布注意事项

1. **二次确认**: 取消发布需点击确认对话框的"确定"按钮
2. **数据保留**: 取消发布后已参与用户数据保留，仅新用户无法参与
3. **重新发布**: 取消发布的活动可编辑后重新发布
4. **影响评估**: 取消发布前评估对已参与用户的影响

### 9.3 状态说明

- **未发布**: 活动仅管理员可见，小程序端用户不可见
- **已发布**: 活动在小程序端可见，用户可参与
- **已取消**: 活动下架，小程序端隐藏，但可重新编辑发布
- **已结束**: 活动时间过期自动结束，不可重新发布
