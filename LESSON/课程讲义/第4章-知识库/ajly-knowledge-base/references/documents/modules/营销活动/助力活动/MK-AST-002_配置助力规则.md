# MK-AST-002 配置助力规则 操作手册

> **功能编号**: MK-AST-002  
> **功能名称**: 配置助力规则  
> **所属模块**: 营销活动 - 助力活动  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 配置助力活动的参与规则，包括参与人资格、助力人资格、关联优惠券等核心规则
- **使用场景**: 创建助力活动后，需要设置谁可以参与、谁可以助力、助力成功后发放什么奖励
- **业务价值**: 通过灵活的规则配置，实现精准营销，控制活动成本和覆盖人群

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 营销活动 → 助力活动 → 编辑（未发布活动）

### 2.2 按钮位置（ASCII 结构图）

```

+----------------------------------------------------------+
| 筛选区：                                                  |

| [搜索] [重置] [新建活动] [导出]                           |
+----------------------------------------------------------+
| 数据列表：                                                |
| 序号 | 活动名称 | 活动时间 | 活动状态 | 创建时间 |        |

| 操作：[详情] [表单统计] [发布] [取消发布] [编辑] ← 点击此处  |

```

### 2.3 前置条件

- [ ] 已创建助力活动（MK-AST-001）
- [ ] 活动状态为"未发布"或"已结束"

### 3.2 字段说明

#### 一、参与人资格

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 参与人类型 | 单选框 | 是 | 必选一项 | 所有人/指定公司 |
| 指定公司 | 公司选择器 | 条件必填 | 参与人类型=指定公司时必填 | 可多选 |
| 租客状态 | 单选框 | 是 | 必选一项 | 在租/新客/已注册 |
| 同住人要求 | 单选框 | 是 | 必选一项 | 无要求/需有同住人 |

#### 二、助力人资格

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 助力人类型 | 单选框 | 是 | 必选一项 | 所有人/指定公司 |
| 指定公司 | 公司选择器 | 条件必填 | 助力人类型=指定公司时必填 | 可多选 |
| 助力次数限制 | 数字输入框 | 是 | 正整数，1-999 | 每人最多助力次数 |

#### 三、关联券配置

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 助力成功奖励 | 优惠券选择器 | 是 | 必选 | 选择已创建的优惠券模板 |
| 券名称 | 文本（只读） | - | - | 自动显示所选券名称 |
| 券类型 | 文本（只读） | - | - | 自动显示券类型 |
| 发放数量 | 数字输入框 | 是 | 正整数，1-99999 | 活动总发放数量限制 |

#### 四、活动页配置

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动规则文本 | 多行文本框 | 是 | 1-500 字 | 展示给用户的活动规则说明 |
| 背景图 | 图片上传 | 否 | JPG/PNG，≤5MB | 活动页面背景图 |
| 分享卡片图 | 图片上传 | 否 | JPG/PNG，≤5MB | 分享时显示的卡片图 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 进入营销活动 → 助力活动列表页
步骤 2: 找到未发布的活动，点击"编辑"按钮
步骤 3: 切换到"活动规则"标签页
步骤 4: 配置参与人资格（选择参与人类型、租客状态等）
步骤 5: 配置助力人资格（选择助力人类型、助力次数限制）
步骤 6: 配置关联券（选择优惠券模板、设置发放数量）
步骤 7: 配置活动页（填写规则文本、上传图片、选择按钮颜色）
步骤 8: 点击"保存"按钮
步骤 9: 系统提示"保存成功"，返回列表页
```

| 同住人要求 | 请选择同住人要求 | 提交时 |
| 助力人类型 | 请选择助力人类型 | 提交时 |
| 助力次数限制 | 请输入助力次数限制 | 提交时 |
| 助力成功奖励 | 请选择助力成功奖励 | 提交时 |
| 发放数量 | 请输入发放数量 | 提交时 |
| 活动规则文本 | 请输入活动规则文本 | 提交时 |
| 按钮颜色 | 请选择按钮颜色 | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 助力次数限制 | 正整数，1-999 | 请输入 1-999 之间的数字 | 5 |
| 发放数量 | 正整数，1-99999 | 请输入 1-99999 之间的数字 | 1000 |
| 活动规则文本 | 1-500 字 | 活动规则文本长度为 1-500 字 | "邀请好友助力，双方均可获得优惠券" |
| 背景图 | JPG/PNG，≤5MB | 请上传 JPG 或 PNG 格式图片，大小不超过 5MB | - |
| 分享卡片图 | JPG/PNG，≤5MB | 请上传 JPG 或 PNG 格式图片，大小不超过 5MB | - |
| 按钮颜色 | 十六进制颜色码 | 请选择有效的颜色 | #6B46C1 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 参与人类型与指定公司 | 参与人类型=指定公司时，必须选择至少一个公司 | 请选择指定公司 |
| 助力人类型与指定公司 | 助力人类型=指定公司时，必须选择至少一个公司 | 请选择指定公司 |
| 优惠券状态 | 选择的优惠券必须为可用状态 | 所选优惠券不可用，请选择其他优惠券 |
| 发放数量与库存 | 发放数量不能超过优惠券可用数量 | 发放数量超过优惠券可用数量 |
