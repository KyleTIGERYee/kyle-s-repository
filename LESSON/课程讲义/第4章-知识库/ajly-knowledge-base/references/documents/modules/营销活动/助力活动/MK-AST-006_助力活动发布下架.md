# MK-AST-006 助力活动发布/下架 操作手册

> **功能编号**: MK-AST-006  
> **功能名称**: 助力活动发布/下架  
> **所属模块**: 营销推广 → 营销活动 → 助力活动  
> **操作频率**: 低频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 控制助力活动的上线和下线状态，管理活动的可见性和参与性
- **使用场景**: 活动筹备完成后发布上线，或活动需要提前结束/暂停时下架
- **业务价值**: 灵活控制活动状态，确保活动按计划执行，避免意外参与

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 营销活动 → 助力活动 → 选择活动 → 点击"发布"或"取消发布"按钮

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 营销活动 → 助力活动                                       |
+----------------------------------------------------------+
| 筛选区：                                                  |
| 活动名称 | 活动时间 (起 - 止) | 活动状态 |                   |
| [搜索] [重置] [新建活动] [导出]                           |
+----------------------------------------------------------+
| 数据列表：                                                |
| 序号 | 活动名称 | 活动时间 | 活动状态 | 创建时间 |        |
| 表单总数 | 成交总数 | 活动链接 | 操作 |                    |
| 操作：[详情] [表单统计] [发布] [取消发布] [编辑]            |
|                        ↑          ↑                        |
|                   未发布时显示   已发布时显示               |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 已创建助力活动（MK-AST-001）
- [ ] 活动配置已完成（MK-AST-002）
- [ ] 拥有营销活动发布权限

## 三、页面结构

### 3.1 页面布局

助力活动列表操作列按钮状态：

```
+----------------------------------------------------------+
| 活动状态：未发布                                          |
| 操作按钮：[编辑] [发布]                                   |
+----------------------------------------------------------+
| 活动状态：已发布                                          |
| 操作按钮：[详情] [表单统计] [取消发布] [编辑]              |
+----------------------------------------------------------+
| 活动状态：已结束                                          |
| 操作按钮：[详情] [表单统计]                               |
+----------------------------------------------------------+
| 活动状态：已取消                                          |
| 操作按钮：[详情] [表单统计]                               |
+----------------------------------------------------------+
```

### 3.2 活动状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|-----------|
| 未发布 | 活动已创建但未上线，用户无法参与 | 编辑、发布、删除 |
| 已发布 | 活动正在运行，用户可正常参与 | 详情、表单统计、取消发布、编辑 |
| 已结束 | 活动时间到期自动结束 | 详情、表单统计 |
| 已取消 | 活动被手动取消 | 详情、表单统计 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 发布 | 操作列（未发布状态） | 将活动状态变更为"已发布"，活动上线 | 营销活动发布权限 |
| 取消发布 | 操作列（已发布状态） | 将活动状态变更为"已取消"，活动下线 | 营销活动发布权限 |
| 编辑 | 操作列 | 编辑活动信息 | 营销活动编辑权限 |
| 详情 | 操作列（已发布/已结束/已取消） | 查看活动详细信息 | 营销活动查看权限 |
| 表单统计 | 操作列（已发布/已结束/已取消） | 查看活动统计数据 | 营销活动查看权限 |

## 四、操作流程

### 4.1 发布活动流程

```
步骤 1: 登录系统，进入营销推广 → 营销活动
步骤 2: 点击"助力活动"标签页
步骤 3: 在列表中找到目标活动（状态为"未发布"）
步骤 4: 点击该活动行的"发布"按钮
步骤 5: 系统弹出确认对话框："确定要发布该活动吗？"
步骤 6: 点击"确定"确认发布
步骤 7: 系统提示"发布成功"
步骤 8: 活动状态变更为"已发布"，操作按钮更新为 [详情] [表单统计] [取消发布] [编辑]
```

### 4.2 取消发布流程

```
步骤 1: 登录系统，进入营销推广 → 营销活动
步骤 2: 点击"助力活动"标签页
步骤 3: 在列表中找到目标活动（状态为"已发布"）
步骤 4: 点击该活动行的"取消发布"按钮
步骤 5: 系统弹出确认对话框："确定要取消发布该活动吗？取消后用户将无法参与"
步骤 6: 点击"确定"确认取消发布
步骤 7: 系统提示"取消发布成功"
步骤 8: 活动状态变更为"已取消"，操作按钮更新为 [详情] [表单统计]
```

### 4.3 流程图

```
发布流程：
开始
  ↓
进入助力活动列表
  ↓
选择"未发布"状态活动
  ↓
点击"发布"按钮
  ↓
确认发布？
  ├─ 否 → 取消操作
  └─ 是 → 执行发布
        ↓
      状态变更为"已发布"
        ↓
      活动上线，用户可参与
        ↓
      结束

取消发布流程：
开始
  ↓
进入助力活动列表
  ↓
选择"已发布"状态活动
  ↓
点击"取消发布"按钮
  ↓
确认取消发布？
  ├─ 否 → 取消操作
  └─ 是 → 执行取消发布
        ↓
      状态变更为"已取消"
        ↓
      活动下线，用户无法参与
        ↓
      结束
```

## 五、校验规则

### 5.1 发布前校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 活动配置完整性 | 活动基本信息、规则配置、关联券配置必须完整 | "请先完善活动配置" |
| 活动时间校验 | 活动结束时间必须晚于开始时间 | "活动结束时间必须晚于开始时间" |
| 关联券校验 | 必须配置有效的优惠券 | "请配置关联优惠券" |

### 5.2 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 重复发布 | 已发布的活动不能再次发布 | 按钮不显示 |
| 已过期活动 | 已过期的活动不能发布 | 按钮不显示 |
| 已取消活动 | 已取消的活动不能重新发布 | 按钮不显示 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动配置不完整时点击发布 | "请先完善活动配置" | 点击"编辑"完善配置后再发布 |
| EX-002 | 活动结束时间早于开始时间 | "活动结束时间必须晚于开始时间" | 编辑活动，修改正确的时间 |
| EX-003 | 未配置关联优惠券 | "请配置关联优惠券" | 编辑活动，配置关联券 |
| EX-004 | 发布时网络异常 | "发布失败，请重试" | 检查网络后重试 |
| EX-005 | 无发布权限 | 按钮不显示或点击后提示无权限 | 联系管理员申请权限 |

## 九、注意事项

### 9.1 发布前检查清单

- [ ] 活动名称、时间配置正确
- [ ] 参与人资格配置正确
- [ ] 助力人资格配置正确
- [ ] 关联优惠券配置正确
- [ ] 活动页配置（规则文本、背景图等）完整

### 9.2 取消发布注意事项

- 取消发布后，已参与的用户数据保留
- 取消发布后，新用户无法再参与活动
- 取消发布操作不可逆，需谨慎操作
- 建议在非高峰期执行取消发布操作
