## 一、功能概述

- **功能目的**: 创建新的抽奖类营销活动，通过抽奖吸引客户参与
- **使用场景**: 运营人员需要开展抽奖活动时使用
- **业务价值**: 增加用户参与度，促进房源成交

### 3.2 字段说明

#### 基本信息

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 不能为空，建议 20 字以内 | 抽奖活动的名称 |
| 活动时间 | 日期范围 | 是 | 开始日期≤结束日期，不能早于当前日期 | 活动的有效时间范围 |

#### 活动规则

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 关联业务 | 下拉选择 | 是 | 必须选择 | 选择活动关联的业务类型（如新签、续租等） |
| 参与人资格 | 下拉选择 | 是 | 必须选择 | 设置参与抽奖的用户资格（如新客、老客、全部用户等） |

| 实体奖中奖短信通知手机号 | 手机号 | 否 | 11 位手机号格式 | 实体奖品中奖时接收短信通知的手机号 |
| 适用租期 | 下拉选择 | 是 | 必须选择 | 奖品适用的租期（如 3 个月、6 个月、12 个月等） |
| 奖品类型 | 下拉选择 | 是 | 必须选择 | 奖品类型（如优惠券、实体奖品等） |
| 奖品名称/选择券 | 文本/选择 | 是 | 不能为空 | 选择或输入奖品名称 |
| 实体奖品库存/券赠送量 | 数字 | 是 | 正整数 | 奖品数量或券的赠送量 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航到 营销推广 → 营销活动
步骤 2: 点击顶部标签页"抽奖活动"切换到抽奖活动列表
步骤 3: 点击"新建活动"按钮，打开创建表单
步骤 4: 填写基本信息：
        - 输入活动名称
        - 选择活动开始日期和结束日期
步骤 5: 配置活动规则：
        - 选择关联业务
        - 选择参与人资格
        - 选择排除参与人（可选）
        - 输入抽奖次数
        - 输入实体奖中奖短信通知手机号（可选）
步骤 6: 配置关联奖品：
        - 点击添加奖品（如有需要）
        - 选择适用租期
        - 选择奖品类型
        - 选择或输入奖品名称
        - 输入库存/赠送量
        - 设置中奖概率
步骤 7: 配置活动页：
        - 在富文本编辑器中输入活动规则文本
        - 点击"点击上传图片"上传活动背景图
        - 输入外部 Feedback URL（可选）
步骤 8: 检查所有必填字段已填写
步骤 9: 点击"提交"按钮保存活动
步骤 10: 系统提示创建成功，返回列表页
```

### 4.2 流程图

```
开始
  ↓
进入营销活动 → 抽奖活动
  ↓
点击"新建活动"
  ↓

  ↓

  ↓

  ↓

  ↓
检查必填项
  ↓
  ├─ 有遗漏 → 补充填写
  ↓
点击"提交"
  ↓
系统验证
  ↓
  ├─ 验证失败 → 显示错误提示 → 修正后重新提交
  ↓
  └─ 验证成功 → 保存活动 → 返回列表
  ↓
结束
```
