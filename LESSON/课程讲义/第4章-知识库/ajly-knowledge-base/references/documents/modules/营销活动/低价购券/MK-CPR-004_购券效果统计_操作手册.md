# MK-CPR-004 购券效果统计 操作手册

> **功能编号**: MK-CPR-004  
> **功能名称**: 购券效果统计  
> **所属模块**: 营销活动 - 低价购券  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 查看低价购券活动的参与数据统计和购券记录详情
- **使用场景**: 运营人员需要查看购券活动的参与情况、购券人信息、订单状态等数据
- **业务价值**: 帮助运营人员了解活动效果，分析用户购券行为，为后续活动优化提供数据支持

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 营销活动 → 低价购券 → 表单统计

### 2.3 前置条件


### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 无 | 显示当前统计的活动名称 |
| 收集表单总数 | 数字 | 是 | 无 | 活动收集的表单总数量 |
| 成交总数 | 数字 | 是 | 无 | 活动成交的总数量 |
| 序号 | 数字 | 是 | 无 | 记录序号，从 1 开始 |
| 购券人电话 | 文本 | 是 | 手机号脱敏 | 购券人手机号，中间 4 位隐藏（如 137****8200） |
| 提交时间 | 日期时间 | 是 | 无 | 用户提交购券订单的时间 |
| 订单状态 | 文本 | 是 | 枚举值 | 订单当前状态（已退款/已完成/待支付等） |
| 时间 | 日期时间 | 是 | 无 | 订单状态更新时间 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 对话框顶部 | 导出购券记录为 Excel 文件 | 营销活动查看权限 |
| 确定 | 对话框底部 | 关闭统计对话框 | 无 |
| 关闭 (×) | 对话框右上角 | 关闭统计对话框 | 无 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入营销推广 → 营销活动 → 低价购券
步骤 2: 在低价购券活动列表中，找到目标活动
步骤 3: 点击该活动行中的"表单统计"按钮
步骤 4: 系统弹出统计对话框，显示活动名称、表单总数、成交总数
步骤 5: 查看购券记录列表（序号、购券人电话、提交时间、订单状态、时间）
步骤 6: （可选）点击"导出"按钮导出 Excel 报表
步骤 7: 点击"确定"按钮或右上角"×"关闭对话框
```

### 4.2 流程图

```
开始
  ↓
进入低价购券列表
  ↓
点击"表单统计"按钮
  ↓
显示统计对话框
  ↓
查看统计数据
  ├─→ 点击"导出" → 下载 Excel
  ↓
点击"确定"关闭
  ↓
结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 活动存在性 | 活动不存在或已被删除 | 点击表单统计时 |
| 权限校验 | 无权限查看该活动统计 | 点击表单统计时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 购券人电话 | 手机号脱敏显示（中间 4 位隐藏） | 无 | 137****8200 |
| 提交时间 | YYYY-MM-DD HH:mm:ss | 无 | 2025-12-25 14:39:59 |
| 时间 | YYYY-MM-DD HH:mm:ss | 无 | 2025-12-25 14:41:51 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 活动存在性 | 活动必须存在且未被删除 | 活动不存在或已被删除 |
| 权限校验 | 用户必须有该活动的查看权限 | 无权限查看该活动统计 |
| 数据加载 | 购券记录数据必须可加载 | 数据加载失败，请稍后重试 |
