## 一、功能概述

- **功能目的**: 查看推荐活动的推荐记录详情，包括推荐人信息、被推荐人信息、成交状态等
- **使用场景**: 运营人员需要查看推荐活动的效果，追踪推荐成交情况
- **业务价值**: 帮助运营人员了解推荐活动的实际转化效果，评估活动质量

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 营销活动 → 推荐活动 → 表单统计

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 营销活动 → 推荐活动                                       |
+----------------------------------------------------------+
| 筛选区：                                                  |
| 活动名称 | 活动时间 (起 - 止) | 活动状态 |                   |
| [搜索] [重置] [新建活动] [导出]                           |
+----------------------------------------------------------+
| 数据列表：                                                |
| 序号 | 活动名称 | 活动时间 | 活动状态 | 创建时间 |        |
| 表单总数 | 成交总数 | 活动链接 | 操作 |                    |
| 操作：[详情] [表单统计] [发布] [取消发布] [编辑]            |
+----------------------------------------------------------+
| 分页：共 341 条 | 10 条/页 | 1 2 3 4 5 6 ... 35 前往 [页]    |
+----------------------------------------------------------+
```

**入口按钮**: 点击任意推荐活动行中的【表单统计】按钮

### 2.3 前置条件

- [ ] 已登录系统并具有营销活动查看权限
- [ ] 已创建至少一个推荐活动
- [ ] 推荐活动有推荐记录数据

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入营销推广 → 营销活动
步骤 2: 确保当前在"推荐活动"标签页
步骤 3: 在推荐活动列表中找到目标活动
步骤 4: 点击该活动行中的【表单统计】按钮
步骤 5: 查看弹出的统计对话框中的推荐记录详情
步骤 6: 可点击【导出】按钮导出数据
步骤 7: 查看完毕后点击【确定】或【×】关闭弹窗
```

## 五、校验规则

### 5.1 必填校验

本功能为查询功能，无必填字段校验。

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 推荐人电话 | 11 位手机号 | 无 | 15120745255 |
| 被推荐人电话 | 11 位手机号 | 无 | 15088698100 |
| 成交时间 | YYYY-MM-DD HH:mm:ss | 无 | 2026-02-02 11:28:09 |
| 提交时间 | YYYY-MM-DD HH:mm:ss | 无 | 2026-02-02 11:23:23 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 数据权限 | 只能查看自己有权限的活动数据 | 无权限查看该活动数据 |
| 数据存在性 | 活动必须有推荐记录才能查看 | 该活动暂无推荐记录 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动无推荐记录 | 弹窗显示"共 0 条" | 属于正常情况，表示该活动暂无推荐数据 |
| EX-002 | 导出功能失败 | 根据系统实际提示 | 检查网络或联系技术支持 |
| EX-003 | 权限不足 | 无法看到【表单统计】按钮 | 联系管理员开通营销活动查看权限 |
