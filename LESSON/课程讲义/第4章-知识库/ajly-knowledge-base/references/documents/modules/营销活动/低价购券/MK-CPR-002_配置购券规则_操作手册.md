## 一、功能概述

- **功能目的**: 配置低价购券活动的参与规则、关联优惠券和落地页展示内容
- **使用场景**: 创建低价购券活动后，需要配置活动规则、关联券配置、落地页展示等
- **业务价值**: 定义活动参与资格、优惠券配置和用户端展示内容，确保活动合规有效

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 1-50 字符 | 活动名称，创建后不可修改 |
| 活动时间 | 日期范围 | 是 | 结束日期≥开始日期 | 活动有效期 |

| 活动规则文本 | 富文本 | 是 | 不能为空 | 活动详细说明 |
| 活动背景图 | 图片上传 | 是 | 推荐 375x812px | 落地页背景图 |
| 按钮颜色 | 颜色选择 | 是 | 有效颜色值 | 购买按钮背景色 |
| 按钮文字颜色 | 颜色选择 | 是 | 有效颜色值 | 购买按钮文字色 |
| 商品 ID | 文本 | 是 | 有效商品 ID | 关联的商品 ID |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 导航到营销推广 → 营销活动 → 低价购券
步骤 2: 找到需要编辑的活动（状态为"未发布"或"已取消"）
步骤 3: 点击"编辑"按钮进入配置页面
步骤 4: 配置参与人资格（必选）
步骤 5: 配置排除参与人（可选）
步骤 6: 配置关联券（选择优惠券 + 设置发放数量）
步骤 7: 编辑活动规则文本（富文本编辑器）
步骤 8: 上传活动背景图（推荐 375x812px）
步骤 9: 配置按钮颜色和文字颜色
步骤 10: 填写商品 ID 和跳转链接
步骤 11: 填写外部 Feedback URL（可选）
步骤 12: 点击"生成预览"查看效果（可选）
步骤 13: 点击"提交"保存配置
```

### 4.2 流程图

```
开始
  ↓
进入编辑页面
  ↓
配置活动规则 (参与人资格/关联券)
  ↓
配置落地页 (规则文本/背景图/按钮颜色)
  ↓
填写商品 ID 和跳转链接
  ↓
点击生成预览？ → 是 → 查看预览效果
  ↓否
点击提交
  ↓
系统校验
  ↓
校验通过？ → 否 → 显示错误提示
  ↓是
保存成功
  ↓
返回活动列表
  ↓
结束
```

| 关联券 - 选择券 | 请选择优惠券 | 提交时 |
| 关联券 - 发放数量 | 请输入发放数量 | 提交时 |
| 活动规则文本 | 活动规则不能为空 | 提交时 |
| 活动背景图 | 请上传活动背景图 | 提交时 |

| 商品 ID | 商品 ID 不能为空 | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 活动时间 | 结束日期≥开始日期 | 结束日期不能早于开始日期 | 2025-12-01 至 2025-12-31 |
| 发放数量 | 正整数≥1 | 发放数量必须为正整数 | 1, 2, 3 |
| 跳转链接 | 有效 URL 格式 | 请输入有效的 URL 地址 | https://example.com/page |
| 外部 Feedback URL | 有效 URL 格式 | 请输入有效的 URL 地址 | https://example.com/api/leads |
| 商品 ID | 数字字符串 | 请输入有效的商品 ID | 1998945769001046016 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 优惠券有效性 | 选择的优惠券必须为有效状态 | 选择的优惠券已失效，请重新选择 |
| 优惠券时间 | 优惠券有效期需覆盖活动时间 | 优惠券有效期需覆盖活动时间 |
| 配置完整性 | 所有必填字段必须填写 | 请完善所有必填配置项 |
| 背景图尺寸 | 推荐 375x812px | 建议上传 375x812px 图片以获得最佳效果 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 优惠券已下架 | 选择的优惠券已失效，请重新选择 | 更换其他有效优惠券 |
| EX-002 | 活动已发布 | 已发布活动不可编辑，请先取消发布 | 先取消发布再编辑 |
| EX-003 | 无编辑权限 | 您没有编辑该活动的权限 | 联系管理员授权 |
| EX-004 | 网络失败 | 保存失败，请检查网络连接 | 检查网络后重试 |
| EX-005 | 商品 ID 无效 | 商品 ID 不存在或无效 | 核实商品 ID 后重新填写 |
| EX-006 | 跳转链接无效 | 跳转链接格式不正确 | 检查 URL 格式后重新填写 |
| EX-007 | 背景图上传失败 | 图片上传失败，请重试 | 检查图片格式和大小后重试 |
| EX-008 | 活动时间已过 | 活动开始时间不能早于当前时间 | 调整活动开始时间 |
