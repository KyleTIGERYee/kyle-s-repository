## 一、功能概述

- **功能目的**: 配置拼团活动的详细规则，包括参与人资格、成团目标、奖励配置等
- **使用场景**: 创建拼团活动后，需要配置具体的拼团规则和奖励机制
- **业务价值**: 灵活配置拼团活动的参与条件和奖励策略，吸引用户参与拼团

### 3.2 字段说明

#### 活动规则区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 发起人资格 | 下拉选择 | 是 | 必须选择 | 可选：全部用户/新客/老客/毕业生等 |
| 排除发起人 | 下拉多选 | 否 | 无 | 排除特定用户群体 |
| 参与人资格 | 下拉选择 | 是 | 必须选择 | 可选：全部用户/新客/老客/毕业生等 |
| 排除参与人 | 下拉多选 | 否 | 无 | 排除特定用户群体 |
| 成团目标人数 | 数字输入 | 是 | 正整数，≥2 | 需要多少人拼团成功 |
| 成团目标类型 | 下拉选择 | 是 | 必须选择 | 可选：新签/续约/不关联 |
| 成团有效期 | 数字输入 | 否 | 正整数 | 需在 X 天内拼团成功 |
| 找房单 | 单选 | 否 | 是/否 | 是否需要关联找房单 |

#### 关联券区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 发起人与参与人奖励 | 单选 | 是 | 一致/不一致 | 是否给予相同奖励 |
| 根据签约周期奖励 | 单选 | 是 | 是/否 | 是否根据签约周期差异化奖励 |
| 仅可获得一次优惠券 | 单选 | 是 | 是/否 | 用户是否只能获得一次奖励 |
| 阶段奖励 - 拼团人数 | 数字输入 | 是 | 正整数 | 达到该人数触发奖励 |
| 阶段奖励 - 发起人奖励券 | 下拉选择 | 是 | 必须选择有效券 | 选择优惠券 |
| 阶段奖励 - 发放数量 | 数字输入 | 是 | 正整数 | 发放几张优惠券 |

#### 落地页配置区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动规则文本 | 富文本 | 是 | 不能为空 | 支持标题/段落/加粗/斜体等 |
| 活动背景图 | 图片上传 | 是 | 建议 375x812 | 小程序活动页背景 |
| 转发卡片图 | 图片上传 | 是 | 建议 5:4 比例 | 分享时显示的卡片图 |
| 外部 Feedback URL | 文本输入 | 否 | URL 格式 | 外部反馈链接 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 进入营销活动 → 拼团活动，点击"新建活动"或"编辑"
步骤 2: 填写基本信息（活动名称、活动时间）
步骤 3: 配置活动规则：
        - 选择发起人资格和参与人资格
        - 设置成团目标人数（≥2）
        - 选择成团目标类型（新签/续约/不关联）
        - 可选设置成团有效期和找房单
步骤 4: 配置关联券：
        - 选择发起人与参与人奖励是否一致
        - 选择是否根据签约周期奖励
        - 选择是否仅可获得一次优惠券
        - 配置阶段奖励（可添加多阶段）
步骤 5: 配置落地页：
        - 编辑活动规则文本（富文本编辑器）
        - 上传活动背景图（375x812）
        - 上传转发卡片图（5:4）
        - 可选填写外部 Feedback URL
步骤 6: 点击"提交"保存配置
```

### 4.2 流程图

```
开始
  ↓
填写基本信息
  ↓
配置活动规则 → 校验资格/人数/类型
  ↓
配置关联券 → 选择优惠券/设置数量
  ↓
配置落地页 → 编辑文本/上传图片
  ↓
提交保存
  ↓
结束
```

| 字段 | 格式要求 | 错误提示 | 示例 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 优惠券有效性 | 必须选择状态有效的优惠券 | 请选择有效的优惠券 |
| 时间合理性 | 活动时间不能早于当前时间 | 活动时间设置不合理 |
| 配置完整性 | 所有必填字段必须填写 | 请完善配置信息 |
| 图片尺寸 | 背景图建议 375x812，卡片图建议 5:4 | 图片尺寸不符合建议（非强制） |
| 阶段奖励 | 至少配置一个奖励阶段 | 请配置阶段奖励 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 未选择发起人资格提交 | 请选择发起人资格 | 选择发起人资格后重试 |
| EX-002 | 成团目标人数<2 | 成团目标人数必须≥2 | 修改为≥2 的数字 |
| EX-003 | 优惠券已下架/无效 | 请选择有效的优惠券 | 更换其他有效优惠券 |
