## 一、功能概述

- **功能目的**: 查看推荐活动的表单收集和成交数据统计，追踪推荐活动的实际效果
- **使用场景**: 
  - 运营人员需要查看推荐活动的转化效果
  - 分析推荐活动的投入产出比
  - 追踪推荐人和被推荐人的成交情况
- **业务价值**: 通过数据化分析推荐活动效果，为活动优化提供依据

### 3.2 字段说明

#### 统计摘要区

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 自动显示当前活动名称 | 展示当前统计的活动名称 |
| 收集表单总数 | 数字 | 是 | 非负整数 | 该活动累计收集的表单数量 |
| 成交总数 | 数字 | 是 | 非负整数 | 该活动带来的成交数量 |

#### 数据列表区

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 序号 | 数字 | 是 | 自动编号 | 列表序号 |
| 推荐人电话 | 手机号 | 是 | 11 位手机号 | 推荐人的手机号码 |
| 推荐人领券 | 文本 | 否 | - | 推荐人领取的优惠券名称 |
| 被推荐人电话 | 手机号 | 是 | 11 位手机号 | 被推荐人的手机号码 |
| 被推荐人领券 | 文本 | 否 | - | 被推荐人领取的优惠券名称 |
| 成交时间 | 日期时间 | 否 | YYYY-MM-DD HH:mm:ss | 成交的具体时间 |
| 提交时间 | 日期时间 | 是 | YYYY-MM-DD HH:mm:ss | 表单提交的时间 |
| 意向地区 | 文本 | 否 | - | 被推荐人意向的行政区 |
| 意向房型 | 文本 | 否 | - | 被推荐人意向的房型（如一居室、二居室） |
| 租金预算 | 文本 | 否 | - | 被推荐人的租金预算范围 |
| 意向入住时间 | 文本 | 否 | - | 被推荐人期望的入住时间 |
| 成交状态 | 文本 | 是 | 枚举值 | 已成交/未成交 |
| 成交数 | 数字 | 是 | 非负整数 | 成交的房间数量 |
| 企业名称 | 文本 | 否 | - | 关联的企业名称（如有） |
| 管家名称 | 文本 | 否 | - | 负责该客户的管家姓名 |
| 管家电话 | 手机号 | 否 | 11 位手机号 | 管家的联系电话 |

### 3.3 按钮说明

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入"营销推广 → 营销活动"页面
步骤 2: 在推荐活动列表中，找到需要查看的活动
步骤 3: 点击该活动行操作列的"表单统计"按钮
步骤 4: 系统弹出统计对话框，显示活动名称、表单总数、成交总数
步骤 5: 查看详细的推荐记录列表，包含推荐人、被推荐人、成交信息等
步骤 6: 如需导出数据，点击"导出"按钮
步骤 7: 查看完成后，点击"确 定"按钮关闭对话框
```

## 五、校验规则

### 5.1 必填校验

本功能为数据查看功能，无必填字段校验。

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 推荐人电话 | 11 位手机号 | 无（只读显示） | 15120745255 |
| 被推荐人电话 | 11 位手机号 | 无（只读显示） | 15088698100 |
| 成交时间 | YYYY-MM-DD HH:mm:ss | 无（只读显示） | 2026-02-02 11:28:09 |
| 提交时间 | YYYY-MM-DD HH:mm:ss | 无（只读显示） | 2026-02-02 11:23:23 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 数据权限 | 只能查看本人创建或有权限查看的活动统计 | 无权限查看 |
| 数据完整性 | 表单总数应等于列表记录数 | - |
| 成交数统计 | 成交总数应等于"已成交"状态的记录数之和 | - |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动无任何表单数据 | 列表显示为空，表单总数和成交总数均为 0 | 正常现象，说明活动尚未收集到数据 |
| EX-002 | 导出功能失败 | 根据系统实际提示 | 检查网络或联系技术支持 |
| EX-003 | 数据量过大加载缓慢 | 无提示，加载时间较长 | 等待加载完成或使用筛选功能 |
