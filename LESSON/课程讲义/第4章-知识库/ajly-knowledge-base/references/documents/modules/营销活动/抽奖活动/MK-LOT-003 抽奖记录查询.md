> **所属模块**: 营销活动 - 抽奖活动  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 查看抽奖活动的参与记录，包括参与人信息、关联业务、签约月份和奖品信息
- **使用场景**: 
  - 运营人员查看抽奖活动参与情况
  - 核对中奖用户信息和奖品发放
  - 导出抽奖记录进行数据分析
- **业务价值**: 帮助运营人员追踪抽奖活动效果，核销奖品发放，分析用户参与行为

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 序号 | 数字 | 是 | 自动递增 | 记录序号，从 1 开始 |
| 姓名 | 文本 | 是 | 无 | 参与抽奖的用户姓名 |
| 电话 | 文本 | 是 | 脱敏显示 | 用户手机号，中间 4 位隐藏（如 135****6632） |
| 关联业务 | 文本 | 是 | 无 | 抽奖活动关联的业务类型（如：续约、新签等） |
| 签约月份 | 文本 | 是 | 无 | 用户签约的租期（如：1 个月、2 个月等） |
| 奖品 | 文本 | 是 | 无 | 用户抽中的奖品名称或优惠券名称 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 对话框左上角 | 导出抽奖记录为 Excel 文件 | 营销活动查看权限 |
| 确 定 | 对话框底部 | 关闭对话框，返回列表页 | 无 |
| × (关闭) | 对话框右上角 | 关闭对话框 | 无 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入"营销推广 → 营销活动"页面
步骤 2: 点击"抽奖活动"标签页，切换到抽奖活动列表
步骤 3: 找到目标抽奖活动，点击操作列的"表单统计"按钮
步骤 4: 系统弹出对话框，显示该活动的抽奖记录列表
步骤 5: 查看记录详情（姓名、电话、关联业务、签约月份、奖品）
步骤 6: （可选）点击"导出"按钮导出记录
步骤 7: 点击"确 定"按钮关闭对话框
```

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动无参与记录 | 对话框显示"共 0 条" | 正常现象，说明活动尚未有用户参与 |
| EX-002 | 网络加载失败 | 对话框无法弹出或数据加载失败 | 检查网络连接，刷新页面重试 |
| EX-003 | 导出功能失败 | 无响应或提示导出失败 | 检查浏览器弹窗设置，允许下载 |
| EX-004 | 权限不足 | "表单统计"按钮不显示 | 联系管理员分配营销活动查看权限 |
