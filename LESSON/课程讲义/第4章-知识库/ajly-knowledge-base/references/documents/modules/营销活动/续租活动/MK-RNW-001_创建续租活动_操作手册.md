## 一、功能概述

- **功能目的**: 创建老客续租优惠类营销活动，促进在租客户续租
- **使用场景**: 运营人员需要针对在租客户推出续租优惠活动时
- **业务价值**: 提高续租率，降低空置损失，增强客户粘性

---

### 3.2 字段说明

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航到 营销推广 → 营销活动
步骤 2: 点击顶部标签页"续租活动"切换到续租活动列表
步骤 3: 点击"新建活动"按钮，进入创建活动表单页
步骤 4: 填写活动名称（如：续租活动测试 0323）
步骤 5: 选择活动时间（开始日期和结束日期）
步骤 6: 在关联券表格中配置券信息：
        - 点击"适用续租周期"下拉框选择租期要求
        - 点击"适用预计居住时长"下拉框选择居住时长要求
        - 点击"选择券"下拉框选择要关联的优惠券
        - 填写赠送量（如：1）
步骤 7: （可选）点击"继续添加"配置多组关联券规则
步骤 8: 点击"提交"按钮保存活动
步骤 9: 系统校验通过后创建活动成功，返回列表页
```

### 4.2 流程图

```
开始
  ↓
进入续租活动列表页
  ↓

  ↓

  ↓

  ↓
点击"提交"
  ↓
┌─────────────────┐
│  系统校验        │
└─────────────────┘
  ↓

  ↓
  └─ 校验成功 → 创建活动成功 → 返回列表页
  ↓
结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 活动名称 | 请输入活动名称 | 提交时 |
| 活动时间 | 请选择活动时间 | 提交时 |
| 适用续租周期 | 请将表格内容填写完整 | 提交时 |
| 适用预计居住时长 | 请将表格内容填写完整 | 提交时 |
| 选择券 | 请将表格内容填写完整 | 提交时 |
| 赠送量 | 请将表格内容填写完整 | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 活动名称 | 2-50 字符 | 活动名称长度不符合要求 | 续租活动 0323 |
| 开始日期 | 日期格式 YYYY-MM-DD | 请选择有效的开始日期 | 2026-03-24 |
| 结束日期 | 日期格式 YYYY-MM-DD | 请选择有效的结束日期 | 2026-04-30 |
| 结束日期 | 必须≥开始日期 | 结束日期不能早于开始日期 | - |
| 赠送量 | 正整数≥1 | 赠送量必须为大于 0 的整数 | 1 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 优惠券有效性 | 选择的优惠券必须为有效状态 | 请选择有效的优惠券 |
| 关联券配置完整性 | 关联券表格所有必填字段必须填写完整 | 请将表格内容填写完整 |
| 活动时间合理性 | 活动时间不能为过去的时间 | 活动时间必须大于等于当前日期 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动名称为空提交 | 请输入活动名称 | 填写活动名称后重新提交 |
| EX-002 | 活动时间为空提交 | 请选择活动时间 | 选择活动开始和结束日期 |
| EX-003 | 关联券配置不完整提交 | 请将表格内容填写完整 | 填写完整关联券表格所有必填字段 |
| EX-004 | 选择已下架的优惠券 | 请选择有效的优惠券 | 重新选择有效状态的优惠券 |
| EX-005 | 结束日期早于开始日期 | 结束日期不能早于开始日期 | 调整结束日期晚于或等于开始日期 |
| EX-006 | 赠送量为 0 或负数 | 赠送量必须为大于 0 的整数 | 填写大于 0 的整数 |
| EX-007 | 无营销活动管理权限 | 无权限访问该功能 | 联系管理员开通权限 |
| EX-008 | 网络请求失败 | 网络异常，请稍后重试 | 检查网络连接后重试 |
