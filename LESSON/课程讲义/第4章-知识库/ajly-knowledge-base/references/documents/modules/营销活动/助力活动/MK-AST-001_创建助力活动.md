## 一、功能概述

- **功能目的**: 创建新的助力活动，通过邀请好友助力的方式吸引客户参与，获取优惠奖励
- **使用场景**: 
  - 运营人员策划社交传播类营销活动
  - 通过助力机制扩大活动影响力
  - 吸引新用户注册或老用户复购
- **业务价值**: 利用社交关系链进行裂变传播，降低获客成本，提升活动参与度

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录管理后台，进入营销推广 → 营销活动 → 助力活动
步骤 2: 点击"新建活动"按钮
步骤 3: 填写基本信息：
        - 输入活动名称（如："邀请好友助力赢大奖"）
        - 选择活动开始日期和结束日期
步骤 4: 配置活动规则：
        - 选择参与人资格（如："全部用户"）
        - 选择助力人资格（如："全部用户"）
        - 配置关联券：输入适用助力人数、选择优惠券、设置赠送量
        - 如需多档位奖励，点击"继续添加"
步骤 5: 配置活动页：
        - 在富文本编辑器中输入活动规则说明
        - 上传活动背景图（375x812 像素）
        - 上传转发卡片图（5:4 比例）
        - 选择按钮颜色和按钮文字颜色
步骤 6: （可选）填写外部 Feedback URL
步骤 7: 点击"提交"按钮保存
步骤 8: 系统验证通过后，活动创建成功，返回列表页
```

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 活动名称 | 活动名称不能为空 | 提交时 |
| 活动时间 | 活动时间不能为空 | 提交时 |
| 参与人资格 | 请选择参与人资格 | 提交时 |
| 助力人资格 | 请选择助力人资格 | 提交时 |
| 关联券 - 选择券 | 请选择优惠券 | 提交时 |
| 活动规则文本 | 活动规则文本不能为空 | 提交时 |
| 活动背景图 | 请上传活动背景图 | 提交时 |
| 转发卡片图 | 请上传转发卡片图 | 提交时 |
| 按钮颜色 | 请选择按钮颜色 | 提交时 |
| 按钮文字颜色 | 请选择按钮文字颜色 | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 活动名称 | 文本，建议 20 字以内 | 活动名称长度超限 | "新春助力赢好礼" |
| 活动时间 | 开始日期 ≤ 结束日期 | 结束日期不能早于开始日期 | 2026-03-23 至 2026-04-23 |
| 适用助力人数 | 正整数 | 请输入有效的助力人数 | 5 |
| 赠送量 | 正整数 | 请输入有效的赠送数量 | 1 |
| 外部 Feedback URL | 有效的 URL 格式 | 请输入有效的 URL 地址 | https://example.com/callback |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 活动时间范围 | 活动结束日期不能早于当前日期 | 活动结束日期必须大于当前日期 |
| 优惠券状态 | 选择的优惠券必须处于可用状态 | 该优惠券不可用，请选择其他优惠券 |
| 图片尺寸 | 背景图推荐 375x812，卡片图推荐 5:4 | 图片尺寸不符合推荐要求（警告，可继续） |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 未填写必填字段直接提交 | 对应字段显示红色边框和错误提示 | 根据提示补充完整必填信息 |
| EX-002 | 活动结束日期早于开始日期 | "结束日期不能早于开始日期" | 调整日期选择，确保结束日期≥开始日期 |
| EX-003 | 选择的优惠券已被禁用 | "该优惠券不可用，请选择其他优惠券" | 更换为可用状态的优惠券 |
| EX-004 | 图片上传失败 | "图片上传失败，请重试" | 检查图片格式和大小，重新上传 |
| EX-005 | 网络异常导致提交失败 | "网络异常，请重试" | 检查网络连接后重新提交 |
| EX-006 | 无营销活动管理权限 | "您没有权限执行此操作" | 联系管理员开通权限 |
