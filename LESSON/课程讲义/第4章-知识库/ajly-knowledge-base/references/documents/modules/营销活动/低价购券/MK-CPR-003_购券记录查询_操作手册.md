## 一、功能概述

- **功能目的**: 查询低价购券活动的用户购券记录，包括购券人信息、提交时间、订单状态等
- **使用场景**: 运营人员需要查看低价购券活动的参与情况、购券明细、订单状态统计
- **业务价值**: 帮助运营人员了解活动效果、跟踪购券订单状态、进行数据分析和导出

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 只读显示 | 显示当前统计的低价购券活动名称 |
| 收集表单总数 | 数字 | 是 | 只读显示 | 该活动收集的购券表单总数 |
| 成交总数 | 数字 | 是 | 只读显示 | 该活动成功成交的订单总数 |
| 序号 | 数字 | 是 | 自动编号 | 购券记录的序号 |
| 购券人电话 | 文本 | 是 | 脱敏显示 | 购券人手机号，中间 4 位隐藏（如 137****8200） |
| 提交时间 | 日期时间 | 是 | 只读显示 | 用户提交购券订单的时间，格式：YYYY-MM-DD HH:mm:ss |
| 订单状态 | 文本 | 是 | 只读显示 | 订单当前状态（已退款/已完成/待支付等） |
| 时间 | 日期时间 | 是 | 只读显示 | 订单状态更新时间或支付时间，格式：YYYY-MM-DD HH:mm:ss |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 对话框顶部 | 导出购券记录为 Excel 文件 | 营销活动查看权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航到"租赁管理 → 优惠券 → 营销活动"
步骤 2: 点击"低价购券"标签页，进入低价购券活动列表
步骤 3: 找到需要查询的活动，点击操作列的"表单统计"按钮
步骤 4: 系统弹出统计对话框，显示活动名称、表单总数、成交总数
步骤 5: 查看购券记录列表，包含购券人电话、提交时间、订单状态等
步骤 6: 如需导出，点击"导出"按钮下载 Excel 文件
步骤 7: 查看完成后，点击"确定"按钮或右上角"×"关闭对话框
```

### 4.2 流程图

```
开始
  ↓
导航到营销活动 → 低价购券
  ↓
点击"表单统计"按钮
  ↓
弹出统计对话框
  ↓
查看购券记录列表
  ↓
    ├─→ 需要导出？─是─→ 点击"导出"按钮
    │                    ↓
    └─←───────────────┘
  ↓
点击"确定"关闭对话框
  ↓
结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 活动存在性 | "活动不存在或已被删除" | 点击表单统计时 |
| 权限校验 | "无权限查看该活动" | 点击表单统计时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 购券人电话 | 手机号脱敏显示（中间 4 位隐藏） | 无 | 137****8200 |
| 提交时间 | YYYY-MM-DD HH:mm:ss | 无 | 2025-12-25 14:39:59 |
| 时间 | YYYY-MM-DD HH:mm:ss | 无 | 2025-12-25 14:41:51 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 活动有效性 | 活动必须存在且未被删除 | "活动不存在或已被删除" |
| 数据权限 | 用户必须有该活动的查看权限 | "无权限查看该活动" |
| 数据加载 | 购券记录必须能正常加载 | "数据加载失败，请重试" |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动不存在或已被删除 | "活动不存在或已被删除" | 确认活动 ID 是否正确，联系管理员 |
| EX-002 | 无权限查看该活动 | "无权限查看该活动" | 申请相应权限或联系活动创建人 |
| EX-003 | 数据加载失败 | "数据加载失败，请重试" | 检查网络连接，刷新页面重试 |
| EX-004 | 导出失败 | "导出失败，请重试" | 检查浏览器下载设置，重试导出 |
| EX-005 | 无购券记录 | 列表显示"共 0 条" | 正常情况，表示暂无用户购券 |
| EX-006 | 数据加载超时 | "数据加载超时，请重试" | 检查网络状态，稍后重试 |
