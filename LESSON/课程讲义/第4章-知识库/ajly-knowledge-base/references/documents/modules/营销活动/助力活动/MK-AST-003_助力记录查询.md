# MK-AST-003 助力记录查询 操作手册

> **功能编号**: MK-AST-003  
> **功能名称**: 助力记录查询  
> **所属模块**: 营销活动 - 助力活动  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 查看助力活动的参与记录详情，包括助力人、邀请人信息及成交状态
- **使用场景**: 运营人员需要查看助力活动的参与情况、统计表单数量和成交转化
- **业务价值**: 帮助运营人员了解活动效果，追踪参与用户转化情况

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 营销活动 → 助力活动 → 选择活动 → 表单统计

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 营销活动 → 助力活动                                       |
+----------------------------------------------------------+
| 筛选区：                                                  |
| 活动名称 | 活动时间 (起 - 止) | 活动状态 |                   |
| [搜索] [重置] [新建活动] [导出]                           |
+----------------------------------------------------------+
| 数据列表：                                                |
| 序号 | 活动名称 | 活动时间 | 活动状态 | 创建时间 |        |
| 表单总数 | 成交总数 | 活动链接 | 操作 |                    |
| 操作：[详情] [表单统计] [发布] [取消发布] [编辑]            |
+----------------------------------------------------------+
| 点击 [表单统计] 按钮进入助力记录查询                       |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 已创建助力活动（MK-AST-001）
- [ ] 活动已有参与用户提交表单
- [ ] 用户拥有营销活动查看权限

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 无 | 显示当前统计的活动名称 |
| 收集表单总数 | 数字 | 是 | 无 | 活动收集的表单总数量 |
| 成交总数 | 数字 | 是 | 无 | 通过活动成交的总数量 |
| 序号 | 数字 | 是 | 自增 | 记录序号 |
| 电话 | 文本 | 是 | 脱敏显示 | 参与用户的手机号（中间 4 位隐藏） |
| 角色类型 | 单选 | 是 | 枚举值 | 助力人/邀请人 |
| 关联的邀请人 | 文本 | 否 | 无 | 助力人关联的邀请人电话，邀请人此项为"-" |
| 成交状态 | 文本 | 是 | 枚举值 | 已成交/未成交 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 对话框顶部 | 导出助力记录为 Excel 文件 | 营销活动管理权限 |
| × (关闭) | 对话框右上角 | 关闭统计对话框 | 无 |
| 确 定 | 对话框底部 | 确认并关闭对话框 | 无 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入营销推广 → 营销活动
步骤 2: 点击"助力活动"标签页
步骤 3: 在助力活动列表中找到目标活动
步骤 4: 点击该活动行操作列的"表单统计"按钮
步骤 5: 系统弹出统计对话框，显示活动名称、表单总数、成交总数
步骤 6: 查看助力记录列表，包含序号、电话、角色类型、关联的邀请人、成交状态
步骤 7: 可点击"导出"按钮导出记录
步骤 8: 点击"确 定"按钮关闭对话框
```

### 4.2 流程图

```
开始
  ↓
进入营销活动 → 助力活动列表
  ↓
点击目标活动的"表单统计"按钮
  ↓
系统弹出统计对话框
  ↓
查看活动汇总信息（表单总数、成交总数）
  ↓
查看助力记录明细列表
  ↓
          ↓
    [可选] 点击"导出"
          ↓
    下载 Excel 文件
          ↓
点击"确 定"关闭对话框
  ↓
结束
```

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动无任何参与记录 | 对话框显示表单总数：0，列表为空 | 属于正常情况，等待用户参与 |
| EX-002 | 活动已被删除 | 点击表单统计无反应或提示"活动不存在" | 联系管理员确认活动状态 |
| EX-003 | 导出功能失败 | 浏览器下载失败或无响应 | 检查浏览器下载设置，重试 |
| EX-004 | 网络异常导致数据加载失败 | 对话框加载超时或显示空白 | 刷新页面后重新尝试 |
