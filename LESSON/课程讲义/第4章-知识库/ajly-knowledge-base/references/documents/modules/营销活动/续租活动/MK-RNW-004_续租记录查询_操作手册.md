## 一、功能概述

- **功能目的**: 查询续租活动的参与记录和续租办理情况
- **使用场景**: 
  - 查看续租活动的客户参与情况
  - 跟踪续租办理进度
  - 统计续租活动效果
- **业务价值**: 帮助运营人员了解续租活动的实际转化情况，便于后续优化活动策略

---

### 3.2 字段说明

#### 筛选区字段

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本输入 | 否 | 最大长度 50 字符 | 支持模糊搜索 |
| 活动时间 - 起 | 日期选择 | 否 | 日期格式 YYYY-MM-DD | 活动开始时间 |
| 活动时间 - 止 | 日期选择 | 否 | 日期格式 YYYY-MM-DD | 活动结束时间 |
| 活动状态 | 下拉选择 | 否 | 选项：全部/未发布/已发布/已取消 | 默认"全部" |

#### 数据列表字段

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 序号 | 数字 | 列表序号 |
| 活动名称 | 文本 | 续租活动名称 |
| 活动时间 | 日期范围 | 活动开始时间 - 结束时间 |
| 活动状态 | 文本 | 未发布/已发布/已取消 |
| 创建时间 | 日期时间 | 活动创建时间 |
| 表单总数 | 数字 | 参与活动的表单总数 |
| 成交总数 | 数字 | 通过活动成交的总数 |
| 活动链接 | 文本 | 活动分享链接（可复制） |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录管理系统
步骤 2: 点击左侧菜单"营销推广" → "营销活动"
步骤 3: 在营销活动页面，点击顶部标签页"续租活动"
步骤 4: （可选）在筛选区输入查询条件
步骤 5: 点击"搜索"按钮执行查询
步骤 6: 查看续租活动列表数据
步骤 7: （可选）点击"详情"查看活动详细信息
步骤 8: （可选）点击"表单统计"查看统计数据
步骤 9: （可选）点击"导出"导出 Excel 报表
```

### 4.2 流程图

```
开始
  ↓
登录系统
  ↓
进入营销活动 → 续租活动
  ↓
是否需要筛选？
  ├─ 是 → 输入筛选条件 → 点击搜索
  └─ 否 → 显示全部数据
  ↓
查看续租记录列表
  ↓
是否需要导出？
  ├─ 是 → 点击导出 → 下载 Excel
  └─ 否 → 继续查看
  ↓
结束
```

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 无续租活动数据 | 列表显示"暂无数据" | 先创建续租活动 |
| EX-002 | 筛选条件无匹配结果 | 列表显示"暂无数据" | 调整筛选条件 |
| EX-003 | 导出失败 | "导出失败，请稍后重试" | 检查网络/权限后重试 |
| EX-004 | 无查看权限 | 页面提示"无权限访问" | 联系管理员开通权限 |
| EX-005 | 数据加载超时 | "数据加载超时，请刷新页面" | 刷新页面重试 |
| EX-006 | 活动链接复制失败 | "复制失败，请手动复制" | 手动选择链接复制 |
