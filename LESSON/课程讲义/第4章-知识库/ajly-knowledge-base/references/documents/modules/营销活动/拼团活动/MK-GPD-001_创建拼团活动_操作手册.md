> **功能名称**: 创建拼团活动  
> **所属模块**: 营销推广 → 营销活动 → 拼团活动  
> **操作频率**: 中频

> **执行人**: 紫灵

## 一、功能概述

- **功能目的**: 创建新的拼团活动，配置活动规则、奖励机制和落地页
- **使用场景**: 运营人员需要发起拼团活动时使用
- **业务价值**: 通过拼团形式吸引客户参与，促进房源签约成交

### 3.2 字段说明

#### 基本信息

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 1-50 个字符 | 拼团活动的名称 |
| 活动时间 | 日期范围 | 是 | 结束日期≥开始日期 | 活动的有效时间范围 |

#### 活动规则

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 发起人资格 | 下拉选择 | 是 | 必须选择 | 可选项：全部用户/新注册用户/老用户等 |
| 排除发起人 | 下拉选择 | 否 | - | 排除特定用户群体 |
| 参与人资格 | 下拉选择 | 是 | 必须选择 | 可选项：全部用户/新注册用户/老用户等 |
| 排除参与人 | 下拉选择 | 否 | - | 排除特定用户群体 |
| 成团目标人数 | 数字输入 | 是 | 正整数，最小值 2 | 需要多少人参与才能成团 |
| 成团目标类型 | 下拉选择 | 是 | 必须选择 | 可选项：新签/续约/不关联 |
| 成团有效期 | 数字输入 | 否 | 正整数 | 需在多少天内拼团成功 |
| 找房单 | 单选 | 否 | - | 是否关联找房单 |

#### 关联券

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 发起人与参与人奖励 | 单选 | 否 | - | 一致：双方奖励相同；不一致：可分别配置 |
| 根据签约周期奖励 | 单选 | 否 | - | 是否根据不同签约周期给予不同奖励 |
| 仅可获得一次优惠券 | 单选 | 否 | - | 用户是否只能获得一次奖励 |

#### 阶段奖励

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 拼团人数 | 数字输入 | 是 | 正整数 | 达到多少人时触发该阶段奖励 |
| 发起人奖励券 | 下拉选择 | 是 | 必须选择有效优惠券 | 选择要发放的优惠券模板 |
| 发放数量 | 数字输入 | 是 | 正整数 | 发放的优惠券数量 |

#### 落地页配置

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动规则文本 | 富文本 | 是 | 不能为空 | 活动规则说明，支持格式化 |
| 活动背景图 | 图片上传 | 是 | 推荐 375x812px | 移动端活动页面背景图 |
| 转发卡片图 | 图片上传 | 是 | 推荐 5:4 比例 | 分享转发时显示的卡片图 |
| 外部 Feedback URL | URL 文本 | 否 | URL 格式 | 外部反馈链接 |

### 3.3 按钮说明

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录管理系统，进入营销推广 → 营销活动
步骤 2: 点击"拼团活动"标签页
步骤 3: 点击"新建活动"按钮
步骤 4: 填写基本信息（活动名称、活动时间）
步骤 5: 配置活动规则（发起人资格、参与人资格、成团目标人数等）
步骤 6: 配置关联券（发起人与参与人奖励规则）
步骤 7: 配置阶段奖励（拼团人数、奖励券、发放数量）
步骤 8: 配置落地页（活动规则文本、背景图、卡片图）
步骤 9: 点击"提交"按钮保存活动
步骤 10: 系统验证通过后，活动创建成功，返回列表页
```

### 4.2 流程图

```
开始
  ↓
进入营销活动 → 拼团活动
  ↓
点击"新建活动"
  ↓
填写基本信息
  ↓
配置活动规则
  ↓
配置关联券
  ↓
配置阶段奖励
  ↓

  ↓

  ↓

  ↓

  ↓

  ↓
返回列表页
  ↓
结束
```

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 未填写活动名称提交 | 请输入活动名称 | 填写活动名称后重新提交 |
| EX-002 | 活动时间结束早于开始 | 结束日期不能早于开始日期 | 调整活动时间范围 |
| EX-003 | 成团目标人数<2 | 成团目标人数必须≥2 | 修改为≥2 的数字 |
| EX-004 | 未选择优惠券提交 | 请选择优惠券 | 从下拉列表选择有效优惠券 |
| EX-005 | 优惠券已下架 | 选择的优惠券已失效 | 选择其他有效优惠券 |
| EX-006 | 未上传背景图提交 | 请上传活动背景图 | 点击上传按钮上传图片 |
| EX-007 | 未上传卡片图提交 | 请上传转发卡片图 | 点击上传按钮上传图片 |
| EX-008 | 网络请求失败 | 网络异常，请稍后重试 | 检查网络连接后重试 |
| EX-009 | 无操作权限 | 您没有操作权限 | 联系管理员开通权限 |
| EX-010 | 活动已发布后尝试编辑 | 活动已发布，无法编辑 | 取消发布后再编辑或重新创建 |
