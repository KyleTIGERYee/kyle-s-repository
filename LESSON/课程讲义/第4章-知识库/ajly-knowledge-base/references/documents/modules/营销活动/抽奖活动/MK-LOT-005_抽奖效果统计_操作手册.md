> **功能编号**: MK-LOT-005  
> **功能名称**: 抽奖效果统计  
> **所属模块**: 营销活动 - 抽奖活动  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 查看抽奖活动的参与记录和效果统计，包括中奖用户信息、关联业务、奖品发放情况
- **使用场景**: 
  - 运营人员查看抽奖活动参与情况
  - 统计中奖用户数据和奖品发放情况
  - 导出抽奖记录进行数据分析
- **业务价值**: 帮助运营人员了解活动效果，追踪奖品发放，进行数据分析

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 序号 | 数字 | 是 | 自动递增 | 记录序号，从 1 开始 |
| 姓名 | 文本 | 是 | 无 | 参与抽奖的用户姓名 |
| 电话 | 文本 | 是 | 脱敏显示 | 用户手机号，中间 4 位隐藏（如 135****6632） |

| 签约月份 | 文本 | 是 | 无 | 签约的租期月份数（如：1 个月、3 个月等） |
| 奖品 | 文本 | 是 | 无 | 用户中奖的奖品名称（优惠券名称或实体奖名称） |

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 导出 | 对话框顶部 | 导出抽奖记录为 Excel 文件 | 营销活动查看权限 |
| 确定 | 对话框底部 | 关闭统计对话框 | 无 |
| 表单统计 | 活动列表操作列 | 打开抽奖效果统计对话框 | 营销活动查看权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录管理后台，进入"营销推广 → 营销活动"
步骤 2: 点击"抽奖活动"标签页，切换到抽奖活动列表
步骤 3: 找到目标抽奖活动，点击操作列的"表单统计"按钮
步骤 4: 查看抽奖效果统计对话框，包含活动名称、中奖记录列表
步骤 5: 可点击"导出"按钮导出记录为 Excel 文件
步骤 6: 查看完毕后点击"确定"按钮关闭对话框
```

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 活动状态 | 只有已发布或已结束的活动才有统计数据 | 无数据时列表为空 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动无参与记录 | 列表显示为空，分页显示"共 0 条" | 正常情况，说明活动无人参与 |
| EX-002 | 导出失败 | 系统提示"导出失败，请稍后重试" | 检查网络，重新尝试导出 |
| EX-003 | 数据加载失败 | 对话框无法打开或显示 loading 超时 | 刷新页面，重新点击表单统计 |
| EX-004 | 无权限查看 | 操作列不显示"表单统计"按钮 | 联系管理员开通权限 |
