# MK-AST-005 助力效果统计 操作手册

> **功能编号**: MK-AST-005  
> **功能名称**: 助力效果统计  
> **所属模块**: 营销推广 → 营销活动 → 助力活动  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 查看助力活动的整体效果统计数据，包括表单总数、成交总数及详细助力记录
- **使用场景**: 运营人员需要评估助力活动的参与度和转化效果时
- **业务价值**: 通过数据统计分析活动效果，为后续活动优化提供依据

---

## 二、入口方式

### 2.1 导航路径

菜单路径：营销推广 → 营销活动 → 助力活动 → 选择活动 → 点击"表单统计"按钮

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 营销活动 → 助力活动                                       |
+----------------------------------------------------------+
| 筛选区：                                                  |
| 活动名称 | 活动时间 (起 - 止) | 活动状态 |                   |
| [搜索] [重置] [新建活动] [导出]                           |
+----------------------------------------------------------+
| 数据列表：                                                |
| 序号 | 活动名称 | 活动时间 | 活动状态 | 创建时间 |        |
| 表单总数 | 成交总数 | 活动链接 | 操作 |                    |
| 操作：[详情] [表单统计] [发布] [取消发布] [编辑]            |
|              ↑                                           |
|              点击此处进入助力效果统计                     |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 已创建助力活动
- [ ] 活动有参与数据（表单或成交）
- [ ] 拥有营销活动查看权限

## 三、页面结构

### 3.1 页面布局

点击"表单统计"按钮后弹出统计对话框：

```
+----------------------------------------------------------+
| 表单统计                                            [×]  |
+----------------------------------------------------------+
| 活动名称：[活动名称]                                      |
+----------------------------------------------------------+
| 统计概览：                                                |
| +------------------+------------------+                   |
| |   表单总数       |    成交总数      |                   |
| |      [数字]      |      [数字]      |                   |
| +------------------+------------------+                   |
+----------------------------------------------------------+
| 助力记录列表：                                            |
| +----+----------+--------+------------+------------+     |
| |序号|   电话   |角色类型| 邀请人电话 | 成交状态   |     |
| +----+----------+--------+------------+------------+     |
| | 1  | 138****  |助力人  | 139****    | 已成交/未成交 |  |
| | 2  | 139****  |邀请人  | -          | -          |     |
| +----+----------+--------+------------+------------+     |
+----------------------------------------------------------+
| 分页：第 1 页/共 X 条 | 1 2 3 ... 前往 [页] [确认]          |
+----------------------------------------------------------+
```

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 活动名称 | 文本 | 是 | 无 | 显示当前统计的活动名称 |
| 表单总数 | 数字 | 是 | 无 | 活动累计收到的表单数量 |
| 成交总数 | 数字 | 是 | 无 | 活动带来的成交数量 |
| 电话 | 文本 | 是 | 脱敏显示 | 用户手机号，中间 4 位隐藏 |
| 角色类型 | 单选 | 是 | 助力人/邀请人 | 用户在活动中的角色 |
| 邀请人电话 | 文本 | 否 | 脱敏显示 | 仅助力人显示，邀请人为空 |
| 成交状态 | 单选 | 是 | 已成交/未成交 | 该用户是否最终成交 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 关闭 (×) | 右上角 | 关闭统计对话框 | 无 |
| 导出 | 对话框顶部 | 导出助力记录为 Excel | 数据导出权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入营销推广 → 营销活动
步骤 2: 点击"助力活动"标签页
步骤 3: 在助力活动列表中找到目标活动
步骤 4: 点击该活动行的"表单统计"按钮
步骤 5: 查看弹出的统计对话框，包含：
        - 活动名称
        - 表单总数
        - 成交总数
        - 助力记录列表（电话、角色类型、邀请人、成交状态）
步骤 6: 可点击"导出"按钮导出记录
步骤 7: 点击"×"关闭对话框
```

### 4.2 流程图

```
开始
  ↓
进入营销活动→助力活动
  ↓
选择目标活动
  ↓
点击"表单统计"按钮
  ↓
弹出统计对话框
  ↓
查看统计数据
  ↓
[可选] 导出记录
  ↓
关闭对话框
  ↓
结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 活动名称 | 无 | 页面加载时自动显示 |
| 表单总数 | 无 | 页面加载时自动显示 |
| 成交总数 | 无 | 页面加载时自动显示 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 电话 | 11 位手机号脱敏 | 无 | 138****1234 |
| 表单总数 | 正整数 | 无 | 128 |
| 成交总数 | 正整数 | 无 | 45 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 数据权限 | 仅显示当前用户有权限查看的活动数据 | 无数据提示 |
| 角色区分 | 邀请人记录不显示"邀请人电话"字段 | - |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 活动无任何参与数据 | 列表显示为空，表单总数和成交总数均为 0 | 属于正常情况，活动刚创建或无人参与 |
| EX-002 | 活动已被删除 | 提示"活动不存在" | 联系管理员确认活动状态 |
| EX-003 | 无查看权限 | 对话框不弹出或提示无权限 | 申请营销活动查看权限 |
| EX-004 | 导出数据量过大 | 导出时间较长 | 耐心等待或联系技术支持 |
