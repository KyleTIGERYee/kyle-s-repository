---

## 一、功能概述

- **功能目的**: 筛选符合续租活动资格的客户，帮助运营人员精准定位目标客群
- **使用场景**: 运营人员需要查看哪些在租客户符合续租活动的参与条件时
- **业务价值**: 提高续租活动投放精准度，提升续租转化率，减少无效营销

---

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 租客姓名 | 文本 | 否 | 支持模糊搜索 | 按租客姓名筛选 |
| 手机号 | 数字 | 否 | 11 位手机号码格式 | 按手机号精确搜索 |
| 门店 | 下拉选择 | 否 | 从门店列表选择 | 按门店筛选 |
| 楼栋 | 下拉选择 | 否 | 从楼栋列表选择 | 按楼栋筛选，依赖门店选择 |
| 房间号 | 文本 | 否 | 支持模糊搜索 | 按房间号筛选 |
| 合同到期日期 - 起 | 日期 | 否 | 不能为空（如果填写） | 合同到期日期范围起始 |
| 合同到期日期 - 止 | 日期 | 否 | 不能为空（如果填写），必须≥起 | 合同到期日期范围结束 |
| 续租周期 | 下拉选择 | 否 | 从预设选项选择 | 筛选特定续租周期的客户（如 1 年、2 年） |
| 预计居住时长 | 下拉选择 | 否 | 从预设选项选择 | 筛选特定居住时长的客户 |
| 资格状态 | 单选/复选 | 否 | 全部/符合资格/不符合资格 | 按资格状态筛选 |
| 当前租期 | 文本 | 只读 | 系统自动计算 | 显示客户当前合同的租期 |
| 预计居住时长 | 文本 | 只读 | 系统自动计算 | 显示客户预计居住时长 |
| 资格状态 | 文本 + 图标 | 只读 | 系统自动判定 | ✓符合资格 / ✗不符合资格（显示原因） |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 搜索 | 筛选区下方 | 根据筛选条件查询符合条件的客户 | 营销活动查看权限 |
| 重置 | 筛选区下方 | 清空所有筛选条件，恢复默认状态 | 无 |
| 导出 | 页面顶部右侧 | 导出当前筛选结果为客户列表（Excel） | 营销活动导出权限 |
| 关闭 | 页面顶部右侧 | 关闭筛选页面，返回活动详情 | 无 |
| 查看 | 数据列表每行操作列 | 查看客户详细信息 | 无 |
| 手动标记 | 数据列表每行操作列 | 手动标记客户资格状态（需权限） | 营销活动管理权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 进入营销活动 → 续租活动列表
步骤 2: 找到目标续租活动，点击"详情"按钮
步骤 3: 在活动详情页面，点击"续租资格筛选"标签/按钮
步骤 4: （可选）设置筛选条件（姓名/手机号/门店/合同到期日等）
步骤 5: 点击"搜索"按钮，系统显示符合条件的客户列表
步骤 6: 查看客户资格状态（✓符合资格 / ✗不符合资格及原因）
步骤 7: （可选）点击"导出"按钮导出筛选结果
步骤 8: （可选）对特定客户点击"查看"查看详细信息
步骤 9: 点击"关闭"返回活动详情页面
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 手机号（如填写） | 请输入正确的 11 位手机号码 | 失焦/提交 |
| 合同到期日期 - 起（如填写） | 请选择合同到期起始日期 | 失焦/提交 |
| 合同到期日期 - 止（如填写） | 请选择合同到期结束日期 | 失焦/提交 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 手机号 | 11 位数字 | 请输入正确的 11 位手机号码 | 13512345678 |
| 合同到期日期 | YYYY-MM-DD | 日期格式不正确 | 2026-12-31 |
| 房间号 | 字母 + 数字组合 | 请输入正确的房间号 | A-101 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 日期范围 | 合同到期日期 - 止必须≥合同到期日期 - 起 | 结束日期不能早于开始日期 |
| 楼栋依赖 | 选择楼栋前必须先选择门店 | 请先选择门店 |
| 资格判定 | 系统根据活动规则自动判定资格状态 | - |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 未选择活动直接进入筛选 | 请先选择续租活动 | 返回活动列表选择活动 |
| EX-002 | 活动未发布 | 该活动尚未发布，无法查看资格筛选 | 先发布活动或联系管理员 |
| EX-003 | 无符合筛选条件的客户 | 暂无符合条件的客户 | 调整筛选条件或确认活动规则 |
| EX-004 | 导出时网络失败 | 导出失败，请重试 | 检查网络后重试 |
| EX-005 | 无权限查看资格筛选 | 您没有权限查看此功能 | 联系管理员开通权限 |
| EX-006 | 数据加载超时 | 数据加载超时，请刷新页面 | 刷新页面或联系技术支持 |
