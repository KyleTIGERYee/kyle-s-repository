> **操作频率**: 中频  
> **最后更新**: 2026-03-22  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 提供资产数据的多维度统计分析，包括资产分类统计、状态统计、分布统计、价值统计等，支持管理决策
- **使用场景**: 资产盘点前的数据统计、资产采购计划制定、资产折旧核算、管理层汇报等场景
- **业务价值**: 帮助管理者全面掌握资产状况，识别资产管理问题，支持资产优化和更新决策

| 字段编号 | 字段名称 | 字段类型 | 是否必填 | 校验规则 | 默认值 | 说明 |

### 3.3 统计指标说明

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| 资产总数 | 所有资产记录数 | 系统中资产的总数量 |
| 空闲资产 | 状态=空闲的资产数 | 已入库但未分配的资产数量 |
| 已分配资产 | 状态=已分配的资产数 | 已分配到具体房源/房间的资产数量 |
| 损坏资产 | 状态=损坏的资产数 | 已报修且确认损坏的资产数量 |
| 已报废 | 状态=已报废的资产数 | 已报废的资产数量 |
| 资产总原值 | ∑(资产采购价 × 数量) | 所有资产的原始采购总价值 |
| 累计折旧 | ∑(资产原值 - 资产净值) | 所有资产已计提的折旧总额 |
| 资产净值 | 资产总原值 - 累计折旧 | 所有资产的当前净值 |
| 平均使用年限 | ∑(资产已使用月数) / 资产总数 | 资产的平均已使用时间 |

## 四、操作流程

### 4.1 查看整体统计

1. 进入租赁管理 → 资产管理 → 资产统计页面
2. 查看统计卡片区（资产总数、状态分布）
3. 查看价值统计卡片（资产原值、累计折旧、资产净值）
4. 查看统计图表区（分类占比、状态分布）
5. 查看统计明细表（按分类汇总）

### 4.2 按条件筛选统计

1. 选择统计维度（如"已分配资产"）
2. 选择统计日期（如"2026-03-22"）
3. 选择物业地址（如"XX 大厦"）
4. 选择资产分类（如"家电类"）
5. 点击「搜索」按钮
6. 查看筛选后的统计数据

### 4.3 查看趋势分析

1. 进入资产统计页面
2. 查看"资产月度趋势"折线图
3. 切换时间范围（近 6 个月/近 1 年/自定义）
4. 分析资产数量变化趋势

### 4.4 导出统计报表

1. 执行筛选（可选）
2. 点击「导出统计报表」按钮
3. 系统创建后台导出任务
4. 提示"导出任务已创建，您可以稍后在导出中心查看导出进度和下载文件"
5. 前往数据导出中心下载 Excel 文件（包含统计卡片、图表数据、明细表）

| 其他类 | - | - | 不属于以上分类的其他资产 |

### 5.2 按资产状态统计

| 状态 | 说明 | 管理重点 |
|------|------|----------|
| 空闲 | 已入库但未分配 | 提高资产利用率，及时分配 |
| 已分配 | 已分配到具体房源/房间 | 确保资产正常使用，定期盘点 |
| 损坏 | 已报修且确认损坏 | 及时维修或报废，避免影响使用 |
| 已报废 | 已报废的资产 | 规范处置，更新资产台账 |

### 5.3 按物业地址统计

| 维度 | 说明 | 用途 |
|------|------|------|
| 城市 | 资产所在城市 | 区域资产分布分析 |
| 物业地址 | 具体物业名称 | 物业资产管理 |
| 座栋/单元 | 物业内具体位置 | 精细化资产管理 |

## 八、常见问题

**Q1: 统计数据多久更新一次？**  
A1: 统计数据实时计算，每次进入页面或点击"刷新数据"时重新计算。

**Q2: 统计日期可以选择未来日期吗？**  
A2: 不可以。统计日期只能选择今天及之前的日期。

**Q3: 资产净值如何计算？**  
A3: 资产净值 = 资产原值 - 累计折旧。折旧按资产类别和使用年限自动计算。

**Q4: 导出统计报表包含图表吗？**  
A4: 导出的 Excel 文件包含统计卡片数据、明细表数据。图表以数据形式呈现，不包含图片。

**Q5: 如何查看某个物业的详细资产统计？**  
A5: 在筛选查询区选择该物业地址，点击搜索，即可查看该物业的统计数据。

**Q6: 统计明细表可以导出吗？**  
A6: 可以。点击"导出统计报表"后，Excel 文件包含统计明细表。

**Q7: 平均使用年限如何计算？**  
A7: 平均使用年限 = ∑(资产已使用月数) / 资产总数，结果以年为单位显示。

**Q8: 统计数据与资产查询结果不一致怎么办？**  
A8: 请确认筛选条件是否一致。如仍不一致，请联系管理员检查数据同步。
