## 一、功能概述

- **功能目的**: 通过 Excel 模板批量导入资产数据，快速录入大量资产信息，提高资产入库效率
- **使用场景**: 新公寓开业资产批量录入、资产盘点后批量补录、系统迁移资产数据导入等场景
- **业务价值**: 避免手动逐条录入的低效操作，支持大批量资产快速入库，减少人工录入错误

## 四、操作流程

### 4.1 正常流程

1. 进入租赁管理 → 资产管理页面
2. 点击「批量导入」按钮，打开批量导入面板
3. 点击「下载」按钮下载资产导入模板（Excel 格式）
4. 打开模板文件，按格式填写资产数据
5. 保存 Excel 文件（确保格式为 xls 或 xlsx）
6. 返回批量导入面板
7. 拖拽文件到上传区域，或点击「选择文件」按钮选择文件
8. 系统开始上传并解析文件
9. 系统校验数据格式和内容
10. 校验通过后，系统批量导入资产数据
11. 导入完成后，显示导入结果（成功数量/失败数量）
12. 如有失败数据，点击「下载失败数据」查看原因
13. 修正失败数据后，可重新上传

### 4.2 模板填写流程

1. 下载资产导入模板（Excel 格式）
2. 打开模板文件，阅读填写说明（通常在第一个工作表）
3. 在数据工作表中逐行填写资产信息
4. 确保必填字段（物业地址、门牌号、资产名称、数量、单位）已填写
5. 确保数据格式正确（日期格式、数字格式等）
6. 不要修改模板的列标题和格式
7. 保存文件（确保格式为 xls 或 xlsx）

### 4.3 异常流程

| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 无导入权限 | 当前用户无批量导入权限 | 按钮不显示或点击提示无权限 | 联系管理员授权 |
| 文件格式错误 | 上传非 xls/xlsx 格式文件 | 提示"仅支持 xls、xlsx 格式文件" | 转换文件格式后重新上传 |
| 文件大小超限 | 文件>10MB | 提示"文件大小不能超过 10MB" | 拆分文件后分批上传 |
| 数据量超限 | 单次导入>1000 条 | 提示"单次最多导入 1000 条数据" | 拆分文件后分批上传 |
| 必填字段缺失 | 模板中必填字段未填写 | 导入失败，提示缺失字段 | 补充必填字段后重新上传 |
| 数据格式错误 | 日期/数字格式不正确 | 导入失败，提示格式错误 | 修正格式后重新上传 |
| 物业地址不存在 | 填写的物业地址系统中不存在 | 导入失败，提示地址无效 | 确认物业地址或先添加物业 |
| 重复数据 | 导入数据与现有资产重复 | 提示重复数据，跳过或覆盖（可配置） | 确认是否需要导入 |
| 网络异常 | 上传时网络中断 | 提示"上传失败，请重试" | 检查网络后重试 |

| 数量 | 不能为空，>0 的整数 | "数量必须为正整数" |
| 单位 | 不能为空，1-20 字符 | "单位不能为空" |

| 字段名称 | 校验规则 | 错误提示 |
|----------|----------|----------|
| 数量 | 必须为正整数 | "数量格式错误，必须为正整数" |
| 采购价 | 必须为≥0 的数字，保留 2 位小数 | "采购价格式错误" |

| 日期范围 | 不能晚于今天 | "购入日期不能晚于今天" |

| 校验项 | 校验规则 | 错误提示 |
|--------|----------|----------|
| 物业地址 | 必须是系统中已存在的物业地址 | "物业地址不存在，请先添加物业" |
| 资产分类 | 一类/二类/三类必须是系统中已存在的分类 | "资产分类不存在" |
| 单位 | 必须是标准单位（台/个/张/件等） | "单位格式错误" |

## 八、常见问题

**Q1: 导入模板可以修改列标题吗？**  
A1: 不可以。修改列标题会导致系统无法识别字段，导入失败。请保持模板原格式。

**Q2: 单次最多可以导入多少条数据？**  
A2: 单次最多导入 1000 条数据。如需导入更多数据，请拆分文件后分批上传。

**Q3: 导入失败后如何查看原因？**  
A3: 导入完成后，在上传记录列表中找到对应记录，点击"下载失败数据"可下载包含失败原因标注的 Excel 文件。

**Q4: 导入的资产默认状态是什么？**  
A4: 导入的资产默认状态为"空闲"，可在导入后通过资产分配功能分配到具体房间。

**Q5: 导入时可以自动分配资产到房间吗？**  
A5: 当前版本不支持。导入的资产状态为"空闲"，需后续通过资产分配功能手动分配。

**Q6: 导入过程中可以取消吗？**  
A6: 导入开始后无法取消，需等待导入完成。如文件较大，导入可能需要几分钟时间。

**Q7: 导入的资产如何删除？**  
A7: 如导入错误，可在资产管理列表中逐个删除，或联系管理员通过后台批量删除。

**Q8: 模板中的下拉选项可以修改吗？**  
A8: 不建议修改。模板中的下拉选项是系统标准数据，修改后可能导致导入失败。

**Q9: 导入记录保存多久？**  
A9: 导入记录永久保存，可随时查看历史导入情况。

**Q10: 支持 Mac 版 Excel 编辑的模板吗？**  
A10: 支持。模板为标准 Excel 格式（xlsx），Windows 和 Mac 版 Excel 均可编辑。
