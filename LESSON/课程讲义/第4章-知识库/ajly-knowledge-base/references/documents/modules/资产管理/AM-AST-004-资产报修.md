## 一、功能概述

- **功能目的**: 当资产（家具、家电、设备等）出现损坏或故障时，创建报修记录并发起维修流程
- **使用场景**: 资产损坏需要维修、资产故障需要更换、定期保养维护等场景
- **业务价值**: 实现资产维修全流程管理，追踪资产维修历史，支持维修成本统计和资产寿命分析

## 四、操作流程

### 4.1 正常流程

1. **进入资产管理页面**
   - 点击左侧菜单"租赁管理" → "资产管理"

2. **找到目标资产**
   - 使用筛选查询区定位目标资产（物业地址/门牌号/资产分类等）
   - 或在数据列表中浏览找到目标资产

3. **打开资产报修弹窗**
   - 点击目标资产操作列的"操作"按钮
   - 在下拉菜单中选择"资产报修"选项

4. **确认资产信息**
   - 系统自动填充资产信息（资产编号/物业地址/门牌号/摆放位置/资产分类/品牌型号/采购日期/保修期限）
   - 确认资产信息无误

5. **填写报修信息**
   - 选择报修类型（必填，如"自然损耗"）
   - 选择紧急程度（必填，如"一般"）
   - 填写故障描述（必填，500 字以内，详细描述故障现象）

6. **上传附件**
   - 上传故障照片（可选，最多 9 张，建议上传清晰照片展示故障部位）
   - 上传故障视频（可选，最多 3 个，适用于动态故障展示）

7. **填写处理信息**
   - 选择期望处理时间（可选，如勾选"紧急"建议填写）
   - 选择维修人员（可选，如不选择后续由管理员指派）
   - 填写备注信息（可选，300 字以内）

8. **选择是否创建工单**
   - 如需立即安排维修，勾选"同时创建维修工单"
   - 如仅需记录报修，暂不安排维修，不勾选

9. **提交报修**
   - 点击"提交报修"按钮
   - 系统校验必填字段和格式
   - 校验通过后保存成功，提示"报修提交成功"
   - 如勾选创建工单，自动跳转到工单创建页面

10. **查看报修记录**
    - 报修提交后，可在资产详情页查看报修历史
    - 或在工单管理模块查看关联的维修工单（如创建）

### 4.2 异常流程

| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 必填字段未填 | 点击提交时有必填字段为空 | 提示"请填写必填项"并高亮缺失字段 | 补充完整必填信息后重新提交 |
| 报修类型未选择 | 报修类型字段为空 | 提示"请选择报修类型" | 选择合适的报修类型 |
| 紧急程度未选择 | 紧急程度未选择 | 提示"请选择紧急程度" | 选择紧急/一般/不紧急 |
| 故障描述为空 | 故障描述字段为空 | 提示"请填写故障描述" | 详细描述故障现象 |
| 故障描述超长 | 故障描述>500 字 | 提示"故障描述不能超过 500 字" | 精简故障描述内容 |
| 图片格式错误 | 上传非 png/jpg/GIF 格式 | 提示"仅支持 png/jpg/GIF 格式" | 转换图片格式后重新上传 |
| 图片大小超限 | 单张图片>2M | 提示"单张图片不能超过 2M" | 压缩图片后重新上传 |
| 图片数量超限 | 故障照片>9 张 | 提示"最多上传 9 张图片" | 删除多余图片 |
| 视频格式错误 | 上传非 mp4 格式 | 提示"仅支持 mp4 格式" | 转换视频格式后重新上传 |
| 视频大小超限 | 单个视频>50M | 提示"单个视频不能超过 50M" | 压缩视频后重新上传 |
| 视频数量超限 | 故障视频>3 个 | 提示"最多上传 3 个视频" | 删除多余视频 |
| 期望时间早于当前时间 | 选择的期望时间 < 当前时间 | 提示"期望时间不能早于当前时间" | 选择当前时间之后的时间 |
| 备注超长 | 备注>300 字 | 提示"备注不能超过 300 字" | 精简备注内容 |
| 资产已报废 | 尝试对已报废资产报修 | 操作列不显示报修选项或提示"已报废资产不支持报修" | 已报废资产应走报废流程而非报修 |
| 无报修权限 | 当前用户无资产管理权限 | 操作列不显示报修选项或点击无响应 | 联系管理员分配权限 |
| 网络异常 | 提交时网络中断 | 提示"操作失败，请重试" | 检查网络后重新提交 |

## 五、状态变更

资产报修后的状态流转：

```
使用中 --[报修提交]--> 维修中 --[维修完成]--> 使用中
                          |
                          +---> [无法修复] --> 已报废
```

报修单状态流转：

```
待受理 --[受理]--> 处理中 --[维修完成]--> 待验收 --[验收通过]--> 已完成
                                              |
                                              +---> [验收不通过] --> 处理中
```

## 八、常见问题

**Q1: 报修后如何查看维修进度？**  
A1: 报修提交后，可在以下位置查看进度：
- 资产详情页：查看该资产的报修历史记录
- 工单管理模块：如创建了工单，可在工单列表查看工单状态
- 系统消息：维修状态变更时会发送通知

**Q2: 报修类型选择错误可以修改吗？**  
A2: 报修提交后，报修类型通常不支持直接修改。如需修改，可联系管理员在后台调整，或撤销报修后重新提交。

**Q3: 如何判断是否在保修期内？**  
A3: 系统会自动根据资产的采购日期和保修期限计算是否在保修期内。报修弹窗中会显示"保修期限"字段，如当前日期在保修期内，系统可能自动标记为"保修期内"，维修费用可能由供应商承担。

**Q4: 报修后多久会有人处理？**  
A4: 处理时效取决于紧急程度：
- 紧急：2 小时内响应
- 一般：24 小时内响应
- 不紧急：72 小时内响应
如超过时效未处理，系统可能自动升级或发送催办通知。

**Q5: 人为损坏的报修如何处理？**  
A5: 选择报修类型="人为损坏"后，系统可能要求填写责任人信息。维修费用可能由责任人承担，具体根据公司政策执行。如无法确定责任人，可选择"其他"并备注说明。

**Q6: 报修后可以取消吗？**  
A6: 报修提交后，如维修尚未开始，报修人通常可以取消报修。如已创建工单且工单已派单，需先撤回工单再取消报修，或联系管理员处理。
