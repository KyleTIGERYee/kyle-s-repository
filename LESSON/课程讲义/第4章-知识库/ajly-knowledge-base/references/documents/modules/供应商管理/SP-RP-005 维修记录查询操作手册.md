## 一、功能概述

- **功能目的**: 查询维修供应商列表及合同信息，支持按多维度筛选和查看
- **使用场景**: 查看维修供应商合作状态、联系信息、质保时长、关联楼栋等
- **业务价值**: 快速定位目标维修供应商，管理供应商合同生命周期

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 供应商 | 下拉选择 | 否 | 无 | 支持模糊搜索供应商名称 |
| 供应商状态 | 下拉选择 | 否 | 无 | 已通过、待审批、已过期、已禁用 |
| 联系人名称 | 文本输入 | 否 | 无 | 支持模糊搜索 |
| 联系人手机号 | 文本输入 | 否 | 11 位手机号格式 | 支持模糊搜索 |
| 启用状态 | 下拉选择 | 否 | 无 | 已启用、已禁用 |
| 序号 | 只读 | - | 自动编号 | 列表行号 |
| 供应商 | 只读 | - | 无 | 供应商名称 |
| 联系人 | 只读 | - | 无 | 联系人姓名 |
| 联系人手机号 | 只读 | - | 脱敏显示 | 中间 4 位隐藏（如 151****5255） |
| 状态 | 只读 | - | 无 | 已通过、待审批、已过期 |
| 启用状态 | 只读 | - | 无 | 已启用、已禁用、-（未设置） |
| 质保时长 | 只读 | - | 无 | 如 3 年、2 年 |
| 关联楼栋 | 操作按钮 | - | 无 | 点击查看相关楼栋 |

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|

| 新增供应商合同 | 操作区 | 打开新增供应商合同对话框 | 供应商管理权限 |
| 审批配置 | 操作区 | 配置供应商合同审批流程 | 供应商管理权限 |
| 查看相关楼栋 | 操作列 | 查看该供应商关联的楼栋列表 | 供应商管理权限 |
| 查看 | 操作列 | 查看供应商合同详情 | 供应商管理权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，进入租赁管理 → 供应商管理
步骤 2: 点击"维修供应商"标签页切换到维修供应商列表
步骤 3: （可选）在筛选区输入查询条件
步骤 4: 点击"搜索"按钮执行查询
步骤 5: 在列表中查看供应商信息
步骤 6: （可选）点击操作列按钮执行查看/编辑/禁用等操作
步骤 7: （可选）点击"导出"按钮导出数据
```

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 无操作权限 | 按钮不可见或点击无响应 | 联系管理员开通权限 |
| EX-002 | 网络超时 | 网络繁忙，请稍后再试 | 检查网络连接后重试 |

| EX-004 | 导出失败 | 导出失败，请稍后重试 | 检查浏览器下载权限和网络连接 |
| EX-005 | 数据加载失败 | 页面显示加载失败 | 刷新页面或联系技术支持 |
