## 一、功能概述

- **功能目的**: 新增维修供应商合同，将维修供应商录入系统并设置基本信息
- **使用场景**: 当需要与新的维修供应商（如电梯维保、建筑施工、设备维修等）合作时，通过此功能录入供应商信息
- **业务价值**: 建立维修供应商档案，为后续维修派单、供应商考核、结算等业务提供基础数据支持

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航至 租赁管理 → 供应商管理 → 维修供应商
步骤 2: 确认当前在"维修供应商"标签页
步骤 3: 点击"新增供应商合同"按钮
步骤 4: 在弹出的对话框中填写基本信息：
        - 输入供应商名称
        - 输入质保时长（年 + 月）
        - 输入联系人名称
        - 输入联系人手机号
步骤 5: （可选）选择质保项目
步骤 6: 点击"下一步"按钮
步骤 7: 系统验证必填字段，如有错误提示修正
步骤 8: 进入下一步流程（合同详情录入）
```

### 4.2 流程图

```
开始 → 导航至维修供应商页面 → 点击"新增供应商合同"
       ↓
   填写基本信息
       ↓
   供应商名称 → 必填校验
   质保时长 → 必填校验
   联系人名称 → 必填校验
   联系人手机号 → 必填校验 + 格式校验
       ↓
   （可选）选择质保项目
       ↓
   点击"下一步"
       ↓
   校验通过？ → 否 → 显示错误提示 → 修正输入
       ↓ 是
   进入下一步流程 → 结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 供应商名称 | 系统提示（具体文案待验证） | 提交时 |
| 质保时长 | 系统提示（具体文案待验证） | 提交时 |
| 联系人名称 | 系统提示（具体文案待验证） | 提交时 |
| 联系人手机号 | 系统提示（具体文案待验证） | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 联系人手机号 | 11 位手机号码 | 待验证 | 13800138000 |
| 质保时长 - 年 | 正整数 | 待验证 | 1, 2, 3 |
| 质保时长 - 月 | 0-11 的整数 | 待验证 | 0, 6, 12 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 供应商名称唯一性 | 待验证是否与现有供应商重名 | 待验证 |
| 手机号唯一性 | 待验证是否已存在于其他供应商 | 待验证 |
| 质保项目有效性 | 必须从下拉列表中选择 | 待验证 |

## 九、注意事项

1. **供应商名称**: 建议使用公司全称，便于后续识别和管理
2. **质保时长**: 根据实际合同约定填写，年 + 月组合表示
3. **联系人手机号**: 必须填写有效的 11 位手机号码，用于后续联系
4. **质保项目**: 根据供应商实际提供的服务项目选择，可多选
5. **权限要求**: 需要供应商管理权限才能执行此操作
6. **数据保存**: 点击"下一步"后进入合同详情录入，需完成全部流程才能保存
