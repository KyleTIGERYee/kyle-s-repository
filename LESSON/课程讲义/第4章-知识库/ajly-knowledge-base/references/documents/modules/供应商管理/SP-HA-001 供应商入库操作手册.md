> **功能编号**: SP-HA-001  
> **功能名称**: 供应商入库  
> **所属模块**: 供应商管理 - 家电家具供应商  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 新增家电家具供应商信息，完成供应商入库操作
- **使用场景**: 当需要与新的家电家具供应商合作时，通过此功能录入供应商基本信息
- **业务价值**: 建立供应商档案，为后续合同管理、派单、结算提供基础数据支持

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 供应商名称 | 文本 | 是 | 不能为空，最多 50 字符 | 供应商公司或个人名称 |
| 统一社会信用代码 | 文本 | 是 | 18 位字符，字母数字组合 | 工商注册号/统一社会信用代码 |
| 签约公司主体 | 下拉选择 | 是 | 必须从列表选择 | 与公司签约的主体名称 |
| 合同时间 - 开始日期 | 日期 | 是 | 格式 YYYY-MM-DD，不能晚于结束日期 | 合同生效日期 |
| 合同时间 - 结束日期 | 日期 | 是 | 格式 YYYY-MM-DD，不能早于开始日期 | 合同到期日期 |
| 联系人名称 | 文本 | 是 | 不能为空，最多 20 字符 | 供应商联系人姓名 |
| 联系人手机号 | 文本 | 是 | 11 位手机号格式 | 联系人手机号码 |
| 质保类型 | 单选 | 是 | 二选一：存在质保/不存在质保 | 是否提供质保服务 |
| 质保时间 - 开始日期 | 日期 | 条件必填 | 选择"存在质保"时必填 | 质保生效日期 |
| 质保时间 - 结束日期 | 日期 | 条件必填 | 选择"存在质保"时必填 | 质保到期日期 |
| 关联业务 | 下拉选择 | 否 | 可选择关联的楼栋或业务 | 供应商服务的业务范围 |
| 合同附件 | 文件上传 | 否 | 支持 PDF/JPG/PNG 格式 | 上传合同扫描件 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 下一步 | 对话框底部 | 进入下一步填写详细信息 | 供应商管理权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 导航至租赁管理 → 供应商管理 → 家电家具供应商
步骤 2: 点击"新增供应商合同"按钮
步骤 3: 在弹出的对话框中填写供应商基本信息
步骤 4: 选择质保类型（存在质保/不存在质保）
步骤 5: 如选择"存在质保"，填写质保时间范围
步骤 6: 选择关联业务（可选）
步骤 7: 上传合同附件（可选）
步骤 8: 点击"下一步"按钮进入下一步
步骤 9: 完成后续步骤后提交
步骤 10: 系统显示"保存成功"提示
```

### 4.2 流程图

```
开始
  ↓
导航至供应商管理页面
  ↓
点击"新增供应商合同"
  ↓
填写基本信息（供应商名称/统一信用代码/签约主体/合同时间/联系人）
  ↓
选择质保类型
  ↓
是存在质保？ → 填写质保时间
  ↓
选择关联业务（可选）
  ↓
上传合同附件（可选）
  ↓
点击"下一步"
  ↓
完成后续步骤
  ↓
提交保存
  ↓
结束
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 供应商名称 | 请输入供应商名称 | 提交时 |
| 统一社会信用代码 | 请输入统一社会信用代码 | 提交时 |
| 签约公司主体 | 请选择签约公司主体 | 提交时 |
| 合同时间 - 开始日期 | 请选择合同开始日期 | 提交时 |
| 合同时间 - 结束日期 | 请选择合同结束日期 | 提交时 |
| 联系人名称 | 请输入联系人名称 | 提交时 |
| 联系人手机号 | 请输入联系人手机号 | 提交时 |
| 质保时间（存在质保时） | 请选择质保时间 | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 统一社会信用代码 | 18 位字母数字组合 | 统一社会信用代码格式不正确 | 91440300MA5XXXXX |
| 联系人手机号 | 11 位手机号 | 手机号格式不正确 | 13800138000 |
| 合同开始日期 | YYYY-MM-DD | 日期格式不正确 | 2026-01-01 |
| 合同结束日期 | YYYY-MM-DD | 日期格式不正确 | 2027-12-31 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 合同时间范围 | 开始日期不能晚于结束日期 | 合同开始日期不能晚于结束日期 |
| 质保时间范围 | 开始日期不能晚于结束日期 | 质保开始日期不能晚于结束日期 |
| 统一信用代码唯一性 | 不能与现有供应商重复 | 该统一社会信用代码已存在 |
| 质保时间逻辑 | 选择"不存在质保"时质保时间为空 | - |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 统一信用代码已存在 | 该供应商已存在 | 检查是否重复录入，使用编辑功能修改现有供应商 |
| EX-002 | 必填字段未填写 | 请填写必填字段 | 补充完整所有必填字段后重新提交 |
| EX-003 | 手机号格式错误 | 手机号格式不正确 | 检查手机号是否为 11 位数字 |
| EX-004 | 合同时间范围错误 | 开始日期不能晚于结束日期 | 调整日期范围 |
| EX-005 | 网络超时 | 网络繁忙，请稍后再试 | 检查网络连接后重试 |
| EX-006 | 无操作权限 | 无权限执行此操作 | 联系管理员分配供应商管理权限 |
| EX-007 | 文件上传失败 | 文件上传失败，请重试 | 检查文件格式和大小后重试 |
