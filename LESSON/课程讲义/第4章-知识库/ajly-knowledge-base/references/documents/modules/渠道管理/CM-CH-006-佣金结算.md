## 一、功能概述

佣金结算用于执行已审批通过的结算单的付款操作，记录付款信息并更新结算状态，是渠道佣金支付的最终环节。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限
- 系统中已存在待付款的结算单（CM-CH-005）

## 三、操作步骤

### 3.1 进入佣金结算页面

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 点击【佣金结算】标签页
4. 查看结算单列表

### 3.2 筛选待付款结算单

1. 在结算状态筛选中选择「待付款」
2. 点击「搜索」按钮
3. 列表显示所有待付款的结算单

### 3.3 发起结算

1. 找到目标结算单
2. 点击「结算」按钮
3. 弹出佣金结算表单对话框
4. 系统自动带出结算单信息

### 3.4 填写付款信息

| 字段 | 类型 | 必填 | 说明 | 校验规则 |
|------|------|------|------|----------|
| 付款方式 | 单选按钮 | ✅ | 选择付款渠道 | 默认"银行转账" |
| 收款账户 | 文本输入 | ✅ | 收款方账号 | 自动带出，可修改 |
| 开户行 | 文本输入 | ❌ | 收款方开户行 | 自动带出 |
| 付款日期 | 日期选择 | ✅ | 实际付款日期 | 不能晚于当前日期 |
| 付款备注 | 文本输入 | ❌ | 付款说明 | 最多 200 字 |

### 3.5 上传付款凭证

1. 点击「选择文件」按钮
2. 选择银行转账截图或回单
3. 支持 JPG、PNG、PDF 格式
4. 文件大小不超过 10MB
5. 凭证为必填项

### 3.6 确认结算

1. 检查所有必填字段已填写
2. 确认付款凭证已上传
3. 点击「确认结算」按钮
4. 系统校验数据合法性
5. 校验通过则结算成功，提示"佣金结算成功"
6. 自动关闭对话框，返回列表页
7. 结算单状态更新为"已付款"

## 六、结算状态流转

```
新建结算单 → 待审批 → 已驳回 ─┐
              ↓               │
            待付款 ←──────────┘
              ↓
            已付款（终态）
```

| 状态 | 说明 | 可操作 |
|------|------|--------|
| 待审批 | 已提交，等待审批 | 撤销、修改 |
| 已驳回 | 审批未通过 | 修改后重新提交 |
| 待付款 | 审批通过，等待付款 | 执行结算 |
| 已付款 | 已完成付款 | 查看，不可修改 |

## 七、常见问题

### Q1: 结算后可以修改吗？
**A**: 不可以。已付款的结算单为终态，不能修改。如确需修改，需联系管理员进行后台处理。

### Q2: 付款凭证可以补传吗？
**A**: 不可以。付款凭证是结算的必要条件，必须在结算时上传。

### Q3: 可以批量结算多个结算单吗？
**A**: 当前版本不支持批量结算，需逐个处理。建议联系管理员通过后台批量处理。

### Q4: 付款方式有哪些？
**A**: 支持银行转账、支付宝、微信三种付款方式。默认选中银行转账。

### Q5: 结算后如何查看记录？
**A**: 在佣金结算页面，将结算状态筛选为「已付款」，即可查看历史结算记录。

## 八、注意事项

1. **凭证必填**: 付款凭证是结算的必要条件，必须上传
2. **信息准确**: 收款账户信息务必准确，避免付款失败
3. **日期真实**: 付款日期应填写实际付款日期
4. **状态确认**: 结算前确认结算单状态为「待付款」
5. **及时结算**: 审批通过后及时执行结算，避免延误
