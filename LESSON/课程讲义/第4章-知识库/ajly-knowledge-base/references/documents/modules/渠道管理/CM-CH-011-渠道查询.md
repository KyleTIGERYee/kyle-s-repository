## 一、功能概述

渠道查询用于快速查找和筛选渠道信息，支持多维度筛选和模糊搜索，方便管理渠道数据和查看渠道详情。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限
- 系统中存在渠道数据（CM-CH-001）

## 三、操作步骤

### 3.1 进入渠道查询页面

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 点击【渠道查询】标签页
4. 查看渠道列表

### 3.2 设置筛选条件

| 筛选条件 | 类型 | 说明 |
|----------|------|------|
| 关键词 | 文本输入 | 支持渠道名称、负责人、电话模糊搜索 |
| 渠道类型 | 下拉选择 | 筛选特定渠道类型 |
| 合作状态 | 下拉选择 | 合作中/暂停/终止 |
| 战区 | 下拉选择 | 筛选特定战区 |
| 评级 | 下拉选择 | A/B/C/D 级 |

### 3.3 执行查询

1. 设置筛选条件
2. 点击「查询」按钮
3. 列表显示符合条件的渠道

### 3.4 查看渠道列表

列表显示以下字段：

| 字段 | 说明 |
|------|------|
| 序号 | 列表序号 |
| 渠道 | 渠道名称 |
| 类型 | 渠道类型分类 |
| 负责人 | 渠道对接人 |
| 状态 | 合作状态 |
| 评级 | 最新评级等级 |
| 操作 | 详情/编辑按钮 |

### 3.5 查看渠道详情

1. 找到目标渠道
2. 点击「详情」按钮
3. 弹出渠道详情弹窗
4. 查看完整信息

### 3.6 导出数据

1. 点击「导出」按钮
2. 选择导出格式（Excel）
3. 系统生成渠道列表文件
4. 下载导出文件

## 四、筛选规则

### 4.1 关键词搜索

```
支持模糊匹配以下字段:
- 渠道名称
- 负责人姓名
- 联系电话（支持后 4 位）
```

### 4.2 多条件组合

```
多个筛选条件为"与"关系:
- 渠道类型 = X AND
- 合作状态 = Y AND
- 战区 = Z AND
- 评级 = A
```

### 4.3 排序规则

```
默认排序:
1. 合作状态（合作中优先）
2. 评级等级（A>B>C>D）
3. 渠道名称（字母顺序）
```

## 六、常见问题

### Q1: 关键词搜索支持哪些字段？
**A**: 支持渠道名称、负责人姓名、联系电话（后 4 位）模糊搜索。

### Q2: 如何查看已终止的渠道？
**A**: 在合作状态筛选中选择「终止」，点击查询即可。

### Q3: 导出数据包含哪些字段？
**A**: 导出 Excel 包含渠道全部字段，包括基本信息、结算信息、统计数据。

### Q4: 渠道详情中的统计数据如何计算？
**A**: 统计数据来自带看、成交、结算记录，实时计算汇总。

### Q5: 列表可以自定义显示字段吗？
**A**: 当前版本字段固定，不支持自定义。建议联系管理员通过后台配置。

## 七、注意事项

1. **权限控制**: 不同角色可能看到不同范围的渠道数据
2. **数据隐私**: 联系电话等敏感信息注意脱敏显示
3. **导出限制**: 大批量导出时可能需要等待，建议分批导出
4. **数据准确性**: 统计数据可能有延迟，以实际为准
5. **筛选优化**: 多条件组合查询时，建议先缩小范围再精确筛选
