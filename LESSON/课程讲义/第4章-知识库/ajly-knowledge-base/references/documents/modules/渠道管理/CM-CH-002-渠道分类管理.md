## 一、功能概述

渠道分类管理用于维护系统中的渠道类型分类体系，支持新增、编辑、删除渠道分类，便于对渠道进行标准化管理和统计分析。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限
- 建议管理员角色操作

## 三、操作步骤

### 3.1 查看渠道分类

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 点击【渠道分类】标签页
4. 查看现有渠道分类列表

**实测渠道分类体系**（基于系统现有数据）:

| 一级分类 | 二级分类（示例） |
|----------|------------------|
| 公司 - 自有渠道 | 小程序、400、公众号、企业客户、老带新、全民营销、深梦扬帆、线上广告投放、线下广告投放、其他 |
| 公司 - 新媒体渠道 | 抖音、小红书、其他（微博、B 站等） |
| 公司 - 异业合作 | - |
| 个人 - 垂类平台 | 贝壳、58、其他闲鱼/豆瓣等 |
| 个人 - 新媒体渠道 | 抖音、小红书、其他微博/B 站等 |
| 个人 - 自有渠道 | 自然来访、其他老客回归/地推/宣讲等 |
| 个人 - 异业合作 | - |
| 中介 | 蜂猴、智享、寓联、馨洋 |
| 在线找房平台 | - |
| 存量客户迁移 | - |

### 3.2 新增渠道分类

1. 点击页面右上角「新增分类」按钮
2. 弹出新增分类表单对话框
3. 填写分类信息：
   - **分类名称**（必填）：输入分类名称
   - **父级分类**（选填）：选择上级分类，不选则为顶级分类
   - **排序**（选填）：输入数字，越小越靠前
   - **备注**（选填）：补充说明
4. 点击「保存」按钮
5. 系统校验数据合法性
6. 保存成功，提示"分类添加成功"
7. 新分类显示在列表中

### 3.3 编辑渠道分类

1. 在分类列表中找到目标分类
2. 点击「编辑」按钮
3. 修改分类信息（分类名称、父级分类、排序、备注）
4. 点击「保存」按钮
5. 更新成功，提示"分类更新成功"

### 3.4 删除渠道分类

1. 在分类列表中找到目标分类
2. 点击「删除」按钮
3. 系统弹出确认对话框："确定要删除该分类吗？删除后关联的渠道将受到影响。"
4. 点击「确认删除」
5. 删除成功，提示"分类删除成功"

**注意**: 如果分类下已有渠道关联，系统应阻止删除或要求先转移渠道。

## 七、常见问题

### Q1: 渠道分类可以有几级？
**A**: 系统支持两级分类（一级分类 + 子分类）。建议保持扁平结构，便于管理和统计。

### Q2: 修改分类名称会影响已关联的渠道吗？
**A**: 会同步更新。修改分类名称后，所有关联该分类的渠道会自动使用新名称。

### Q3: 分类排序如何生效？
**A**: 排序数字越小，在列表和下拉选项中越靠前。相同排序按创建时间倒序。

### Q4: 为什么无法删除某个分类？
**A**: 如果分类下已有渠道关联，系统会阻止删除。需先将渠道转移至其他分类或删除渠道后再删除分类。

### Q5: 可以批量导入渠道分类吗？
**A**: 当前版本不支持批量导入，需逐个添加。建议联系管理员通过后台配置。

## 八、注意事项

1. **谨慎删除**: 删除分类前确认无渠道关联，避免数据丢失
2. **命名规范**: 分类名称应清晰明确，避免歧义
3. **层级控制**: 建议最多两级，避免层级过深影响使用
4. **权限管理**: 分类管理建议限制为管理员操作
5. **变更通知**: 分类变更后应通知相关人员，避免影响渠道录入
