## 一、功能概述

渠道信息录入用于在系统中注册新的合作渠道，包括线上平台、线下中介、合作伙伴等各类获客渠道的基本信息和结算配置。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限

## 三、操作步骤

### 3.1 进入渠道列表

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 进入渠道管理页面（默认显示数据统计视图）
4. 点击「切换」按钮或导航至【渠道列表】标签页

**实测页面元素**:
| 元素 | 类型 | 说明 |
|------|------|------|
| 战区筛选 | 下拉框 | 按战区筛选渠道数据 |
| 渠道来源筛选 | 下拉框 | 按渠道类型筛选 |
| 时间筛选 | 日期范围 | 选择统计时间范围 |
| 搜索按钮 | 按钮 | 执行筛选查询 |
| 重置按钮 | 按钮 | 清空筛选条件 |
| 数据表格 | 表格 | 显示各战区渠道分布百分比 |

### 3.2 新增渠道

1. 点击页面右上角「新增渠道」按钮
2. 弹出新增渠道表单对话框

### 3.3 填写基本信息

| 字段 | 类型 | 必填 | 说明 | 校验规则 |
|------|------|------|------|----------|
| 渠道名称 | 文本输入 | ✅ | 渠道的唯一标识名称 | 不能为空，不能重复 |
| 渠道类型 | 单选下拉 | ✅ | 选择渠道所属分类 | 必须选择一项 |
| 渠道负责人 | 文本输入 | ❌ | 渠道对接人姓名 | - |
| 联系电话 | 文本输入 | ❌ | 联系人手机号码 | 手机号格式校验 |
| 联系地址 | 文本输入 | ❌ | 渠道办公地址 | - |
| 合作状态 | 单选按钮 | ✅ | 当前合作状态 | 默认选中"合作中" |
| 合作开始日期 | 日期选择 | ❌ | 合作开始时间 | 不能晚于当前日期 |

### 3.4 填写结算信息

| 字段 | 类型 | 必填 | 说明 | 校验规则 |
|------|------|------|------|----------|
| 佣金类型 | 单选按钮 | ✅ | 固定金额或比例 | 默认选中"比例" |
| 佣金比例/金额 | 数字输入 | ✅ | 根据类型填写比例或金额 | 比例 0-100%，金额>0 |
| 结算周期 | 单选按钮 | ✅ | 佣金结算频率 | 必须选择一项 |
| 开户银行 | 文本输入 | ❌ | 收款银行名称 | - |
| 银行账号 | 文本输入 | ❌ | 收款银行账号 | 账号格式校验 |
| 开户姓名 | 文本输入 | ❌ | 收款人姓名 | - |

### 3.5 保存提交

1. 检查所有必填字段已填写
2. 点击「保存」按钮
3. 系统校验数据合法性
4. 校验通过则保存成功，提示"渠道添加成功"
5. 自动关闭对话框，返回列表页
6. 新渠道显示在列表第一行

## 六、常见问题

### Q1: 渠道名称是否可以修改？
**A**: 可以。在渠道列表中点击「编辑」按钮，修改后保存即可。但建议谨慎修改，避免影响历史数据关联。

### Q2: 已合作的渠道如何暂停？
**A**: 编辑渠道信息，将合作状态改为"暂停"。暂停状态的渠道不能新增带看记录。

### Q3: 渠道信息删除后历史数据如何处理？
**A**: 渠道信息不支持删除，只能修改状态为"终止"。历史带看和成交记录仍保留。

### Q4: 佣金比例可以后续调整吗？
**A**: 可以。编辑渠道信息可调整佣金比例，调整后新成交按新比例计算，历史成交不受影响。

## 七、注意事项

1. **渠道名称唯一性**: 确保渠道名称不重复，建议使用"平台名 + 区域"格式
2. **信息准确性**: 结算信息涉及佣金支付，务必准确填写
3. **状态管理**: 及时更新渠道合作状态，避免与已终止渠道合作
4. **权限控制**: 渠道信息修改建议设置权限，仅管理员可操作
5. **数据关联**: 渠道信息关联带看、成交、结算等数据，修改需谨慎
