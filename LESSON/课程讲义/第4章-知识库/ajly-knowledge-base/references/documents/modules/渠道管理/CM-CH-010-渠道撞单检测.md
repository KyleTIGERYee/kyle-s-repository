## 一、功能概述

渠道撞单检测用于自动检测多个渠道跟进同一客户的情况，避免佣金纠纷。系统在客户报备、带看、成交等环节自动触发撞单检测。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限
- 系统中存在客户报备数据（CM-CH-009）

## 三、操作步骤

### 3.1 进入撞单检测页面

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 点击【撞单检测】标签页
4. 查看撞单检测界面

### 3.2 选择查询方式

1. 选择查询方式：
   - **客户电话**: 通过手机号查询（推荐）
   - **客户姓名**: 通过姓名查询
   - **房源编号**: 通过房源查询

### 3.3 输入查询内容

1. 根据选择的查询方式输入内容
2. 客户电话需输入完整 11 位手机号
3. 客户姓名支持模糊查询
4. 房源编号输入完整编号

### 3.4 执行查询

1. 点击「查询」按钮
2. 系统搜索相关报备记录
3. 显示查询结果

### 3.5 查看撞单检测结果

系统显示以下信息：

| 字段 | 说明 |
|------|------|
| 查询客户 | 显示客户姓名和脱敏电话 |
| 报备记录列表 | 所有相关报备记录 |
| 渠道 | 报备渠道名称 |
| 报备时间 | 报备提交时间 |
| 状态 | 有效/撞单/已失效 |
| 有效期 | 报备截止日期 |
| 优先级 | 第一报备/第二报备... |

### 3.6 查看判定结果

系统根据撞单规则自动判定归属：
- **第一报备**: 优先归属该渠道
- **已带看**: 如第一报备未带看，已带看渠道优先
- **已成交**: 已成交渠道优先

## 四、撞单规则

### 4.1 撞单判定条件

```
撞单条件（同时满足）:
1. 客户电话相同（或姓名 + 房源相同）
2. 报备在有效期内
3. 报备状态为"有效"
```

### 4.2 优先级规则

```
优先级判定（依次判断）:
1. 已成交 > 已带看 > 有效报备
2. 同状态下，先报备优先
3. 同一渠道多次报备，取最早有效报备
```

### 4.3 有效期计算

```
有效期 = 报备日期 + 有效期限

默认有效期限：7 天
可调整范围：1-30 天
超时状态：自动变更为"已失效"
```

## 六、撞单处理建议

### 6.1 协商处理

| 情况 | 建议 |
|------|------|
| 第一报备未带看 | 建议转让给已带看渠道 |
| 多渠道同时跟进 | 建议协商分成 |
| 客户主动指定 | 尊重客户选择 |

### 6.2 系统处理

| 操作 | 说明 |
|------|------|
| 转让报备 | 第一报备渠道可转让给其他渠道 |
| 撤销报备 | 渠道可主动撤销无效报备 |
| 延长有效期 | 如客户跟进中，可延长有效期 |

## 七、常见问题

### Q1: 撞单检测的触发时机？
**A**: 系统在客户报备提交时自动触发撞单检测，也可手动查询。

### Q2: 电话脱敏显示规则？
**A**: 中间 4 位隐藏显示，如 138****0000，保护客户隐私。

### Q3: 如何避免撞单？
**A**: 带看前先进行客户报备，系统会自动检测并提示。

### Q4: 撞单后如何判定归属？
**A**: 按优先级规则：已成交>已带看>有效报备，同状态下先报备优先。

### Q5: 报备有效期过后如何处理？
**A**: 报备自动失效，其他渠道可重新报备。原渠道如仍需跟进，需重新报备。

---

## 八、注意事项

1. **及时报备**: 带看前务必先报备，避免撞单纠纷
2. **信息准确**: 客户电话务必准确，影响撞单检测
3. **定期查询**: 定期查询跟进中客户的报备状态
4. **协商优先**: 发生撞单时优先协商解决
5. **隐私保护**: 客户信息注意保密，避免泄露
