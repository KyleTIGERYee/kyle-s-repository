## 一、功能概述

成交确认用于记录渠道带看后的成交结果，关联带看记录并计算佣金金额，是渠道结算的前置环节。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限
- 系统中已存在带看记录（CM-CH-003）

## 三、操作步骤

### 3.1 进入成交管理页面

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 点击【成交管理】标签页
4. 查看成交记录列表

### 3.2 新增成交

1. 点击页面右上角「新增成交」按钮
2. 弹出新增成交表单对话框

### 3.3 选择带看记录

1. 点击「选择带看」下拉框
2. 系统显示所有未关联成交的带看记录
3. 选择目标带看记录
4. 系统自动带出客户姓名、房源信息

### 3.4 填写成交信息

| 字段 | 类型 | 必填 | 说明 | 校验规则 |
|------|------|------|------|----------|
| 选择带看 | 下拉选择 | ✅ | 选择已存在的带看记录 | 必须选择 |
| 成交房源 | 文本输入 | ✅ | 成交的房源信息 | 自动带出，可修改 |
| 成交客户 | 文本输入 | ✅ | 成交客户姓名 | 自动带出，可修改 |
| 合同编号 | 文本输入 | ❌ | 租赁合同编号 | - |
| 成交日期 | 日期选择 | ✅ | 成交签约日期 | 不能晚于当前日期 |

### 3.5 填写佣金信息

| 字段 | 类型 | 必填 | 说明 | 校验规则 |
|------|------|------|------|----------|
| 合同金额 | 数字输入 | ✅ | 租赁合同总金额 | 正数 |
| 佣金类型 | 单选按钮 | ✅ | 比例或固定金额 | 默认选中"比例" |
| 佣金比例/金额 | 数字输入 | ✅ | 佣金比例或固定金额 | 比例 0-100%，金额>0 |
| 佣金金额 | 数字输入 | ✅ | 最终佣金金额 | 自动计算，可调整 |
| 结算状态 | 单选按钮 | ✅ | 佣金结算状态 | 默认"待结算" |

### 3.6 上传附件（可选）

1. 点击「选择文件」按钮
2. 选择合同扫描件或相关文件
3. 支持 JPG、PNG、PDF 格式
4. 文件大小不超过 10MB

### 3.7 保存提交

1. 检查所有必填字段已填写
2. 点击「保存」按钮
3. 系统校验数据合法性
4. 校验通过则保存成功，提示"成交记录添加成功"
5. 自动关闭对话框，返回列表页
6. 新成交记录显示在列表第一行

## 六、佣金计算规则

### 6.1 自动计算逻辑

```
if 佣金类型 == "比例":
    佣金金额 = 合同金额 × 佣金比例
else:
    佣金金额 = 固定金额
```

### 6.2 渠道预设佣金

| 渠道类型 | 默认佣金比例 |
|----------|-------------|
| 公司 - 自有渠道 | 0%（内部渠道） |
| 个人 - 垂类平台 | 50%（首月租金） |
| 中介 | 50-100%（首月租金） |
| 新媒体渠道 | 30-50%（首月租金） |

## 七、常见问题

### Q1: 成交记录可以修改吗？
**A**: 可以。在成交列表中找到对应记录，点击「编辑」按钮即可修改。但已结算的成交记录不能修改佣金信息。

### Q2: 成交记录可以删除吗？
**A**: 可以。但已结算的成交记录不能删除，需先撤销结算。

### Q3: 一个带看可以关联多个成交吗？
**A**: 不可以。一个带看记录只能关联一个成交记录，避免重复计算佣金。

### Q4: 佣金金额可以手动调整吗？
**A**: 可以。系统自动计算后，允许手动调整佣金金额，适应特殊情况。

### Q5: 成交后多久需要结算佣金？
**A**: 根据渠道约定的结算周期。系统支持配置自动提醒，到期前通知财务处理。

## 八、注意事项

1. **带看关联**: 成交必须关联带看记录，确保渠道溯源
2. **佣金准确**: 佣金金额涉及财务支付，务必准确计算
3. **附件完整**: 建议上传合同扫描件，便于后续核对
4. **状态及时**: 成交后及时更新结算状态，避免遗漏
5. **数据一致**: 成交房源、客户信息应与带看记录一致
