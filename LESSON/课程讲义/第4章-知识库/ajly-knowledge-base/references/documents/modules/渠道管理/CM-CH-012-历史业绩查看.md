## 一、功能概述

历史业绩查看用于查询渠道的历史成交业绩数据，支持按时间范围、渠道类型、战区等多维度筛选，帮助管理者评估渠道长期表现和趋势分析。

**前置条件**:
- 已登录安居乐寓管理系统
- 拥有渠道管理模块操作权限
- 系统中存在渠道成交数据

## 三、操作步骤

### 3.1 进入历史业绩查看页面

1. 点击左侧菜单【租赁管理】展开
2. 点击【渠道管理】
3. 点击【历史业绩查看】标签页
4. 查看渠道历史业绩数据

### 3.2 设置筛选条件

| 筛选条件 | 类型 | 说明 |
|----------|------|------|
| 时间范围 | 日期范围 | 选择开始日期和结束日期 |
| 渠道类型 | 下拉选择 | 筛选特定渠道类型分类 |
| 战区 | 下拉选择 | 筛选特定战区 |
| 渠道来源 | 下拉选择 | 筛选特定渠道来源 |
| 业务员 | 下拉选择 | 筛选特定业务员 |

### 3.3 执行查询

1. 设置筛选条件
2. 点击「查询」按钮
3. 列表显示符合条件的历史业绩数据

### 3.4 查看业绩统计卡片

统计卡片显示以下汇总数据：

| 指标 | 说明 |
|------|------|
| 累计带看 | 筛选时间范围内的总带看次数 |
| 累计成交 | 筛选时间范围内的总成交次数 |
| 转化率 | 成交数/带看数 × 100% |
| 累计佣金 | 筛选时间范围内的总佣金金额 |

### 3.5 查看趋势图表

趋势图表展示：
- 月度成交趋势（折线图）
- 月度带看趋势（柱状图）
- 转化率趋势（折线图）

### 3.6 查看业绩明细列表

列表显示以下字段：

| 字段 | 说明 |
|------|------|
| 序号 | 列表序号 |
| 渠道 | 渠道名称 |
| 带看 | 该渠道带看次数 |
| 成交 | 该渠道成交次数 |
| 转化率 | 成交/带看 × 100% |
| 佣金 | 该渠道佣金金额 |
| 月份 | 统计月份 |

### 3.7 查看渠道详情

1. 找到目标渠道
2. 点击渠道名称或「详情」按钮
3. 弹出渠道业绩详情弹窗
4. 查看完整业绩数据和月度趋势

### 3.8 导出数据

1. 点击「导出」按钮
2. 选择导出格式（Excel）
3. 系统生成历史业绩报表文件
4. 下载导出文件

## 四、筛选规则

### 4.1 时间范围

```
- 支持自定义选择开始日期和结束日期
- 默认显示最近 3 个月数据
- 最大查询范围：12 个月
```

### 4.2 多条件组合

```
多个筛选条件为"与"关系:
- 时间范围 = X 至 Y AND
- 渠道类型 = A AND
- 战区 = B AND
- 渠道来源 = C AND
- 业务员 = D
```

### 4.3 排序规则

```
默认排序:
1. 月份（倒序，最近月份优先）
2. 成交数（倒序）
3. 渠道名称（字母顺序）
```

## 五、统计指标说明

### 5.1 核心指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 带看量 | 统计周期内带看记录数 | 渠道带客户看房次数 |
| 成交量 | 统计周期内成交记录数 | 渠道促成签约次数 |
| 转化率 | 成交量/带看量 × 100% | 渠道转化效率 |
| 佣金 | 成交合同佣金总和 | 渠道应得佣金金额 |

### 5.2 趋势分析

```
月度趋势分析:
- 对比各月带看量变化
- 分析成交量波动
- 观察转化率趋势
- 评估渠道稳定性
```

### 5.3 渠道评级参考

| 转化率 | 评级参考 | 说明 |
|--------|----------|------|
| ≥70% | A 级 | 优质渠道 |
| 50%-70% | B 级 | 良好渠道 |
| 30%-50% | C 级 | 一般渠道 |
| <30% | D 级 | 需优化渠道 |

## 七、常见问题

### Q1: 历史业绩数据更新频率？
**A**: 业绩数据实时更新，带看/成交记录完成后立即统计。

### Q2: 时间范围最大能查多久？
**A**: 最大支持查询 12 个月的历史数据，超过范围需分段查询。

### Q3: 转化率如何计算？
**A**: 转化率 = 成交量 / 带看量 × 100%，保留 1 位小数。

### Q4: 导出数据包含哪些内容？
**A**: 导出 Excel 包含统计卡片、趋势数据、明细列表全部内容。

### Q5: 为什么某些渠道没有业绩数据？
**A**: 可能原因：该渠道在筛选时间范围内无带看/成交记录，或渠道已停用。

## 八、注意事项

1. **数据准确性**: 业绩数据基于带看/成交记录，确保记录完整准确
2. **时间范围**: 查询大范围数据时可能需要加载时间，建议分段查询
3. **权限控制**: 不同角色可能看到不同范围的业绩数据
4. **导出限制**: 大批量导出时可能需要等待，建议分批导出
5. **趋势分析**: 趋势图表基于月度汇总，日度数据请在明细中查看
