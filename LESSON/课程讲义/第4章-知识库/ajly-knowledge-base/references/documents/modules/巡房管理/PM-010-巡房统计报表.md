## 一、功能概述

- **功能目的**: 提供巡房工作的数据统计和分析，帮助管理者掌握巡房任务执行情况、巡房人工作量、问题整改率等关键指标
- **使用场景**: 管理者需要查看巡房工作整体情况、评估巡房人绩效、分析巡房问题分布时
- **业务价值**: 通过数据驱动优化巡房工作计划，提升巡房效率和问题整改率

## 三、统计报表功能说明

### 3.1 状态统计卡片

| 统计项 | 说明 | 子项 |
|--------|------|------|
| 全部 | 所有巡房任务总数 | - |
| 待巡房 | 状态为"待巡房"的任务数 | 超时/时效内 |
| 已巡房 | 状态为"已巡房"的任务数 | 超时/时效内 |
| 已取消 | 状态为"已取消"的任务数 | 手动/自动 |

**点击交互**:
- 点击统计卡片可筛选对应状态的任务列表
- 点击子项（超时/时效内/手动/自动）可进一步筛选

### 3.2 巡房统计维度

| 统计维度 | 统计内容 | 展示形式 |
|----------|----------|----------|
| 巡房任务统计 | 总任务数/已完成/待完成/超期数 | 状态卡片 |
| 巡房人工作量 | 各巡房人负责任务数/完成数 | 列表/图表 |
| 问题整改率 | 发现问题数/已整改数/整改率 | 百分比 |
| 巡房类型分布 | 空房巡检/租客入住巡检/租客退房巡检/定期巡检 | 比例图 |
| 店面巡房情况 | 各店面巡房任务数/完成率 | 列表 |

### 3.3 数据列表字段

| 列名 | 说明 | 数据来源 |
|------|------|----------|
| 巡房状态 | 待巡房/巡房中/已巡房/已取消 | 任务状态 |
| 房源编号 | 房间唯一标识 | 房源信息 |
| 店面 | 所属门店 | 房源信息 |
| 物业地址 | 详细房间地址 | 房源信息 |
| 巡房计划 | 计划巡房日期和时间范围 | 巡房计划 |
| 巡房人 | 负责巡房的人员 | 任务分配 |
| 实际巡房时间 | 实际执行巡房的时间 | 巡房记录 |
| 巡房结果 | 正常/异常/其他 | 巡房记录 |
| 维修状态 | 正常/有待处理工单/已派单/处理中/已完成/已验收 | 工单关联 |
| 保洁状态 | 正常/有待处理工单/已派单/处理中/已完成/已验收 | 工单关联 |
| 操作 | 开始巡房/记录结果/发现问题登记/整改派单/整改进度跟踪/整改验收等 | 功能入口 |

## 四、操作流程

### 4.1 查看巡房统计

1. 进入巡房管理页面
2. 查看顶部状态统计卡片，了解整体巡房任务情况
3. 点击统计卡片筛选对应状态的任务
4. 查看数据列表详细信息

### 4.2 导出巡房统计

1. 设置筛选条件（店面/业务类型/计划巡房日期等）
2. 点击"导出"按钮
3. 系统创建后台导出任务
4. 前往数据导出中心下载 Excel 文件

## 五、筛选功能

### 5.1 基础筛选

| 字段 | 类型 | 说明 |
|------|------|------|
| 店面 | 下拉单选 | 选择特定门店 |
| 业务类型 | 下拉单选 | 公寓/商业 |
| 计划巡房 | 日期范围 | 开始日期至结束日期 |
| 搜索 | 文本输入 | 房源编号/巡房事项模糊匹配 |

### 5.2 高级筛选（展开后）

| 字段 | 类型 | 说明 |
|------|------|------|
| 巡房人 | 下拉单选 | 选择特定巡房人 |
| 巡房结果 | 下拉单选 | 正常/异常/其他 |
| 租赁状态 | 下拉单选 | 空置/已租/已订等 |
| 选择房间 | 级联选择 | 物业地址/楼层/门牌号 |
| 巡房类型 | 下拉单选 | 空房巡检/租客入住巡检/租客退房巡检/定期巡检 |
| 实际巡房日期 | 日期范围 | 实际巡房时间范围 |

## 六、异常场景

| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 无巡房任务数据 | 未创建巡房计划 | 列表显示"暂无数据" | 先创建巡房计划（PM-001） |
| 无已巡房记录 | 当前账号无已完成的巡房任务 | 点击"已巡房"筛选显示 0 条 | 分配巡房任务并完成巡房 |
| 导出任务失败 | 数据量过大或系统异常 | 提示导出失败 | 缩小筛选范围后重试 |

## 七、关联影响

- **影响模块**: 工单管理（维修/保洁工单关联）、公寓管理/商业管理（房源信息）
- **影响数据**: 巡房任务状态、巡房记录、维修状态、保洁状态
- **后续操作**: 开始巡房（PM-003）、记录巡房结果（PM-004）、发现问题登记（PM-005）

## 九、注意事项

1. **数据权限**: 普通业务员仅可查看自己负责的任务，店长/经理可查看全部
2. **统计时效**: 统计数据实时计算，可能存在秒级延迟
3. **导出限制**: 大量数据导出采用后台异步任务，需前往导出中心下载
4. **超期标识**: 超期未巡房任务在列表中以红色"【已超期 N 天】"标识
