> **功能名称**: 偏好设置

> **操作频率**: 低频  
> **最后更新**: 2026-03-21  
> **执行人**: 紫灵

## 一、功能概述
- **功能目的**: 配置商机管理模块的个人使用偏好和默认设置
- **使用场景**: 需要自定义列表显示、筛选条件、通知设置等
- **业务价值**: 提升个人使用体验，提高工作效率

## 二、入口方式
### 2.1 导航路径
菜单路径：租赁管理 → 商机管理 → 求租标签页

### 2.2 按钮位置（ASCII 结构图）

```
+------------------------------------------------------------------+
|  【筛选查询区】                                                    |
|  +------------------------------------------------------------+  |
|  |  店面：[请选择▼]  ...                                      |  |
|  +------------------------------------------------------------+  |
+------------------------------------------------------------------+
```

### 2.3 前置条件
- [ ] 已登录系统并具有商机管理模块访问权限
- [ ] 具有偏好设置权限（如有权限控制）

## 四、操作流程
### 4.1 正常流程
1. 进入租赁管理 → 商机管理 → 求租标签页
2. 点击"偏好设置"按钮
3. 打开偏好设置弹窗
4. 修改需要调整的设置项
5. 点击"保存"按钮保存设置
6. 设置立即生效
7. 点击"关闭"按钮取消修改（可选）

### 4.2 异常流程
| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 无权限修改 | 用户权限不足 | 设置项禁用或提示无权限 | 联系管理员开通权限 |
| 保存失败 | 网络异常 | 提示保存失败 | 检查网络后重试 |

## 五、状态变更
```
默认设置 --[修改并保存]--> 自定义设置

设置保存后：
- 列表显示设置：立即应用到当前列表
- 筛选条件设置：下次进入页面时生效
- 通知设置：立即生效
```

## 六、关联影响
- **影响模块**: 商机管理列表页
- **影响数据**: 仅影响个人视图，不影响他人
- **后续操作**: 可随时修改偏好设置

## 八、常见问题
Q1: 偏好设置是个人设置还是全局设置？  
A1: 偏好设置是个人设置，只影响当前用户的视图，不影响其他用户。

Q2: 修改偏好设置后多久生效？  
A2: 大部分设置立即生效，部分设置（如默认筛选条件）在下次进入页面时生效。

Q3: 如何恢复默认设置？  
A3: 重新打开偏好设置，将各项修改为默认值后保存。
