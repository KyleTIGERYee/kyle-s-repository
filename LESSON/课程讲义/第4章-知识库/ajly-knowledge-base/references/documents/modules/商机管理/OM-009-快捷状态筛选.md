# OM-009 快捷状态筛选操作手册

> **功能编号**: OM-009  
> **功能名称**: 快捷状态筛选  
> **所属模块**: 商机管理  
> **操作频率**: 高频  
> **最后更新**: 2026-03-21  
> **执行人**: 紫灵

---

## 一、功能概述
- **功能目的**: 通过快捷按钮快速筛选特定状态的客源
- **使用场景**: 需要快速查看有效客源、我的成交客源或快到期客源
- **业务价值**: 提高常用筛选场景的操作效率，一键直达目标数据

## 二、入口方式
### 2.1 导航路径
菜单路径：租赁管理 → 商机管理 → 求租标签页

### 2.2 按钮位置（ASCII 结构图）
```
+------------------------------------------------------------------+
|  求租  |  我的客源                                                |
+------------------------------------------------------------------+
|  [偏好设置] [列表自定义] [操作日志] [使用说明]                    |
+------------------------------------------------------------------+
|  【筛选查询区】                                                    |
|  +------------------------------------------------------------+  |
|  |  店面：[请选择▼]  分组：[请选择▼]  ...  [搜索] [重置] [展开▼]|  |
|  |  [有效] [我的成交] [快到期]  ← 快捷筛选按钮                  |  |
|  +------------------------------------------------------------+  |
+------------------------------------------------------------------+
```

### 2.3 前置条件
- [ ] 已登录系统并具有商机管理模块访问权限
- [ ] 系统中存在客源数据

## 三、页面结构
### 3.1 快捷筛选按钮区（ASCII 结构图）
```
+------------------------------------------------------------------+
|  +------------------------------------------------------------+  |
|  |  [有效]      [我的成交]      [快到期]                       |  |
|  |    ↑           ↑               ↑                            |  |
|  |  筛选状态    筛选我的        筛选快到期                      |  |
|  |  为有效的    成交的客源      的客源                          |  |
|  |  客源                                              |  |
|  +------------------------------------------------------------+  |
+------------------------------------------------------------------+
```

### 3.2 快捷按钮说明
| 按钮名称 | 功能说明 | 筛选条件 |
|----------|----------|----------|
| 有效 | 筛选客户状态为"有效"的客源 | 客户状态 = 有效 |
| 我的成交 | 筛选客源归属为"我的成交"的客源 | 客源归属 = 我的成交 |
| 快到期 | 筛选即将到期的客源（如有） | 限期字段接近当前日期 |

## 四、操作流程
### 4.1 正常流程
1. 进入租赁管理 → 商机管理 → 求租标签页
2. 查看当前列表数据（默认显示全部）
3. 点击快捷筛选按钮（有效/我的成交/快到期）
4. 列表实时刷新，显示筛选后的数据
5. 查看分页信息变化（如：从 41198 条变为 27738 条）
6. 点击其他快捷按钮可切换筛选条件（单选互斥）
7. 再次点击已选中的按钮可取消筛选（可选）

### 4.2 异常流程
| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 无匹配数据 | 该状态下无客源 | 列表显示"暂无数据" | 切换其他筛选条件 |
| 网络延迟 | 网络较慢 | 列表刷新延迟 | 等待加载完成 |

## 五、状态变更
```
全部客源 --[点击"有效"]--> 有效客源
全部客源 --[点击"我的成交"]--> 我的成交客源
全部客源 --[点击"快到期"]--> 快到期客源

筛选状态为单选互斥，每次只能应用一个快捷筛选条件
```

## 六、关联影响
- **影响模块**: 无
- **影响数据**: 仅影响列表展示，不修改数据
- **后续操作**: 可对筛选结果进行查看、编辑、导出、批量操作等

## 八、常见问题
Q1: 快捷筛选和基础筛选可以同时使用吗？  
A1: 可以。快捷筛选和基础筛选条件是 AND 逻辑，会同时生效。

Q2: 如何取消快捷筛选？  
A2: 再次点击已选中的快捷按钮，或使用"重置"按钮清除所有筛选。

Q3: 快捷筛选的"有效"状态和客户状态下拉框的"有效"有什么区别？  
A3: 功能相同，快捷按钮提供更便捷的操作方式。
