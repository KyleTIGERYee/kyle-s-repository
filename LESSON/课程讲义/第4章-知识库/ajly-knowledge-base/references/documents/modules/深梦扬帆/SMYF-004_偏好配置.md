## 一、功能概述

- **功能目的**: 配置深梦扬帆活动的入住日期相关偏好设置，控制用户可选择的入住日期范围和通知时机
- **使用场景**: 
  - 运营人员需要设置毕业生可选择的入住日期范围
  - 配置系统自动通知租客确认入住的时间点
  - 设置排房操作的时间窗口
- **业务价值**: 通过灵活的日期配置，优化深梦扬帆活动的用户体验和运营效率

---

### 2.1 导航路径

菜单路径：营销推广 → 深梦扬帆 → 偏好配置

## 三、页面结构

### 3.1 页面布局

```
+----------------------------------------------------------+
| 偏好设置                                          [×]    |
+----------------------------------------------------------+
| 入住日期配置                                              |
|                                                           |
| *入住日期最早可选 [2] 天后                               |
| *入住日期最晚可选 [30] 天内                              |
| *入住日期前 [5] 天内，通知租客确认是否入住               |
| *入住日期前 [5] 天内，可以对用户进行排房                 |
| *选择第 [10] 个门店的最早有房日期，作为默认入住日期      |
|                                                           |
|                                      [保存]               |
+----------------------------------------------------------+
```

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 入住日期最早可选 | 数字输入 | 是 | 正整数 | 用户可选择的最早入住日期为当前日期后的第 N 天，默认 2 天 |
| 入住日期最晚可选 | 数字输入 | 是 | 正整数 | 用户可选择的最晚入住日期为当前日期后的第 N 天，默认 30 天 |
| 通知租客确认入住 | 数字输入 | 是 | 正整数 | 入住日期前 N 天内系统自动通知租客确认是否入住，默认 5 天 |
| 可排房时间窗口 | 数字输入 | 是 | 正整数 | 入住日期前 N 天内可以对用户进行排房操作，默认 5 天 |
| 默认入住日期 | 数字输入 | 是 | 正整数 | 选择第 N 个门店的最早有房日期作为默认入住日期，默认 10 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 保存 | 对话框底部右侧 | 保存偏好配置并关闭对话框 | 深梦扬帆配置权限 |
| 关闭 (×) | 对话框右上角 | 关闭对话框不保存 | 无 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录管理后台，进入"营销推广 → 深梦扬帆"
步骤 2: 点击顶部"偏好配置"按钮，弹出偏好设置对话框
步骤 3: 查看当前配置值（默认：最早 2 天/最晚 30 天/通知 5 天/排房 5 天/默认 10）
步骤 4: 根据运营需求修改各字段值
步骤 5: 点击"保存"按钮
步骤 6: 系统显示"保存成功!"提示，对话框关闭
步骤 7: 返回深梦扬帆列表页，配置生效
```

### 4.2 流程图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐

└─────────────┘    └─────────────┘    └─────────────┘
                                              ↓
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 配置生效返回 │ ← │ 显示保存成功 │ ← │ 点击保存按钮 │
└─────────────┘    └─────────────┘    └─────────────┘
```

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|

| 排房时间合理性 | 排房天数 ≤ 最晚可选天数 | 系统提示（如有） |
