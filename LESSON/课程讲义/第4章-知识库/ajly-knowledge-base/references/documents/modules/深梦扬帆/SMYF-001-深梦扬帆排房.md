## 一、功能概述

- **功能目的**: 管理深梦扬帆活动报名租客的排房操作，包括查看报名信息、排房、确认入住等
- **使用场景**: 
  - 运营人员查看深梦扬帆活动报名记录
  - 为通过认证的毕业生租客安排房源
  - 确认租客入住状态
  - 管理排房状态（已排房/可排房/不可排房/已入住）
- **业务价值**: 帮助运营人员高效管理毕业生专项租房优惠活动的房源分配，确保符合条件的毕业生顺利入住

## 四、操作流程

### 4.1 查询排房记录

```
步骤 1: 进入深梦扬帆排房页面
步骤 2: （可选）设置筛选条件：
        - 选择报名门店
        - 选择预计入住日期
        - 输入租客姓名或手机号
        - 选择毕业日期范围
步骤 3: （可选）选择认证结果/入住确认状态/排房状态筛选
步骤 4: 点击"搜索"按钮
步骤 5: 查看筛选后的数据列表
```

### 4.2 查看租客详情

```
步骤 1: 在数据列表中找到目标记录
步骤 2: 点击"查看"按钮
步骤 3: 查看租客详细信息和附件
```

### 4.3 重新排房

```
步骤 1: 在数据列表中找到排房状态为"已排房"的记录
步骤 2: 点击"重新排房"按钮
步骤 3: 选择新的房源（门店/楼栋/房间号）
步骤 4: 确认排房信息
步骤 5: 系统更新排房状态
```

### 4.4 取消排房

```
步骤 1: 在数据列表中找到排房状态为"已排房"的记录
步骤 2: 点击"取消排房"按钮
步骤 3: 确认取消操作
步骤 4: 系统更新排房状态为"可排房"
```

### 4.5 确认入住

```
步骤 1: 在数据列表中找到排房状态为"已排房"的记录
步骤 2: 点击"确认入住"按钮
步骤 3: 确认入住信息
步骤 4: 系统更新入住确认状态为"已确认"
步骤 5: 系统更新排房状态为"已入住"
```

### 4.6 取消入住

```
步骤 1: 在数据列表中找到排房状态为"已入住"的记录
步骤 2: 点击"取消入住"按钮
步骤 3: 确认取消操作
步骤 4: 系统更新入住确认状态和排房状态
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 无 | 无 | 筛选条件均为可选 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 手机号 | 11 位数字 | 请输入正确的手机号 | 13812345678 |
| 日期 | YYYY-MM-DD | 请选择有效日期 | 2026-03-20 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 毕业日期范围 | 开始日期不能晚于结束日期 | 开始日期不能晚于结束日期 |
| 排房操作 | 只有"可排房"状态的记录才能排房 | 当前记录不可排房 |
| 重新排房 | 只有"已排房"状态的记录才能重新排房 | 当前记录未排房 |
| 取消排房 | 只有"已排房"状态的记录才能取消排房 | 当前记录未排房 |
| 确认入住 | 只有"已排房"状态的记录才能确认入住 | 请先排房 |
| 取消入住 | 只有"已入住"状态的记录才能取消入住 | 当前记录未确认入住 |
