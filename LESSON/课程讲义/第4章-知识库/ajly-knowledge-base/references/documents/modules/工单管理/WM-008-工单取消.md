> **所属模块**: 工单管理  
> **操作频率**: 低频  
> **最后更新**: 2026-03-21  
> **执行人**: 紫灵

## 一、功能概述
- **功能目的**: 取消/删除不需要或错误创建的工单记录
- **使用场景**: 工单登记错误、重复创建、租客取消报修、问题解决无需维修等场景
- **业务价值**: 保持工单数据准确性，避免无效工单干扰统计和管理

## 二、入口方式
### 2.1 导航路径
菜单路径：工单管理 → 维修保洁 → 工单列表 → 操作 → 删除

|  |          |     |   +------------+                    |
|  |          |     |   | 修改       |                    |

|  |          |     |   | 批量删除   |                    |

### 2.3 前置条件
- [ ] 已登录系统并具有工单删除权限（店长/经理/管理员）
- [ ] 工单处于可删除状态（通常仅限未派单或特定状态的工单）
- [ ] 已进入工单管理→维修保洁页面

## 四、操作流程
### 4.1 正常流程
1. 进入工单管理→维修保洁页面
2. 在工单列表中找到需要取消的工单
3. 点击该工单"其他操作"列的"操作"按钮
4. 在下拉菜单中选择"删除"选项
5. 系统弹出删除确认弹窗
6. 确认删除信息无误
7. 点击"确定"按钮
8. 系统提示"操作成功"
9. 工单从列表中移除

### 4.2 异常流程
| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 无删除权限 | 普通员工账号 | 操作菜单中无"删除"选项 | 联系店长/经理操作 |
| 工单不可删除 | 工单已处理/已验收 | 可能提示"该工单不可删除" | 使用其他流程处理 |
| 误点删除 | 误操作 | 在确认弹窗阶段 | 点击"取消"按钮 |
| 删除失败 | 系统异常 | 提示"删除失败" | 检查网络后重试 |

### 4.3 权限说明
| 角色 | 删除权限 | 说明 |
|------|----------|------|
| 管理员 | ✅ 可删除 | 可删除所有工单 |
| 店长/经理 | ✅ 可删除 | 可删除本门店工单 |
| 普通员工 | ❌ 不可删除 | 无删除权限，操作菜单中无删除选项 |
| 维修人员 | ❌ 不可删除 | 无删除权限 |

## 五、状态变更
```
工单存在 --[删除操作]--> 工单已删除（记录从系统中移除）
```

**删除影响**:
- 工单记录永久删除，无法恢复
- 关联的维修进展记录同步删除
- 不影响已产生的费用记录（如有）
- 不影响关联房源状态

## 六、关联影响
- **影响模块**: 无直接关联模块
- **影响数据**: 工单记录永久删除
- **后续操作**: 如需重新创建工单，需使用"新建工单"功能

## 八、常见问题
**Q1: 删除工单后还能恢复吗？**  
A1: 不能。删除操作是永久性的，工单记录无法恢复。请在删除前仔细确认。

**Q2: 为什么我看不到"删除"选项？**  
A2: 删除功能需要特定权限（店长/经理/管理员）。如确需删除，请联系有权限的同事操作。

**Q3: 删除工单会影响房源状态吗？**  
A3: 不会。删除工单仅删除工单记录本身，不影响关联房源的租赁状态。

**Q4: 可以批量删除多个工单吗？**  
A4: 可以。在工单列表勾选多个工单前的复选框，然后点击"批量删除"按钮。

**Q5: 什么情况下应该取消工单而不是标记完成？**  
A5: 当工单是错误创建、重复创建、或租客取消报修时，应删除工单。当维修已完成时，应使用"处理完成"功能。
