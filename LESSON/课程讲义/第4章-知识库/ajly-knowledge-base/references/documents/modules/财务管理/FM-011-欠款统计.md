## 一、功能概述

- **功能目的**: 统计和查看系统中所有欠款记录，包括他人欠公司、公司欠他人、分公司之间借款三类欠款数据
- **使用场景**: 财务对账、欠款催收、欠款统计分析、欠款回收跟踪
- **业务价值**: 帮助财务人员全面掌握公司欠款情况，支持多条件筛选和导出，便于欠款管理和催收工作

## 四、操作流程

### 4.1 正常流程

#### 4.1.1 查看欠款列表

1. 进入财务管理 → 欠款统计页面
2. 默认显示"他人欠公司"标签页，欠费状态=未还
3. 系统自动加载欠款数据列表
4. 查看欠款详情（房源编号、欠款金额、还款状态等）

#### 4.1.2 筛选欠款记录

1. 选择欠费状态（未还/已还完/坏账）
2. 选择统计月份（可选）
3. 选择欠款身份（全部/租客/房东/股东/供应商/装修公司/物业/其他）
4. 选择期限（请选择/逾期/未逾期）
5. 点击"搜索"按钮执行筛选
6. 列表实时更新，显示筛选结果

#### 4.1.3 导出欠款数据

1. 设置筛选条件（可选）
2. 点击"导出欠款"按钮
3. 系统创建导出任务
4. 前往数据导出中心查看进度并下载文件

### 4.2 异常流程

| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 无欠款数据 | 筛选条件下无匹配记录 | 列表显示"暂无数据" | 调整筛选条件或确认数据是否存在 |
| 导出失败 | 网络异常或数据量过大 | 提示导出失败 | 重试或联系技术支持 |
| 权限不足 | 账号无财务管理权限 | 无法访问欠款统计页面 | 联系管理员开通权限 |

## 五、状态说明

### 5.1 欠费状态

| 状态 | 说明 | 数据特征 |
|------|------|----------|
| 未还 | 欠款尚未归还 | 剩余欠款 > 0 |
| 已还完 | 欠款已全部归还 | 剩余欠款 = 0，还款状态显示"已还完【日期 + 姓名】" |
| 坏账 | 确认为坏账的欠款 | 特殊标记，需财务审核确认 |

| 身份 | 说明 | 典型场景 |
|------|------|----------|
| 租客 | 房屋承租人 | 房租、水电费欠款 |
| 房东 | 房源所有者 | 租金结算欠款 |
| 股东 | 公司股东 | 股东借款 |
| 供应商 | 服务/商品供应商 | 服务费、采购款欠款 |
| 装修公司 | 装修服务商 | 装修款欠款 |
| 物业 | 物业管理方 | 物业费欠款 |
| 其他 | 其他类型欠款人 | 特殊场景 |

## 八、常见问题

**Q1: 欠款总金额后面的"挂"标识是什么意思？**  
A1: "挂"表示该欠款已挂账处理，通常用于特殊情况的欠款标记，需要财务后续跟进处理。

**Q2: 已还完的欠款为什么还显示在列表中？**  
A2: 欠款统计功能记录所有历史欠款，包括已还完的。可通过欠费状态筛选查看未还款项。已还完记录显示还款日期和还款人，便于追溯。

**Q3: 如何区分逾期和未逾期欠款？**  
A3: 使用"期限"筛选条件，选择"逾期"或"未逾期"进行筛选。逾期表示已超过约定还款期限。

**Q4: 导出的欠款数据包含哪些字段？**  
A4: 导出 Excel 包含列表中所有字段：房源编号、店面、区域物业地址、姓名、身份、电话、欠款时间、欠款总金额、剩余欠款、还款期限、欠款说明、录入人、还款状态。

**Q5: 他人欠公司和公司欠他人有什么区别？**  
A5: "他人欠公司"是应收款项（租客、房东等欠公司），"公司欠他人"是应付款项（公司欠供应商、房东等）。两者业务方向相反。
