## 一、功能概述

- **功能目的**: 管理公寓运营中除房租水电外的其他应收应支计划（如宽带费、物业费、暖气费、装修款等），实现定时费用的计划性管理
- **使用场景**: 财务人员需要按时支付或收取固定费用，避免遗漏产生损失
- **业务价值**: 
  - 系统自动生成统计报表，便于财务人员随时掌握每月应收应支情况
  - 领导可查看每月收支计划，做好资金规划
  - 支持领导审核机制，未通过的计划不出现在统计报表中
  - 可直接从统计报表生成流水，避免多付、漏付风险

## 四、操作流程

### 4.1 查看收支计划列表

1. 登录系统，进入「财务管理」菜单
2. 点击「收支计划」子菜单
3. 系统显示收支计划列表页面
4. 可使用筛选条件（月份/店面/计划类型）查询特定计划
5. 点击「搜索」按钮执行筛选
6. 点击「更多」按钮展开更多筛选条件
7. 点击「导出计划」按钮可导出当前筛选结果

### 4.2 添加收支计划

1. 在收支计划列表页面，点击「添加计划」按钮
2. 系统打开「增加其他计划」弹窗
3. 选择计划类型：应收计划 或 应支计划
4. 填写表单字段：
   - 选择店面（必填）
   - 选择应收种类（必填）
   - 选择款项原因（必填）
   - 输入计划总金额（必填）
   - 输入收款期数（必填）
   - 选择缴费方式（必填）
   - 选择收款间隔（必填）
   - 选择首次收款时间（必填）
   - 选择款项关联人（必填）
   - 填写具体关联人或单位（可选）
   - 填写备注（可选，300 字以内）
5. 点击「生成计划」按钮提交
6. 系统验证数据并创建计划
7. 列表自动刷新显示新创建的计划

### 4.3 查看功能说明

1. 在收支计划列表页面，点击「功能说明」按钮
2. 系统显示功能说明弹窗
3. 阅读说明了解收支计划的作用和好处
4. 点击「关闭」按钮关闭说明弹窗

### 4.4 查看统计报表

1. 在收支计划列表页面，点击「收支计划统计报表」按钮
2. 系统跳转到统计报表页面
3. 查看收支计划的汇总统计数据

## 五、异常流程

| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 必填字段未填 | 点击生成计划时有必填字段为空 | 提示"请填写必填字段"或类似提示 | 补全所有必填字段后重新提交 |
| 金额格式错误 | 计划总金额输入非数字 | 输入框校验失败或提交时报错 | 输入正确的数字格式 |
| 期数为 0 或负数 | 收款期数输入 0 或负数 | 校验失败 | 输入大于 0 的正整数 |
| 日期格式错误 | 首次收款时间格式不正确 | 日期选择器不允许或提交报错 | 使用日期选择器选择正确日期 |
| 备注超长 | 备注超过 300 字 | 无法继续输入或提交时报错 | 精简备注内容到 300 字以内 |
| 列表无数据 | 当前筛选条件下无计划记录 | 显示"列表中无数据存在！" | 调整筛选条件或添加新计划 |

## 六、状态变更（如涉及）

```
计划创建 → 待审核 → 审核通过 → 生效执行
              ↓
          审核驳回（需修改后重新提交）
```

## 七、关联影响

- **影响模块**: 
  - 统计报表：收支计划数据会反映在统计报表中
  - 流水账：可从统计报表直接生成流水
- **影响数据**: 
  - 创建计划后，系统会根据计划自动生成待办提醒
  - 审核通过的计划会纳入资金规划统计
- **后续操作**: 
  - 计划创建后需等待审核
  - 审核通过后可在统计报表中查看和生成流水

## 九、常见问题

**Q1: 收支计划和流水账有什么区别？**  
A1: 收支计划是预先制定的应收应支计划，用于资金规划和提醒；流水账是实际发生的收支记录。可以从收支计划统计报表直接生成流水账，确保不漏付、不多付。

**Q2: 创建的计划为什么没有立即生效？**  
A2: 收支计划需要经过审核流程，审核通过后才会生效并出现在统计报表中。

**Q3: 如何修改已创建的计划？**  
A3: 在计划列表中点击对应计划的「其他操作」列，选择编辑功能进行修改（如支持）。

**Q4: 收款间隔有哪些选项？**  
A4: 收款间隔选项包括：1 个月、2 个月、3 个月、4 个月、6 个月、1 年、2 年。

**Q5: 款项关联人有哪些类型？**  
A5: 款项关联人类型包括：股东、供应商、装修公司、物业、其他。
