# MK-CP-005 生成发放链接操作手册

> **功能编号**: MK-CP-005  
> **功能名称**: 生成发放链接  
> **所属模块**: 优惠券管理  
> **操作频率**: 低频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 生成优惠券领取链接，用户通过链接自主领取优惠券
- **使用场景**: 活动推广、社交媒体分享、短信/邮件营销、二维码投放
- **业务价值**: 降低发券成本，支持病毒式传播，提升活动覆盖面

---

## 二、入口方式

### 2.1 导航路径

菜单路径：租赁管理 → 优惠券 → 券列表 → 选择券 → 详情 → 生成发放链接

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 优惠券详情                                               |
+----------------------------------------------------------+
| 基本信息区域                                             |
| ...                                                      |
+----------------------------------------------------------+
| 发放方式                                                  |
| [生成发放链接] [复制链接] [生成二维码]                   |
+----------------------------------------------------------+
| 链接信息：                                                |
| https://uat-mp.szajly.com/coupon/领取？code=XXXXX        |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 已创建有效的优惠券模板（MK-CP-001）
- [ ] 优惠券状态为"已发布"
- [ ] 领券方式为"链接领取"
- [ ] 拥有优惠券管理权限


### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 选择优惠券 | 下拉选择 | 是 | 仅限已发布券 | 选择要生成链接的券 |
| 领取上限 | 数字输入 | 否 | 正整数 | 限制总领取数量 |
| 有效期 | 日期选择 | 否 | 日期≥当前日期 | 链接有效期 |
| 适用人群 | 下拉选择 | 否 | 全部/新用户/老用户 | 限制领取人群 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 生成链接 | 页面下方 | 生成优惠券领取链接 | 优惠券管理权限 |
| 复制链接 | 结果区 | 复制领取链接到剪贴板 | 无 |
| 生成二维码 | 结果区 | 生成链接对应的二维码 | 无 |
| 下载二维码 | 结果区 | 下载二维码图片 | 无 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 导航至租赁管理 → 优惠券
步骤 2: 选择目标优惠券，点击"详情"
步骤 3: 点击"生成发放链接"按钮
步骤 4: 选择优惠券（如未预先选择）
步骤 5: （可选）配置领取上限/有效期/适用人群
步骤 6: 点击"生成链接"
步骤 7: 系统生成唯一领取链接
步骤 8: 复制链接或下载二维码用于推广
```

### 4.2 流程图

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 选择优惠券 | "请选择优惠券" | 提交时 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 领取上限 | 正整数 | "领取上限必须为正整数" | 1000 |
| 有效期 | 日期≥当前日期 | "有效期不能早于今天" | 2026-12-31 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 优惠券状态 | 优惠券必须为已发布状态 | "优惠券未发布" |
| 领券方式 | 券的领券方式需支持链接领取 | "该券不支持链接领取" |
| 库存校验 | 领取上限 ≤ 优惠券库存 | "领取上限不能超过库存" |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 优惠券已下架 | "优惠券已下架，无法生成链接" | 重新发布优惠券 |
| EX-002 | 库存不足 | "库存不足，无法设置此上限" | 降低领取上限或补充库存 |
| EX-003 | 链接已存在 | "该券已生成过链接" | 使用已有链接或重新生成 |
| EX-004 | 网络超时 | "网络繁忙，请稍后再试" | 检查网络连接后重试 |
| EX-005 | 无权限 | "您没有生成链接权限" | 联系管理员授权 |
