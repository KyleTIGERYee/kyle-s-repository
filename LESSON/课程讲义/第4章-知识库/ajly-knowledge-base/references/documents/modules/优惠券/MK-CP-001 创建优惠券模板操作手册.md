# MK-CP-001 创建优惠券模板操作手册

> **功能编号**: MK-CP-001  
> **功能名称**: 创建优惠券模板  
> **所属模块**: 优惠券管理  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 创建新的优惠券模板，配置优惠券的基本信息、使用规则、适用范围等
- **使用场景**: 运营人员创建新的优惠券用于营销活动、客户激励等场景
- **业务价值**: 提供灵活的优惠券配置能力，支持多种优惠类型和使用规则

---

## 二、入口方式

### 2.1 导航路径

菜单路径：**租赁管理** → **优惠券** → **新建券按钮**

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 优惠券列表页面                                            |
+----------------------------------------------------------+
| 筛选区：                                                  |
| 适用门店 | 用途 | 适用业态 | 类型 | 券名称 | 状态 | ...   |
| [搜索] [重置] [新建券] ← 点击此处进入创建页面             |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 已登录系统且具有租赁管理模块访问权限
- [ ] 具有优惠券创建权限
- [ ] 系统中存在可用的门店数据
- [ ] 系统中存在可用的业态配置

### 3.1 页面布局

```
+----------------------------------------------------------+
| 基本信息                                                  |
+----------------------------------------------------------+
| [重置] [提交] [返回] ← 顶部操作按钮                       |
+----------------------------------------------------------+
| *用途 [下拉选择] | *券名称 [文本输入] | 创建时间 [只读]    |
| *领券方式 [下拉] | *次数 [整数输入] | *发行量 [整数输入]   |
| *生效规则 [下拉] | ☑ 支付时单次用券上限为 1 次              |
+----------------------------------------------------------+
| 用券规则                                                  |
+----------------------------------------------------------+
| *适用业态 [下拉] | *适用消费项 [下拉] | *券类型 [下拉]     |
| *优惠额度 [数字] | 适用签约时长 [数字] 个月及以上          |
| 适用房源 [选择器] | 是否发短信 [开关] | 是否对外展示 [开关]  |
+----------------------------------------------------------+
| 适用门店                                                  |
+----------------------------------------------------------+
| [下载模版] [导入房源] [删除]                              |
| 表格：序号 | 适用门店 | 适用物业地址 | 适用门牌号 | 操作  |
| (支持批量导入房源，暂无数据时显示"暂无数据")               |
+----------------------------------------------------------+
| 互斥设置                                                  |
+----------------------------------------------------------+

+----------------------------------------------------------+
| 详细说明                                                  |
+----------------------------------------------------------+
| 卡券说明 [多行文本]                                       |
| (提示：卡券说明为对客展示项，请谨慎填写)                   |

```

### 3.2 字段说明

#### 基本信息区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 用途 | 下拉选择 | 是 | 必选 | 优惠券用途分类（如活动专用、通用等） |
| 券名称 | 文本输入 | 是 | 必填，长度限制 | 优惠券名称，用于标识和搜索 |
| 创建时间 | 日期 | 否 | 系统自动生成，只读 | 优惠券创建时间 |
| 领券方式 | 下拉选择 | 是 | 必选 | 用户领取优惠券的方式 |
| 次数 | 整数输入 | 是 | 正整数 | 每人可领取次数限制 |
| 发行量 | 整数输入 | 是 | 正整数 | 优惠券总发行数量 |
| 生效规则 | 下拉选择 | 是 | 必选 | 优惠券生效时间规则（如领券后立即生效/指定日期生效） |
| 支付时单次用券上限为 1 次 | 复选框 | 否 | - | 勾选后限制单次支付只能使用 1 张券 |

#### 用券规则区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 适用业态 | 下拉选择 | 是 | 必选 | 适用业态（公寓/商业） |
| 适用消费项 | 下拉选择 | 是 | 必选 | 适用的消费项目（租金/押金/服务费等） |
| 券类型 | 下拉选择 | 是 | 必选 | 券类型（代金券/折扣券） |
| 优惠额度 | 数字输入 | 是 | 正数 | 代金券输入金额，折扣券输入百分比 |
| 适用签约时长 | 数字输入 | 否 | 正整数 | 限制适用签约时长，单位：月 |
| 适用房源 | 房源选择器 | 否 | - | 选择适用的具体房源 |
| 是否发短信 | 开关控件 | 否 | - | 开启后用户领取券时发送短信通知 |
| 是否对外展示 | 开关控件 | 否 | - | 开启后在小程序端对外展示该优惠券 |

#### 适用门店区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 适用门店 | 表格列表 | 否 | - | 优惠券适用的门店列表 |
| 适用物业地址 | 文本 | 否 | - | 门店对应的物业地址 |
| 适用门牌号 | 文本 | 否 | - | 适用的具体门牌号 |

#### 互斥设置区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 互斥类型 | 下拉选择 | 否 | - | 互斥规则类型 |
| 互斥券 | 下拉选择 | 否 | - | 与该券互斥的其他优惠券 |

#### 详细说明区域

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 卡券说明 | 多行文本 | 是 | 必填 | 对客展示的优惠券说明文字 |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 重置 | 页面顶部 | 清空所有已填写的表单内容 | 无 |
| 提交 | 页面顶部 | 保存并提交优惠券配置 | 优惠券创建权限 |
| 返回 | 页面顶部 | 放弃修改返回优惠券列表 | 无 |
| 下载模版 | 适用门店区域 | 下载房源导入模板文件 | 无 |
| 导入房源 | 适用门店区域 | 导入房源 Excel 文件批量设置适用门店 | 无 |
| 删除 | 适用门店区域 | 删除已选择的适用门店记录 | 无 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，导航至租赁管理→优惠券
步骤 2: 点击"新建券"按钮，进入创建优惠券页面
步骤 3: 填写基本信息（用途/券名称/领券方式/次数/发行量/生效规则）
步骤 4: 配置用券规则（适用业态/适用消费项/券类型/优惠额度等）
步骤 5: （可选）配置适用门店（下载模版/导入房源或手动选择）
步骤 6: （可选）配置互斥设置
步骤 7: 填写卡券说明（对客展示内容）
步骤 8: 点击"提交"按钮保存配置
步骤 9: 系统验证通过后显示"保存成功"提示
步骤 10: 返回优惠券列表页面
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 用途 | 请选择用途 | 提交时未选择 |
| 券名称 | 请输入券名称 | 提交时为空 |
| 领券方式 | 请选择领券方式 | 提交时未选择 |
| 次数 | 请输入次数 | 提交时为空 |
| 发行量 | 请输入发行量 | 提交时为空 |
| 生效规则 | 请选择生效规则 | 提交时未选择 |
| 适用业态 | 请选择适用业态 | 提交时未选择 |
| 适用消费项 | 请选择适用消费项 | 提交时未选择 |
| 券类型 | 请选择券类型 | 提交时未选择 |
| 优惠额度 | 请输入优惠额度 | 提交时为空 |
| 卡券说明 | 请输入卡券说明 | 提交时为空 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 次数 | 正整数 | 请输入有效整数 | 1, 5, 10 |
| 发行量 | 正整数 | 请输入有效整数 | 100, 1000 |
| 优惠额度 | 正数 | 请输入有效数字 | 100 (代金券), 20 (折扣券 20%) |
| 适用签约时长 | 正整数 | 请输入有效整数 | 3, 6, 12 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 优惠券名称重复 | 同一用途下券名称不能重复 | 券名称已存在 |
| 发行量限制 | 发行量不能超过系统上限 | 发行量超出限制 |
| 优惠券有效性 | 配置的优惠券必须处于有效状态 | 优惠券已下架/过期 |
| 时间合理性 | 生效日期不能晚于失效日期 | 时间配置不合理 |
| 配置完整性 | 必填字段必须全部填写 | 请完善必填信息 |
| 图片尺寸 | 背景图建议尺寸 375x812 | 图片尺寸不符合建议 |
| 互斥券选择 | 不能选择自身作为互斥券 | 互斥券配置错误 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 必填字段未填写 | 请完善必填信息 | 检查并填写所有带*号的字段 |
| EX-002 | 输入非数字字符 | 请输入有效数字 | 检查数字格式 |
| EX-003 | 输入负数或 0 | 请输入正整数 | 输入大于 0 的整数 |
| EX-004 | 网络超时 | 网络繁忙，请稍后再试 | 检查网络连接后重试 |
| EX-005 | 无操作权限 | 无权限执行此操作 | 联系管理员开通权限 |
| EX-006 | 优惠券已下架 | 该优惠券已下架 | 选择其他有效优惠券 |
| EX-007 | 图片上传失败 | 图片上传失败 | 检查图片格式和大小 |
| EX-008 | 活动已发布 | 活动已发布无法修改 | 取消发布后编辑或重新创建 |
| EX-009 | 券名称重复 | 券名称已存在 | 使用其他名称 |
| EX-010 | 发行量超出限制 | 发行量超出系统限制 | 减少发行量 |
