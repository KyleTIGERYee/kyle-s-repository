# MK-CP-009 作废优惠券操作手册

> **功能编号**: MK-CP-009  
> **功能名称**: 作废优惠券  
> **所属模块**: 优惠券管理  
> **操作频率**: 低频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 将已发布但需要停用的优惠券作废，停止用户领取和使用
- **使用场景**: 活动提前结束、券配置错误、库存异常、风险管控
- **业务价值**: 快速响应异常情况，控制业务风险

---

## 二、入口方式

### 2.1 导航路径

菜单路径：租赁管理 → 优惠券 → 券列表 → 选择券 → 操作列 → 下架

### 2.2 按钮位置（ASCII 结构图）

```
+----------------------------------------------------------+
| 优惠券列表                                               |
+----------------------------------------------------------+
| ...                                                      |
+----------------------------------------------------------+
| 数据列表：                                                |
| 序号 | 券用途 | 券名称 | ... | 状态 | ... | 操作 |       |
| ...                                                      |
| 操作：[下架] [详情] ← 下架即作废操作                     |
+----------------------------------------------------------+
```

### 2.3 前置条件

- [ ] 优惠券状态为"已发布"
- [ ] 拥有优惠券管理权限
- [ ] 作废原因合理

## 三、页面结构

### 3.1 确认对话框

```
+----------------------------------------------------------+
| 确认下架                                                  |
+----------------------------------------------------------+
| 确定要下架该优惠券吗？                                    |
| 券名称：[券名称]                                          |
| 下架后用户将无法领取和使用                                |
+----------------------------------------------------------+
| [取消] [确定]                                             |
+----------------------------------------------------------+
```

### 3.2 字段说明

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 券名称 | 文本 | 只读 | - | 即将下架的券名称 |
| 下架原因 | 文本输入 | 否 | ≤200 字 | 下架原因说明（可选） |

---

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 导航至租赁管理 → 优惠券
步骤 2: 在券列表中找到目标优惠券
步骤 3: 点击操作列的"下架"按钮
步骤 4: 系统弹出确认对话框
步骤 5: 确认下架信息
步骤 6: 点击"确定"
步骤 7: 系统更新券状态为"已下架"
步骤 8: 用户无法再领取或使用该券
```

### 4.2 状态变更

| 原状态 | 操作 | 新状态 | 说明 |
|--------|------|--------|------|
| 已发布 | 下架 | 已下架 | 停止领取和使用 |
| 核销中 | 下架 | 已下架 | 停止新核销，已核销有效 |

---

## 五、校验规则

### 5.1 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 券状态 | 仅限已发布/核销中的券可下架 | "该券状态不可下架" |
| 权限校验 | 操作人有下架权限 | "无下架权限" |
| 关联活动 | 检查是否有关联活动 | "该券关联了活动，下架将影响活动" |

---

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 券已下架 | "该券已下架" | 无需重复操作 |
| EX-002 | 无权限 | "您没有下架权限" | 联系管理员授权 |
| EX-003 | 关联活动 | "该券关联了活动" | 确认是否继续下架 |
| EX-004 | 网络超时 | "网络繁忙，请稍后再试" | 检查网络后重试 |

## 七、关联影响

### 7.1 影响模块

- 优惠券查询（MK-CP-008）

- 小程序端领券页面（不再展示该券）
