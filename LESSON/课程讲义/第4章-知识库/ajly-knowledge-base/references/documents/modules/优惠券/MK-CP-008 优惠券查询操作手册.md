## 一、功能概述

- **功能目的**: 查询和管理系统中的优惠券模板，支持多条件筛选和分页浏览
- **使用场景**: 运营人员查看优惠券列表、筛选特定优惠券、了解优惠券使用状态
- **业务价值**: 提供优惠券全生命周期管理的入口，支持运营人员快速定位和管理优惠券

### 3.2 字段说明

#### 筛选区字段

| 字段名称 | 字段类型 | 必填 | 校验规则 | 说明 |
|---------|---------|------|---------|------|
| 适用门店 | 下拉选择 | 否 | 无 | 选择优惠券适用的门店 |
| 用途 | 下拉选择 | 否 | 无 | 优惠券用途分类（如活动专用、通用等） |
| 适用业态 | 下拉选择 | 否 | 无 | 适用业态（公寓/商业等） |
| 类型 | 下拉选择 | 否 | 无 | 券类型（代金券/折扣券等） |
| 券名称 | 文本输入 | 否 | 无 | 支持模糊搜索券名称 |
| 状态 | 下拉选择 | 否 | 无 | 优惠券状态（已发布/核销中/已下架等） |
| 生效日期 | 日期范围 | 否 | 开始日期≤结束日期 | 筛选生效日期范围内的优惠券 |
| 失效日期 | 日期范围 | 否 | 开始日期≤结束日期 | 筛选失效日期范围内的优惠券 |
| 创建日期 | 日期范围 | 否 | 开始日期≤结束日期 | 筛选创建日期范围内的优惠券 |
| 发行量 | 数字范围 | 否 | 正整数 | 筛选发行量范围内的优惠券 |
| 核销量 | 数字范围 | 否 | 正整数 | 筛选核销量范围内的优惠券 |
| 发行/核销金额 | 数字范围 | 否 | 正数 | 筛选金额范围内的优惠券 |

#### 列表字段

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 序号 | 数字 | 列表序号，从 1 开始 |
| 券用途 | 文本 | 优惠券用途分类 |
| 券名称 | 文本 | 优惠券名称 |
| 适用业态 | 文本 | 适用业态（公寓/商业） |
| 券类型 | 文本 | 券类型（代金券/折扣券） |
| 券优惠 | 文本/数字 | 优惠额度（代金券显示金额，折扣券显示百分比） |
| 状态 | 文本 | 优惠券状态（已发布/核销中/已下架等） |
| 创建时间 | 日期 | 优惠券创建日期，格式 YYYY-MM-DD |
| 生效日期 | 文本 | 生效规则描述（如"领券 0 天后生效"或具体日期） |
| 失效日期 | 文本 | 失效规则描述（如"领券 10 天后失效"或具体日期） |
| 已领 | 数字 | 已领取数量，未领显示"-" |
| 已用 | 数字 | 已核销数量，未用显示"-" |
| 未用 | 数字 | 未使用数量，未用显示"-" |

### 3.3 按钮说明

| 按钮名称 | 位置 | 功能说明 | 权限要求 |
|---------|------|---------|---------|
| 搜索 | 筛选区底部 | 根据筛选条件查询优惠券 | 优惠券查看权限 |
| 重置 | 筛选区底部 | 清空所有筛选条件 | 无 |
| 新建券 | 筛选区底部 | 创建新的优惠券模板 | 优惠券创建权限 |
| 下架 | 列表操作列 | 将已发布的优惠券下架 | 优惠券管理权限 |
| 详情 | 列表操作列 | 查看优惠券详细信息 | 优惠券查看权限 |

## 四、操作流程

### 4.1 正常流程

```
步骤 1: 登录系统，点击左侧菜单"租赁管理"→"优惠券"
步骤 2: 进入优惠券列表页面，默认显示"券"标签页
步骤 3: （可选）在筛选区设置筛选条件
步骤 4: 点击"搜索"按钮执行查询
步骤 5: 浏览列表数据，查看优惠券信息
步骤 6: （可选）点击分页控件翻页查看更多数据
步骤 7: （可选）点击"详情"查看优惠券详细信息
步骤 8: （可选）点击"新建券"创建新的优惠券
```

## 五、校验规则

### 5.1 必填校验

| 字段 | 错误提示 | 触发时机 |
|------|---------|---------|
| 无必填字段 | 无 | 所有筛选条件均为可选 |

### 5.2 格式校验

| 字段 | 格式要求 | 错误提示 | 示例 |
|------|---------|---------|------|
| 生效日期 | 开始日期≤结束日期 | 系统自动校验 | 2026-01-01 至 2026-12-31 |
| 失效日期 | 开始日期≤结束日期 | 系统自动校验 | 2026-01-01 至 2026-12-31 |
| 创建日期 | 开始日期≤结束日期 | 系统自动校验 | 2026-01-01 至 2026-12-31 |
| 发行量 | 正整数 | 仅接受数字输入 | 100 至 1000 |
| 核销量 | 正整数 | 仅接受数字输入 | 0 至 500 |
| 发行/核销金额 | 正数 | 仅接受数字输入 | 1000 至 10000 |

### 5.3 业务校验

| 校验项 | 校验规则 | 错误提示 |
|--------|---------|---------|
| 权限校验 | 需具有优惠券查看权限 | 无权限时页面不显示或按钮禁用 |
| 网络校验 | 需网络连接 | 网络超时显示加载失败 |
| 级联筛选 | 部分筛选条件可能存在级联关系 | 根据实际系统行为 |

## 六、异常场景

| 异常编号 | 异常场景 | 系统提示 | 处理方式 |
|---------|---------|---------|---------|
| EX-001 | 无筛选条件下数据量过大 | 无提示，正常加载 | 建议使用筛选条件缩小范围 |
| EX-002 | 筛选条件无匹配数据 | 列表显示"暂无数据" | 调整筛选条件后重新搜索 |
| EX-003 | 导出时网络超时 | 网络繁忙提示 | 检查网络连接后重试 |
| EX-004 | 无查看权限 | 页面不显示或按钮禁用 | 联系管理员开通权限 |
| EX-005 | 数据加载失败 | 加载失败提示 | 刷新页面或检查网络 |
| EX-006 | 导出文件过大 | 浏览器下载提示 | 使用筛选条件缩小导出范围 |
