# MK-CP-010 数据统计操作手册

> **功能编号**: MK-CP-010  
> **功能名称**: 数据统计  
> **所属模块**: 优惠券管理  
> **操作频率**: 中频  
> **最后更新**: 2026-03-23  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 统计优惠券发放、核销、剩余等数据，支持运营分析
- **使用场景**: 运营数据分析、活动效果评估、库存管理
- **业务价值**: 数据驱动决策，优化券策略，提升 ROI

---

## 二、入口方式

### 2.1 导航路径

菜单路径：租赁管理 → 优惠券 → 券列表 → 选择券 → 详情 → 统计

### 2.2 前置条件

- [ ] 拥有优惠券查看权限
- [ ] 存在优惠券数据

---

## 三、页面结构

### 3.1 统计概览

```
+----------------------------------------------------------+
| 优惠券统计                                               |
+----------------------------------------------------------+
| 基本信息：                                                |
| 券名称：[券名称] | 券类型：[代金券/折扣券]                |
+----------------------------------------------------------+
| 统计概览：                                                |
| ┌──────────┬──────────┬──────────┬──────────┐           |
| │ 发行量   │ 已领取   │ 已核销   │ 剩余量   │           |
| │  1000    │   800    │   500    │   200    │           |
| └──────────┴──────────┴──────────┴──────────┘           |
+----------------------------------------------------------+
| 核销率：50% | 领取率：80%                                 |
+----------------------------------------------------------+
```

### 3.2 字段说明

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 发行量 | 数字 | 优惠券总发行数量 |
| 已领取 | 数字 | 用户已领取数量 |
| 已核销 | 数字 | 已使用核销数量 |
| 剩余量 | 数字 | 未领取的剩余数量 |
| 核销率 | 百分比 | 已核销/发行量 |
| 领取率 | 百分比 | 已领取/发行量 |
