## 一、功能概述

- **功能目的**: 修改已添加到智能门锁的密码信息（密码值、密码名称、有效期等）
- **使用场景**: 租客更换需要修改密码、密码泄露需要重置、密码有效期需要延长
- **业务价值**: 保障门锁密码安全，支持密码动态管理

## 四、操作流程

### 4.1 正常流程

1. **进入智能硬件页面**
   - 导航：租赁管理 → 智能硬件
   - 默认显示"智能门锁"标签页

2. **定位目标设备**
   - 使用筛选条件（业务类型/物业地址/设备品牌/搜索）定位目标门锁
   - 确认设备状态为"在线"或"离线"（在线可实时同步，离线密码将在上线后同步）
   - 确认设备已绑定房源且已添加密码

3. **打开修改密码弹窗**
   - 点击设备列表操作列的"操作"按钮
   - 在下拉菜单中选择"修改密码"选项
   - 系统打开修改密码弹窗，自动填充设备信息

4. **选择要修改的密码**
   - 在"选择密码"下拉框中选择要修改的密码
   - 系统自动填充该密码的当前信息（密码类型、密码值掩码、密码名称、有效期）

5. **修改密码信息**
   - 修改密码值：输入新密码（6-8 位数字）或点击"生成随机密码"
   - 修改密码名称：编辑密码名称（可选）
   - 修改有效期：调整有效期设置（可选）
   - 填写备注：记录修改原因（可选）

6. **保存修改**
   - 点击"保存"按钮
   - 系统进行必填校验和格式校验
   - 校验通过后提交修改
   - 系统提示"修改成功"
   - 弹窗关闭，列表实时更新

### 4.2 异常流程

| 异常场景 | 触发条件 | 系统表现 | 处理方式 |
|----------|----------|----------|----------|
| 设备未绑定 | 设备未绑定房源 | 操作列不显示"修改密码"选项 | 先执行绑定房源操作 (ID-003) |
| 设备无密码 | 设备未添加任何密码 | 操作列不显示"修改密码"选项或选项禁用 | 先执行添加密码操作 (ID-005) |
| 非智能门锁 | 设备类型为电表/水表 | 操作列不显示密码相关选项 | 密码功能仅支持智能门锁 |
| 无操作权限 | 当前用户无设备管理权限 | 操作列不显示或选项禁用 | 联系店长/经理/管理员操作 |
| 密码格式错误 | 密码值非 6-8 位数字 | 保存时提示"请输入 6-8 位数字密码" | 输入符合格式的密码 |
| 密码全相同 | 密码值所有位数字相同（如 111111） | 保存时提示"密码不能全相同" | 输入不全相同的密码 |
| 密码重复 | 新密码与该门锁其他密码重复 | 保存时提示"该密码已被使用" | 使用不同的密码值 |
| 有效期冲突 | 结束日期早于或等于开始日期 | 保存时提示"结束日期必须晚于开始日期" | 调整日期使结束日期晚于开始日期 |
| 密码名称超长 | 密码名称超过 20 个字符 | 保存时提示"密码名称不能超过 20 个字符" | 缩短密码名称 |
| 备注超长 | 备注超过 300 个字符 | 保存时提示"备注不能超过 300 字" | 缩短备注内容 |
| 设备离线 | 设备联网状态为"离线" | 可修改成功，但提示"设备离线，密码将在上线后同步" | 等待设备上线后自动同步 |
| 密码正在使用 | 修改的密码当前有租客在使用 | 系统提示"该密码正在使用中，修改后原密码将失效" | 确认修改后通知租客新密码 |

## 五、状态变更

### 5.1 密码状态流转

```
密码已添加 --[修改密码]--> 密码已修改（生效中）
                ↓
        密码值更新
        有效期更新
        密码名称更新
                ↓
        修改记录保存
        （修改人/修改时间/修改内容）
```

### 5.2 修改记录

| 记录字段 | 说明 |
|----------|------|
| 修改人 | 执行修改操作的用户 |
| 修改时间 | 修改操作的执行时间 |
| 原密码值 | 修改前的密码值（加密存储） |
| 新密码值 | 修改后的密码值（加密存储） |
| 修改内容 | 具体修改的字段（密码值/密码名称/有效期） |

## 六、关联影响

- **影响模块**: 智能门锁设备管理、租客开门权限
- **影响数据**: 
  - 门锁密码表：密码值、密码名称、有效期、修改时间、修改人更新
  - 密码使用记录：原密码使用次数统计保留，新密码使用次数初始化为 0
- **后续操作**: 
  - 通知租客新密码（如为租客密码）
  - 验证新密码是否可用 (ID-008 查看开锁记录)
  - 如需删除密码执行 ID-007 删除密码

## 八、常见问题

**Q1: 修改密码后原密码是否立即失效？**  
A1: 是的，修改密码后原密码立即失效，新密码立即生效。如为租客密码，需及时通知租客新密码。

**Q2: 修改密码是否需要设备在线？**  
A2: 不需要。离线设备也可以修改密码，密码修改记录会保存，待设备上线后自动同步到门锁。

**Q3: 能否批量修改多个密码？**  
A3: 不支持。修改密码功能仅支持单个密码修改，如需批量操作需逐个修改。

**Q4: 修改密码后能否查看修改记录？**  
A4: 可以。在智能硬件配置→操作记录标签页可以查看密码修改记录（修改人/修改时间/修改内容）。

**Q5: 密码修改是否有次数限制？**  
A5: 无次数限制。可根据需要多次修改密码。

## 十、注意事项

⚠️ **重要提示**:

1. **权限限制**: 由于测试环境无智能门锁密码管理权限，本功能文档基于 ID-005 添加密码功能上下文和系统模式推断编写。待分配具有密码管理权限的测试账号后需实际验证完整操作流程。

2. **密码安全**: 
   - 密码值在系统中加密存储
   - 修改密码时原密码以掩码（●●●）显示
   - 修改记录中不显示明文密码

3. **同步机制**:
   - 在线设备：密码修改后实时同步到门锁
   - 离线设备：密码修改记录保存，待设备上线后自动同步

4. **影响范围**:
   - 修改租客密码会影响租客开门，需提前通知
   - 修改管家/管理员密码需确保相关人员知晓新密码

5. **与添加密码的区别**:
   - 添加密码：新增密码记录
   - 修改密码：更新已有密码记录（密码值/名称/有效期）
   - 修改密码需先选择要修改的密码，添加密码直接填写新密码信息
