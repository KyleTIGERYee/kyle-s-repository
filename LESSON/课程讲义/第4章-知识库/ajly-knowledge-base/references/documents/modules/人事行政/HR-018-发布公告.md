# HR-018 发布公告操作手册

> **功能编号**: HR-018  
> **功能名称**: 发布公告  
> **所属模块**: 人事行政 HR  
> **操作频率**: 低频  
> **最后更新**: 2026-03-22  
> **执行人**: 紫灵

---

## 一、功能概述

- **功能目的**: 用于发布公司内部公告、通知、制度等信息
- **使用场景**: 发布公司通知、制度变更、活动公告等
- **业务价值**: 统一信息发布渠道，确保信息传达及时准确

---

## 二、入口方式

菜单路径：人事行政 → 制度管理 → 发布公告
或：系统首页 → 最新公告 → 添加

---

## 三、页面结构

### 3.1 发布公告弹窗

```
+------------------------------------------+
| 发布公告                              [X] |
+------------------------------------------+
| 【基本信息】                              |
|  公告标题：* [文本输入 100 字以内]         |
|  公告类型：* [下拉选择]                   |
|    - 公司通知                             |
|    - 制度变更                             |
|    - 活动公告                             |
|    - 其他                                 |

|    - 全员                                 |
|    - 指定部门                             |
|    - 指定店面                             |
+------------------------------------------+
| 【公告内容】                              |
|  公告内容：* [富文本编辑器]               |

+------------------------------------------+

+------------------------------------------+

+------------------------------------------+
```

---

## 四、操作流程

1. 进入人事行政 → 点击制度管理
2. 点击发布公告按钮
3. 填写公告标题
4. 选择公告类型
5. 选择发布范围
6. 编辑公告内容
7. 上传附件（可选）
8. 设置发布时间（可选）
9. 设置有效期（可选）
10. 点击发布
11. 系统校验并提交
12. 发布成功

---

## 五、状态变更

```
草稿 → 已发布 → 已过期/已撤回
```

## 八、常见问题

**Q1: 发布后可以修改吗？**  
A1: 已发布的公告可以编辑修改，修改后重新发布。

**Q2: 可以撤回已发布的公告吗？**  
A2: 可以，撤回后公告不再显示。
