二、工具调用基础概念

什么是工具调用？

工具调用（Tool Calling）是指大语言模型根据用户的输入，自主判断是否需要调用外部工具（如API、函数、数据库等），并生成符合工具要求的结构化调用参数，最终由系统执行工具并将结果返回给模型。

核心特点

🧠

自主性

模型自主决策是否使用工具

📋

结构化

生成固定格式的调用参数（JSON）

🚀

扩展性

通过工具无限扩展模型能力

二、工具调用基础概念

Tool Calling vs Function Calling

在实际应用中，这两个术语经常被混用，但有一些细微差别：

Tool Calling

侧重点："工具"的概念

使用场景：

  • 调用外部API
  • 插件服务
  • 外部服务集成

Function Calling

侧重点："函数"的概念

使用场景：

  • 调用代码函数
  • 内置方法
  • 程序内部逻辑

💡 在Dify平台中，我们通常使用工具（Tool）或工作流（Workflow）来实现工具调用功能。

三、Function Calling工作原理

核心概念：填槽与调用

填槽（Slot Filling）

将用户的自然语言输入，提取并填充到预定义的参数结构中

类似于表单的自动填写

调用（Function Call）

根据填槽结果，生成结构化的函数调用请求

系统执行对应的工具/函数

  name: house_search
  description: 查询安居乐寓平台的房源信息，支持按城市、预算、户型等条件筛选
  parameters:
    type: object
    properties:
      city:
        type: string
        description: 城市名称，如深圳、北京
      minPrice:
        type: integer
        description: 最低预算（元/月）
      maxPrice:
        type: integer
        description: 最高预算（元/月）
      layoutName:
        type: string
        description: 户型名称，如一室一厅、两室一厅
    required: ["city"]

  • 快速接入、简单调用 → 使用自定义工具模式

案例2：知识库检索工具（防弹版代码）

知识库检索工具需要从向量数据库中检索相关文档，只能使用工作流模式实现。

import json

def main(input_data):
    """
    防弹版提取函数：无论输入什么，保证不报错
    """
    # 1. 预设默认值
    result_text = ""
    
    try:
        # 数据预处理
        if input_data is None:
            data = []
        else:
            data = input_data

        # 如果传入的是字符串，尝试转成对象
        if isinstance(data, str):
            try:
                data = json.loads(data)
            except:
                pass
        
        # 兼容性处理
        if isinstance(data, dict):
            if 'result' in data: 
                data = data['result']
            elif 'data' in data: 
                data = data['data']
            else:
                data = [data]
        
        # 确保 data 是列表
        if not isinstance(data, list):
            data = []

        # 核心提取逻辑
        contents = []
        for item in data:
            if isinstance(item, dict):
                c = item.get('content', '')
                if c:
                    contents.append(str(c).strip())
        
        result_text = "\n\n".join(contents)

    except Exception as e:
        result_text = f"数据处理异常: {str(e)}"

    return {"combined_text": result_text}
