## 二、Function Calling原理详解

### 6. Function Calling基本原理
**来源**：今日头条 - AI大模型的双手-Function Calling调用外部工具的基本原理
**链接**：http://m.toutiao.com/group/7513035653865587235/
**核心观点**：
- Function Calling：自主"解析"内容并结构化【填槽】，自主"决策"使用工具并结构化【调用】
- 自主解析内容：指只要看到Prompt里的内容，就能自动对内容按特定的格式进行解析
- 自主决策使用工具：指根据解析后的结构化内容，自动调用工具（比如天气、地理位置、日期等）
- 结构化：一种Json格式
- 填槽：指大模型根据与用户的历史对话内容，将信息结构化的解析出来
- 槽位：可以理解为"参数"，比如用户询问今天北京天气时，填槽的参数就是：日期、城市
**融入建议**：作为Function Calling原理的核心参考资料
