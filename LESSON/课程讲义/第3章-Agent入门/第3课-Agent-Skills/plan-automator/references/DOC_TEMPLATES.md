## 模板选择指南

| 任务类型 | 推荐模板 | 必需模块 |
|----------|---------|---------|
| 功能操作手册 | 模板一 | ASCII 页面结构图、字段说明、校验规则、操作流程、异常场景 |
| 快速测试 | 模板二 | 测试结果、Bug 列表、测试结论 |
| 日常巡检 | 模板三 | 检查清单、发现问题、巡检结论 |
| 数据质量检查 | 模板三（定制） | 数据准确性、完整性、一致性检查项 |
| 回归测试 | 模板二 + 模板三 | 测试结果、检查清单、Bug 列表 |

---

## 动态生成规则

**根据任务类型自动选择**：

1. **文档重写任务** → 模板一（完整操作手册）
2. **功能测试任务** → 模板一 + 模板二
3. **回归测试任务** → 模板二 + 模板三
4. **质量巡检任务** → 模板三

**根据页面类型自动调整**：

| 页面类型 | 建议模块 |
|----------|---------|
| 表单页面 | 字段说明、校验规则、操作流程 |
| 列表页面 | 字段说明、筛选规则、分页说明 |
| 详情页面 | 字段说明、操作按钮、关联数据 |
| 配置页面 | 配置项说明、生效规则、影响范围 |
| 报表页面 | 统计维度、数据口径、导出说明 |
