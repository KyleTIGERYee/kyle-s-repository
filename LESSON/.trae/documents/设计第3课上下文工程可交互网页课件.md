# 第3课：上下文工程 - 可交互网页课件设计计划

## 课程分析

### 课程结构

1. **课程目标** - 4个学习目标
2. **3.1 概念解析** - 上下文窗口、Token计算、注意力机制
3. **3.2 Coze中的记忆机制** - 短期记忆、长期记忆、变量系统
4. **3.3 RAG基础** - 知识库增强、向量数据库
5. **3.4 实战** - 创建能记住用户的Bot
6. **3.5 实战练习** - 3个练习
7. **3.6 本课小结**

### 核心概念

* 上下文窗口 vs 记忆

* Token计算

* 短期记忆（上下文轮数）

* 长期记忆（数据库）

* RAG检索增强生成

* 向量数据库

## 技术实现

### 文件结构

```
presentation/
├── index.html          # 主课件文件
├── css/
│   └── styles.css      # 从第2课复制Claymorphism样式
└── js/
    └── app.js          # 交互逻辑
```

### 核心交互组件

1. **Token计算器** - 实时计算输入文本的Token数量
2. **上下文轮数滑块** - 调节轮数，显示成本和体验变化
3. **滑动对比器** - 对比短期记忆和长期记忆
4. **Prompt构建器** - 构建记忆型Bot提示词
5. **步骤流程图** - RAG工作流程动画
6. **概念关系图谱** - 向量数据库原理展示
7. **对话模拟器** - 测试记忆功能

### 样式规范

* 使用Claymorphism风格（与第1、2课一致）

* 颜色：primary (#4F46E5), secondary (#22C55E)

* 字体：Baloo 2 + Comic Neue

* 卡片：白色背景 + 柔和阴影 + 28px圆角

### 动画效果

* 页面切换：淡入淡出

* 组件入场：GSAP stagger动画

* 交互反馈：悬停、点击动画

* 流程动画：SVG路径绘制

## 实施步骤

1. **创建目录结构**
2. **复制第2课CSS样式**
3. **创建HTML框架**（22个section）
4. **实现核心组件**

   * Token计算器

   * 上下文轮数滑块

   * 滑动对比器

   * Prompt构建器

   * 步骤流程图

   * 概念图谱

   * 对话模拟器
5. **填充讲义内容**
6. **添加动画效果**
7. **测试和优化**

* **视觉风格**：与第1、2课统一的Claymorphism风格
