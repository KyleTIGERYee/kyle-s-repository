# 第3课-复杂工作流 交互式课件制作计划

## 一、课件结构设计（共约13-15页）

根据讲义内容和页面划分规则，将课件分为以下幻灯片：

| 页码 | 内容 | 组件选择 |
|-----|------|---------|
| 1 | 封面页 | 课程标题 + 核心技能卡片 |
| 2 | 课程导入：场景示例 | 场景对比卡片（多意图/状态管理/空结果） |
| 3 | 课程导入：本节核心内容 | 框架流程动画（6大核心能力） |
| 4 | 多意图路由：四意图分类 | 交互式意图卡片（4个意图） |
| 5 | 多意图路由：参数提取 | 代码高亮扫描（思维链提示词） |
| 6 | 多意图路由：工作流分支 | SVG流程图动画（4分支路由） |
| 7 | 对话状态管理：状态表格 | 交互式对话轮次表格 |
| 8 | 对话状态管理：代码实现 | 代码高亮块 |
| 9 | HTTP请求：搜房API | 代码配置块 + API调用示意 |
| 10 | HTTP请求：管家API | 代码配置块 |
| 11 | 空结果处理：Recovery机制 | 流程步骤图（诊断→建议→引导） |
| 12 | 综合实战：工作流架构 | 工作流架构概览图 |
| 13 | 课程总结 | 核心知识点回顾（6项） |
| 14 | 课后作业：基础作业 | 作业要求卡片（还原安居乐寓AI找房助手） |
| 15 | 课后作业：进阶作业 | 3个选题卡片（智能报修/租客服务/深梦扬帆） |

## 二、交互设计

1. **导航系统**：键盘左右键 + 底部进度指示器
2. **入场动画**：每页使用GSAP实现元素依次淡入
3. **交互组件**：
   - 意图卡片：悬停高亮效果
   - 流程图：节点依次绘制动画
   - 代码块：打字机效果
   - 对话表格：逐步揭示

## 三、技术实现

- HTML5 + TailwindCSS（复用第1课样式）
- GSAP动画库
- 响应式设计（支持1920x1080投影）

## 四、制作步骤

1. 创建 `presentation/` 目录结构
2. 复制基础样式 `css/styles.css`
3. 编写 `index.html` 主文件（包含所有幻灯片）
4. 编写 `js/app.js` 交互逻辑
5. 本地测试预览
