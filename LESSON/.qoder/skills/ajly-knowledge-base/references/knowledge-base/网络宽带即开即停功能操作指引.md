# 网络宽带即开即停功能操作指引

网络宽带即开即停功能操作指引

3月6日更新：

”批量调整“中增加了字段”是否仅储存数据”，默认选择为“是”，当选择为“是”的时候再上传文件，模板Excel中的宽带账号、状态、运营商等数据仅做储存，不会和运营商的接口产生交互。

ps：如果需要删掉某些宽带账号，直接将该楼栋的运营商、宽带账号和启用状态三个字段的内容删掉再上传即可。

当选择为“否”的时候再上传文件，功能和原有功能一样，会调用运营商接口，对每个宽带进行启用或者停用（注：移动的宽带在停用状态下无法通过本功能直接停用，会报错。）

也就是说：存量的房间数据，直接选择“是”，就可以上传所有的宽带数据。后续系统会根据业务情况自动启用或停用宽带，也可以手动启用或停用宽带（具体看下面的原内容）

原版本：

## 请注意：仅支持PC端操作

## 管理入口：PC端公寓管理-网络管理

批量调整

可批量添加、修改、删除宽带账号

点击批量调整，需筛选门店、物业地址，点击搜索后可下载模板。（门店是最小颗粒度，可一次查询同门店多个楼栋的数据，不可跨门店或按公司维度查询）

下载后获得模板文件，文件中包含查询条件下的所有房间，需要填入对应的运营商（只可选择电信或移动），宽带账号（如：07551XXXXXX@163.gd，需运营商提供），宽带状态（以实际情况为准，可选“启用”或“停用”）

如果是已经录入过宽带信息的房间，将会自动带出已经录入的数据，可以直接修改运营商、宽带账号、宽带状态等，以此实现批量修改、批量启用/停用。

完成模板的编辑后，点击上传文件上传模板

文件上传后，系统后台将会逐一执行每一个账号的启用/停用，这里根据宽带账号是数量不同，所需消耗的时间也不同。系统将会将执行结果反馈在“结果”中，如果存在失败的情况，可以点击“下载失败记录”查看具体失败原因，调整后可再次上传。（请只上传失败的房间）

注意：如果某些房间不需要进行调整，请务必在模板文件中删除该行，因为只要文件中存在的房间，系统都会根据模板文件中的要求进行处理。

网络管理

完成批量调整后，可在【网络管理】窗口中对每一个房间的宽带进行单独处理，可以手动启用宽带或停用宽带

可以点击操作记录，查看每一间房的宽带操作记录，如操作人为“系统”则为系统即开即停功能根据业务场景自动进行的操作

即开即停功能说明

概述：系统会根据房间的状态，自动启用或停用宽带账号

租客签约入住，并且缴纳首期账单后，该房间宽带将自动启用；

如果租客需要换房，在换房完成后，新房间宽带将自动启用，原房间宽带将停用；

如果租客需要退房，在退房审批通过后，该房间宽带将自动停用。
