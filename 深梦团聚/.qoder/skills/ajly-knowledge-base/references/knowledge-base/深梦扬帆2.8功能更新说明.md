# 深梦扬帆2.8功能更新说明

深梦扬帆2.8功能更新说明

# 概述：

在小程序深梦扬帆专栏中，默认填写的入住日期增加了“灵活判断”的逻辑，以保证深梦扬帆专栏中不再出现大面积已满租的情况。

同时对已满租的门店，增加“预计x月x日有房”的描述，方便用户调整预计入住日期，以入住心仪的门店

# 计算逻辑调整

现在会计算全部门店的“最早有房日期”并记录

# 偏好设置调整

## 增加如下图的设置，会根据门店“最早有房日期“从早到晚排列，选择第N个门店的“最早有房日期”作为小程序端的默认预计入住日期

# 小程序页面调整

下面三个页面中增加气泡提示：“已匹配有房的时间段，可手动调整”

## 同时默认的预计入住日期不再是固定的当前日期+5天了，而是根据偏好配置的第N个门店的“最早有房日期”

已满租的门店，在列表页和详情页中，都增加提示“预计x月x日有房”
