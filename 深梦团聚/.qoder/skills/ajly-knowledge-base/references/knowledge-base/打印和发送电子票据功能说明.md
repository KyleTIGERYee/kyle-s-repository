# 租客端票据获取

租客端获取票据流程
①打开【安居乐寓】小程序，在【我的】页面，点击【支付记录】按钮，进入支付记录详情页

②点击支付记录卡片下的按钮【获取票据】，即可查看票据，当该笔流水未完成出纳审核，则会弹出系统提示“该笔流水未完成财务审核，暂不可获取票据，请联系管家处理”，出纳审核后，则可直接获取票据（银行渠道收付和线上付款的交易方式需要出纳审核）

③当流水为线上支付或已审核完成的线下支付，点击获取【票据按钮】，进入票据预览页，可分享票据至微信联系人，也可保存票据到本地
