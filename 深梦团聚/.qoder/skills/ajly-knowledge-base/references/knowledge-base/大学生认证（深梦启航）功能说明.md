# 大学生认证（深梦启航）功能说明

大学生认证（深梦启航）功能说明

请注意：毕业生身份认证成功后，该毕业生信息将会与该用户（即认证时登录的手机号）绑定！无法解绑！签约时需使用该手机号才可以享受深梦启航的优惠！

每一个学生身份只能认证一次！

# 客户进行学生身份认证

## 首页调整

将【深梦扬帆】入口菜单放置在下方（功能和原深梦扬帆一致），增加【深梦启航】入口菜单，点击后进入深梦启航的介绍页面，可点击按钮“加入深梦启航计划”跳转到学历认证页。

学历认证分两种类型：境内学历认证和境外学历认证。

## 境内学历认证

境内学历认证只需要在输入框中填入学信网学籍报告验证码，系统会使用该学籍报告验证码在学信网中查询该用户的毕业时间，并且将其与深梦启航活动所规定的毕业时间范围做比对。

如果在用户的毕业时间在活动所规定的毕业时间范围内，则能够通过大学生认证；

如果在用户的毕业时间不在活动所规定的毕业时间范围内，则不能够通过大学生认证；

如果用户的学籍报告验证码错误，则系统直接提示验证码有误。

如果用户一开始未通过境内学历认证，后续可以再进行境内学历或者境外学历的认证。

如果用户通过了境内学历认证，则无法再进行境内学历或境外学历认证。

## 境外学历认证

## 如果用户是境外学历，则需要填写如图所示的全部学历认证资料，并且上传相关证明材料的附件（图片形式，一张图片），上传相关资料后，系统将这些资料汇总到乐寓管理PC端，由工作人员在PC端进行人工审核。

境外学历的用户提交相关资料后需等待审核，再次进入该页面将会得到审核结果（审核通过、拒绝或待审核）

# PC端-境外学历审核

分配到管家侧的，需人工核验境外学历资料的，管家会在企微端收到上图的推送提醒。

需要在乐寓管理系统中进行境外学历的核验：

## 入口：PC端【营销推广】-【大学生认证】，列表页会自动筛选出当前用户可审核的境外学历信息。

用户可以查看租客提交的学籍信息以及学籍附件，点击查看可以查看附件大图，根据租客提交的信息，用户可以自行判断“审核通过”或“审核拒绝”

在点击“审核通过”，系统会校验租客提交的毕业时间是否符合深梦启航的时间段要求，如不符合，系统将会拦截此次“审核通过”的结果，用户只能点击““审核拒绝“。（租客被拒绝后， 可在小程序端再次发起学历认证的申请）

用户也可以查看和导出所有境内学历和境外学历的认证信息和结果。

以上所有租客提交的认证资料，均会通过尘锋系统的“线索对接”功能传到尘锋系统。

注意：由于尘锋线索对接功能的限制，如果租客已经在通过其他途径录入过尘锋系统的，线索对接上传联系人信息时将会报错。此外，毕业生认证状态字段不会根据乐寓管理PC端的审核结果进行更新，尘锋系统的SOP功能可以通过该字段提现对应的管家需要审核该用户的毕业生认证信息。

# PC端-配置说明

偏好设置中，可设置活动规则，活动规则的文本格式和内容会同步到小程序端。

偏好设置中，最多有两条关联的框架协议，每条框架协议都有对应的毕业时间段做选择。

偏好配置中的毕业时间段，影响小程序端租客进行大学生认证时，是否能够认证成功。

当存在两条关联的框架协议，也就是有两个毕业时间段时，小程序端大学生认证的判断是根据这两个毕业时间段的并集来决定的。

例如：存在两条框架协议

此时只要租客符合毕业时间在2024.1.1-2027.12.31，即可通过认证。

# PC端-登记租客

## 当租客通过大学生认证后，在PC端登记租客时，系统将会带出租客毕业时间对应的框架协议，例如：

租客在2024年12月31日毕业，则带出深梦启航2025；

如租客在2027年1月1日毕业，则带出深梦启航2026；

如果租客的毕业时间在两个毕业时间段的交集时，例如毕业于2025.1.1则会自动选择第一个框架协议。

## APP端不会自动带出框架协议，但是PC和APP在提交登记租客时，如果选择了大学生认证中所配置的框架协议，会判断租客是否通过了大学生认证，如果没有通过，则系统会自动拦截，无法提交。

## PC端操作续约、换房的时候也会自动带出大学生认证对应框架协议，如果以前有别的框架，会覆盖原框架协议。
