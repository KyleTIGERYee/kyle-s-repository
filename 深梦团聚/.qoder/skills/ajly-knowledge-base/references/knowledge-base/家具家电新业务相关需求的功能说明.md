# 家具家电新业务相关需求的功能说明

家具家电新业务相关需求的功能说明

# 供应商管理

## 供应商列表页

列表页中可以对使用供应商、统一信用代码、供应商状态、联系人名称/手机号等条件对列表页进行查询，同时可对查询内容进行导出

供应商状态有以下7种，具体逻辑如流程图

供应商列表页可以按照”供应商合同“和”楼栋“两个维度进行查询，当选择按楼栋维度查看时，列表中以楼栋作为最小颗粒度进行展示。

## 新增供应商合同

基本信息中，“签约公司主体”固定为控股、一开、二开、运营等四个公司主体，“合同时间“将会影响该合同是否过期

关联业务中有以下选项，

当选择基础家具家电业务时，登记租客等业务流程中将出现“基础家具家电租金”、“基础家具家电押金”等费项

当选择增值家具家电业务时，登记租客等业务流程中将出现“增值家具家电租金”等费项

点击下一步将关联具体楼栋信息，关联的楼栋将会出现上面所说的费项。这里不能夸战区选择楼栋，如果同种类型的业务已经在该楼栋存在了，那么其他同类型业务的供应商无法选择该楼栋。

选择好楼栋后，将会根据楼栋，以及具体的业务类型，生成业务定价模板，下载后填写模板上传，最后提交审核。

## 审批

审批配置可按照战区进行配置，审批人是读取了乐寓系统的组织架构，可自行随意调整。

审批分为三步，每一步由对应的审批人在列表中点击审批即可进入查看并且审批。

在审批详情中，可以查看定价附件，以及预期收入总额，以及操作记录

如果需要拒绝审批，则需要填写理由，该理由可以在操作记录中查看

## 其他操作

修改只可以修改基本信息

禁用供应商后，相关的费项将不在生成；启用供应商后，相关的费项将继续生成

# 业务主流程调整

## 新签

在【供应商管理】中关联了基础家具家电业务的楼栋，在新签时，系统会自动增加一笔基础家具家电押金和基础家具家电租金，该押金金额等于租金金额，这两条费项不可修改不可删除。其定价来自供应商管理中所上传的定价（需在供应商管理中审批通过）。

如果是关联了增值家具家电业务的楼栋，系统将会自动增加一笔增值家具家电租金，不可修改不可删除。

## 续约

已禁用或已到期的供应商，其对应的楼栋在续约时，将不再生成相关的费项

合作中的供应商，其对应的楼栋在续约时，仍会生成相关的费项（不会产生基础家具家电押金）

租客延期：会根据之前是否存在相关家具家电费项来决定是否生成，就算是禁用或者合同到期，仍会生成相关费项。

## 退房

提前退房，会生成剩余基础家具家电租金，如果是违约退房，会生成基础家具家电违约金

超期退房，会生成超期基础家具家电租金

如果是增值家具家电服务，在退房时，也会生成剩余增值家具家电租金或者超期增值家具家电租金。

可以手动添加基础/增值家具家电赔偿金

就算供应商到期或者停用，退房时仍生成相关费用

## 换房

换房时，如果目标楼栋没有关联供应商，或关联了但已经禁用，则新合同中不生成相关费项；原合同的结算单需对相关费用进行结算

换房时，如果目标楼栋关联了供应商且已经启用，则新合同中会生成相关费项；原合同的结算单需对相关费用进行结算

## 转租

转租时，如果当前楼栋关联了供应商且已经启用，则新合同中会生成相关费项；原合同的结算单需对相关费用进行结算

转租时，如果当前楼栋未关联供应商，或关联了但已经禁用，则新合同中会生成相关费项；原合同的结算单需对相关费用进行结算

# 财务相关调整

流水账调整，现在不再展示其他费，而是展示具体的费项名称。同时如果是家具家电相关业务的费用，将会明细和导出，以及汇总中，增加供应商字段

结算单中，现在不再展示其他费，而是展示具体的费项名称。

# 三方合同

## 电子合同账号注册

系统设置-合同设置中的电子合同账号注册，新增了栏目【供应商公司】，与公司合作的供应商公司，若需要签署三方合同的，在此处点击“添加注册信息”，填写供应商公司信息。

手机号为供应商公司经办人的手机号，后续三方合同的签约提醒会发到这个手机号上；印章编号是供应商公司在签约平台（契约锁或法大大）注册的信息。

添加好供应商的注册信息后，在下图红框中选择对应的合同模板，则该合同模板在签约时，丙方的信息将会调用此供应商的注册信息

这里只能选择三方合同模板，普通的合同模板无法选择。

## 合同模板调整

现在合同模板是根据楼栋配置的，如果是三方合同的话，只需要对有家具家电业务的楼栋单独设置合同模板即可

## 在设置三方合同模板时，选择“新增非标准合同”，右上角选择“三方合同”，在合同模板中，其他功能未做调整，只对字段#SFMC#增加了逻辑：在签约读取对应供应商的名称。

签约时，供应商盖章位置在“丙方：”上

## 电子签约

电子签约路径与现在一致，只是配置了三方合同模板的楼栋可以选择三方合同模板了。

签约时，供应商作为丙方进行签约

签约顺序是先租客签完，再给供应商签，供应商的经办人会接到短信通知，按照短信指引即可完成签约。

供应商签约完成后，此份电子合同才会变为已生效合同

请注意：如果供应商迟迟不签约，租客将看不到电子合同。

初始化：已经有三合一家具家电业务的楼栋，需要先录入供应商，开发同事可以进行数据调整，将原来的三合一家具家电的费项名称改为增值家具家电相关的费项名称，并且关联供应商信息。
