# 深梦扬帆公益行动第二季活动规则

（一）活动主题

主标：深圳喊你来追梦，安居点亮第一站

副标：深梦扬帆公益行动第二季 免费住15天

活动时间

即日起至2026年11月13日23：59

活动对象

计划赴深圳面试或工作的2026年毕业生（即2026年1月1日至2026年12月31日期间获得大专、本科及以上学历或学位的毕业生），即可报名参与活动。

*入住签约人必须符合该活动服务对象，且不是在住或历史客户，每个人只有一次参与机会。

活动房源

依托安居集团旗下“安居乐寓”及“寓见安居”品牌爬坡期和经营期自然空置的市场化房源，通过信息化管理、个性化匹配，为计划来深面试及工作的应届高校毕业生解决临时居住需求。。

活动权益

①15天免费住

②188无门槛租金券（领取后365天内新签可享，不可与其他营销优惠券叠加使用）

活动规则

1.申请说明：

（1）报名须知：客户须提前五天在小程序进行门店选择，并进行学信网信息验证，海外留学生需进行海外留学生验证，经过线上资格审核后，确认客户是否符合活动资格，符合资格且有空余房源即显示报名成功。

（2）资格审核：以毕业时间为唯一依据，凡在此时间段外（含提前或延后毕业）需提供额外补充材料（学校或者相关部门出具的延迟毕业证明或明确字眼体现“26年毕业”的证明文件）。

2.房源说明：

（1）深圳范围内安居乐寓单间房源与寓见安居单间房源（含合租单间）。具体房源由门店根据当时的实际经营情况进行分配。

（2）符合资格的毕业生可选择活动房源享受15天免费住权益，如需要选择非活动房源入住，可咨询门店对应房源价格，自行付费居住。

（3）每个门店活动房源数量有限，客户可提前咨询门店房源可入住情况。如期望入住的门店暂时无活动房源可提供，客户可咨询门店最快入住日期，调整日期入住，客户也可选择其他门店活动房源入住。

3.签约入住：客户签订《免费住活动协议》，阅读并同意租务管理系统的《用户协议》和《隐私政策》，并遵守《入住公约》和《安全须知》。

4.相关费用：

（1）免除15天月租金，签订活动协议时需实缴固定押金500元，退租时可退。

（2）住当天需缴纳所选房源对应的月服务费（188元、238元或288元），办理退租时会按实际居住天数扣除服务费(6-10元/天，具体根据门店收费标准执行）主要用于房间居住后的二次保洁、维护等运营成本，该费用不退；水电费根据实际居住门店水电收费标准进行缴纳，该费用不退。

5.居住租期

（1）居住租期最长为15天，（如您是1号入住，则需在第十五天也就是15号21：00前办理退租），入住后不可换房。

（2）居住期满需继续居住的，根据房源实际挂牌价，按照超期居住天数收取对应日租金与日服务费。

（3）如需长期居住，可在15天期满退租后，重新签约，享受毕业生专属折扣及188元无门槛租金券。

6.退租说明

居住15天期满及提前退租，实际入住天数产生的服务费、水电费将在押金、服务费中抵扣，剩余部分将退还。

如出现人为损坏资产的情况，则需缴纳相应的赔偿费用。

7.特别说明

公益行动房间每人在活动周期内仅享受一次15天免费住权益，房间仅限于毕业生本人居住，不可转租、不可用于其他经营、不可长期空置（3天以上）。

门店不提供日常生活用品，参与免费住活动的客户须自行配备。

门店有权定期抽查房间居住情况，如发现违规情况，门店有权清退。

（4）客户报名参与活动后，如果无法按照预约时间办理入住，我们将不再保留房源，若累计三次预约未提前取消者，则会被取消参与该活动的资格。
