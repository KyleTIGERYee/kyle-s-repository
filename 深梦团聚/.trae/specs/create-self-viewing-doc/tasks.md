# 任务列表 - 自助看房系统功能说明文档

## 任务概述

基于自助看房系统讨论会议纪要（以会议纪要内容为准），创建符合安居乐寓标准的《自助看房系统功能说明文档》。

**重要说明**：所有功能描述以会议纪要为准，原型文件仅作参考。

---

## 任务清单

### Task 1: 阅读书写规范文档

- [ ] 阅读 references/workflow-doc-generation.md 了解文档生成流程
- [ ] 阅读 references/quality-checklist.md 了解质量检查标准

### Task 2: 分析会议纪要（以会议纪要为准）
- [ ] 分析PC端功能：
  - [ ] 看房单管理（查询、列表、导出、评价展示规则）
  - [ ] 偏好设置（天数配置、拦截逻辑）
  - [ ] 楼栋设置（数据来源、剔除规则、展示控制）
- [ ] 分析小程序端功能：
  - [ ] 入口（首页图标、我的看房）
  - [ ] 门店列表（展示规则、定位功能）
  - [ ] 心愿单（加入、展示、已租判断、重复添加、删除、勾选）
  - [ ] 申请看房（申请人信息、二要素校验、时间选择、生成逻辑、数量限制、重复预约拦截）
  - [ ] 我的看房（有效/无效列表、状态流转）
  - [ ] 看房执行（人脸识别、门禁/房门开启、5分钟规则、评价、管家联系）

  - [ ] 重复预约限制规则
  - [ ] 门禁/房门开启规则
  - [ ] 看房单数量限制规则
  - [ ] 数据展示规则

- [ ] 编写概述章节：功能简介、适用角色、使用场景

  - [ ] 列表字段说明（房间号、看房人姓名、手机号、看房时间段、看房状态、看房评价）
  - [ ] 导出功能说明
  - [ ] 评价展示规则（仅已看房显示评价）

  - [ ] 拦截逻辑说明（开门禁或开房门都算）
- [ ] 编写楼栋设置功能
  - [ ] 数据来源说明（从楼栋数据读取）
  - [ ] 剔除规则（没有分配房间的楼栋不展示）
  - [ ] 展示控制说明（控制小程序自助看房列表展示）

1. 首先完成 Task 1 和 Task 2（准备工作）
2. 然后并行执行 Task 3、Task 4、Task 5、Task 6（内容编写）
3. 接着执行 Task 7 和 Task 8（补充章节）
4. 最后执行 Task 9 和 Task 10（检查与输出）

- **所有功能描述以会议纪要为准**
