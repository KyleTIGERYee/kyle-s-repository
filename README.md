# Anjuleyu AI-Driven Project Portfolio 🚀

这是一个由 AI 驱动（Vibe Coding）构建的安居乐寓智能化经营项目矩阵。本项目集展示了从财务核算、流程审批到营销自动化的一站式智能化解决方案。

## 🌟 项目矩阵

### 1. 门店调价审核 Agent (Store Price Review Agent)
- **核心能力**：基于决策树逻辑自动审核租金调整申请。
- **技术栈**：Dify + MCP (Model Context Protocol) + Python。
- **价值**：将小时级的审批流程缩短至分钟级，确保 100% 规则合规。

### 2. 门店核算看板 2.0 (Store Accounting Dashboard)
- **核心能力**：B 端财务分析可视化平台，支持 AI 问数与多店对标。
- **技术栈**：React 19 + TypeScript + Vite + TailwindCSS + Recharts。
- **亮点**：暗黑 Fintech 视觉设计，集成百度地图门店分布。

### 3. 深梦团聚 (Spring Festival Marketing Agent)
- **核心能力**：营销活动自动化审核，支持 OCR 身份核验。
- **亮点**：亲属关系智能识别准确率达 98%。

### 4. AI 管家 (AI Butler)
- **核心能力**：全生命周期租务服务智能体。
- **架构**：Multi-Agent（多智能体协作）系统。

### 5. LESSON (AI 实战课程)
- **内容**：涵盖提示词工程、工作流设计及 RAG 进阶的 AI 落地方法论。

---

## 🛠️ 技术底座 (Tech Stack)
- **AI 平台**: Dify, Coze, Google Antigravity
- **前端**: React, Vite, TailwindCSS
- **数据**: PostgreSQL, MCP
- **自动化**: Python Scripts
